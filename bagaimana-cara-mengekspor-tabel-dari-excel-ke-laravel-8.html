Bagaimana Cara Meng-Export Tabel Dari Excel Ke Laravel 8

Ketika membangun aplikasi web dengan Laravel 8, kadang-kadang Anda perlu mengimpor data dari Excel. Seiring berjalannya waktu, fitur ini menjadi semakin populer di platform Laravel. Di sini, kami akan membahas cara mengekspor tabel dari Excel ke Laravel 8.

Pertama, Anda harus menginstal ekstensi Pusher dari Composer. Ini akan memungkinkan Anda untuk mengimpor file Excel ke Laravel. Untuk menginstalnya, buka terminal dan jalankan perintah berikut:

composer require pusher/pusher

Kemudian, Anda harus membuat tabel di database Anda. Anda dapat melakukannya dengan menggunakan bantuan Migration. Buka terminal dan jalankan perintah berikut untuk membuat tabel:

php artisan make:migration create_yourtablename_table

Kemudian, buka file migration yang baru dibuat dan jalankan perintah berikut untuk membuat kolom tabel:

Schema::create('table_name', function (Blueprint $table) { $table->id(); $table->string('name'); $table->string('email'); $table->timestamps(); });

Kemudian, Anda harus menginstal package Excel. Ini akan memungkinkan Anda untuk mengimpor dan mengekspor file Excel. Untuk menginstalnya, buka terminal dan jalankan perintah berikut:

composer require maatwebsite/excel

Setelah itu, Anda harus membuat controller untuk mengimpor file Excel. Buka terminal dan jalankan perintah berikut untuk membuat controller:

php artisan make:controller YourControllerNameController

Kemudian, buka file controller yang baru dibuat dan tambahkan kode berikut:

use Maatwebsite\Excel\Concerns\ToModel; class YourControllerNameController extends Controller { public function import(Request $request) { $request->validate([ 'import_file' => 'required' ]); Excel::import(new YourModelName, request()->file('import_file')); return back()->with('success', 'Your file successfully import in database!!!'); } }

Setelah itu, Anda harus membuat model untuk mengimpor file Excel. Buka terminal dan jalankan perintah berikut untuk membuat model:

php artisan make:model YourModelName

Kemudian, buka file model yang baru dibuat dan tambahkan kode berikut:

use Illuminate\Database\Eloquent\Model; class YourModelName extends Model { protected $fillable = [ 'name', 'email', ]; }

Setelah itu, Anda harus mengatur rute untuk mengimpor file Excel. Buka file routes/web.php dan tambahkan kode berikut:

Route::post('import', 'YourControllerNameController@import');

Kemudian, Anda harus membuat view untuk mengimpor file Excel. Buka terminal dan jalankan perintah berikut untuk membuat view:

php artisan make:view import

Kemudian, buka file view yang baru dibuat dan tambahkan kode berikut:

@extends('layouts.app') @section('content')

Setelah semua tahap di atas, Anda dapat mengekspor tabel dari Excel ke Laravel 8. Untuk melakukannya, Anda harus membuat model untuk mengekspor tabel. Buka terminal dan jalankan perintah berikut untuk membuat model:

php artisan make:model YourModelName

Kemudian, buka file model yang baru dibuat dan tambahkan kode berikut:

use Illuminate\Database\Eloquent\Model; use Maatwebsite\Excel\Concerns\FromCollection; class YourModelName extends Model implements FromCollection { public function collection() { return YourModelName::all(); } }

Setelah itu, Anda harus membuat controller untuk mengekspor file Excel. Buka terminal dan jalankan perintah berikut untuk membuat controller:

php artisan make:controller YourControllerNameController

Kemudian, buka file controller yang baru dibuat dan tambahkan kode berikut:

use Maatwebsite\Excel\Concerns\FromCollection; class YourControllerNameController extends Controller { public function export() { return Excel::download(new YourModelName, 'yourfilename.xlsx'); } }

Setelah itu, Anda harus mengatur rute untuk mengekspor file Excel. Buka file routes/web.php dan tambahkan kode berikut:

Route::get('export', 'YourControllerNameController@export');

Kemudian, Anda harus membuat view untuk mengekspor file Excel. Buka terminal dan jalankan perintah berikut untuk membuat view:

php artisan make:view export

Kemudian, buka file view yang baru dibuat dan tambahkan kode berikut:

@extends('layouts.app') @section('content')

@endsection

Dengan demikian, Anda dapat mengekspor tabel dari Excel ke Laravel 8. Metode ini sangat berguna untuk memudahkan proses penggunaan data Excel di aplikasi web Anda. Selamat mencoba!
