Mengenal Python

Python adalah bahasa pemrograman yang diciptakan oleh Guido van Rossum pada tahun 1991. Python adalah bahasa pemrograman tingkat tinggi yang banyak digunakan untuk menyelesaikan berbagai macam masalah. Python merupakan bahasa yang mudah dimengerti, fleksibel, dan kuat. Pemrogram menggunakan Python untuk menulis kode yang bersifat alami, yang memudahkan untuk memahami dan memeriksa kode. Python mendukung berbagai macam jenis komputasi seperti komputasi numerik, komputasi visualisasi, dan komputasi wawasan mesin.

Mengenal Loop

Loop adalah sebuah kontrol alur logis yang digunakan untuk mengeksekusi perintah secara berulang-ulang. Python juga menyediakan berbagai macam loop seperti while loop, for loop, dan nested loop. While loop adalah loop yang mengulangi sebuah statement atau blok kode berdasarkan kondisi yang diberikan. While loop akan terus diulang hingga kondisi yang ditentukan bernilai false.

Mencetak Angka Genap dari 1 sampai 10 Menggunakan While Loop di Python

Untuk membuat program yang mencetak angka genap dari 1 sampai 10, kita harus menggunakan while loop. Kita akan menentukan sebuah variabel awal, misalnya num, yang bernilai 1. Lalu kita akan menggunakan while loop untuk mengecek kondisi yang diberikan. Jika num kurang dari sama dengan 10, maka loop akan terus berjalan. Jika angka yang dicetak adalah genap, maka loop akan terus mencetak angka. Jika angka yang dicetak adalah ganjil, loop akan melanjutkan pencetakan angka genap. Setelah loop selesai, nilai num akan ditambah 1 dan loop akan mengecek kondisi kembali. Ini akan terus berulang hingga nilai num lebih dari 10.

Cara Menulis Program di Python

Untuk menulis program di Python, langkah pertama adalah mendeklarasikan variabel num dan menentukan nilainya menjadi 1. Kode ini akan terlihat seperti ini:

num = 1

Setelah itu, kita harus menuliskan kode while loop. Kode ini akan terlihat seperti ini:

while num <= 10:

Langkah selanjutnya adalah menuliskan statement yang akan diulang oleh loop. Kode ini akan terlihat seperti ini:

if num % 2 == 0:
print(num)

Statement di atas menyatakan bahwa jika nilai num dibagi 2 dan hasilnya 0, maka angka genap akan dicetak. Langkah terakhir adalah menambah nilai num dengan 1 setiap kali loop berakhir. Kode ini akan terlihat seperti ini:

num = num + 1

Hasil Program di Python

Setelah semua kode di atas dituliskan, maka program kita akan selesai dan akan mencetak angka genap dari 1 sampai 10. Hasilnya akan terlihat seperti ini:

2
4
6
8
10

Kita juga dapat membuat program yang mencetak angka genap dari 1 sampai 10 dengan menggunakan while loop. Dengan memahami konsep ini, kita dapat menciptakan program yang lebih kompleks dengan menggunakan kombinasi berbagai macam loop dan kondisi.
