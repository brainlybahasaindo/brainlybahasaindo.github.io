Bagaimana Anda Menambahkan Kelas Ke Sesuatu Dalam Javascript

Apa itu Kelas dalam JavaScript?

Kelas adalah suatu konsep dalam pemrograman yang memungkinkan Anda untuk menciptakan objek yang berbasis pada template yang telah ditentukan. Dengan menggunakan kelas, Anda dapat membuat objek dengan properti dan metode yang sama. Pada dasarnya, Anda dapat menggunakan kelas untuk mengurangi kerumitan kode dengan menciptakan struktur yang dapat digunakan berulang kali. Sebelum JavaScript ES6, JavaScript tidak menyediakan cara untuk membuat kelas. Namun, sejak ES6, JavaScript memiliki fitur kelas built-in yang dapat Anda gunakan untuk membuat kelas dan objek.

Bagaimana Cara Menambahkan Kelas ke Sesuatu dalam JavaScript?

Untuk menambahkan kelas ke sesuatu dalam JavaScript, Anda harus menggunakan metode className. Metode ini dapat Anda temukan pada semua elemen HTML. Metode ini menerima string dan mengembalikan string yang menampilkan nama kelas elemen. Untuk menambahkan kelas ke elemen, Anda harus menempatkan nama kelas tersebut di dalam string. Anda juga dapat menggunakan metode ini untuk menghapus atau mengganti kelas yang ada. Untuk menghapus kelas, Anda harus menghapus nama kelas dari string yang dikembalikan.

Contoh

Berikut adalah contoh sederhana mengenai bagaimana Anda dapat menambahkan kelas ke sesuatu dalam JavaScript:

let element = document.querySelector("#myDiv");
 element.className +=" myClass";

Pada contoh di atas, kami menggunakan metode querySelector untuk mengambil elemen dengan id myDiv. Kami lalu menambahkan kelas bernama myClass ke elemen tersebut dengan menggunakan metode className.

Metode lain untuk Menambahkan Kelas

Selain menggunakan metode className, Anda juga dapat menggunakan metode classList untuk menambahkan kelas ke sesuatu dalam JavaScript. Metode ini dapat Anda temukan pada semua elemen HTML. Metode ini menerima string dan mengembalikan objek classList. Objek ini memiliki metode untuk menambahkan, menghapus, dan mengganti kelas. Berikut adalah contoh sederhana mengenai bagaimana Anda dapat menggunakan metode classList:

let element = document.querySelector("#myDiv");
 element.classList.add("myClass");

Pada contoh di atas, kami menggunakan metode querySelector untuk mengambil elemen dengan id myDiv. Kami lalu menambahkan kelas bernama myClass ke elemen tersebut dengan menggunakan metode classList.add.

Kesimpulan

Dalam artikel ini, kami telah membahas bagaimana Anda dapat menambahkan kelas ke sesuatu dalam JavaScript. Kami juga telah membahas dua metode yang dapat Anda gunakan untuk menambahkan kelas ke elemen HTML. Pertama, Anda dapat menggunakan metode className. Kedua, Anda dapat menggunakan metode classList.
