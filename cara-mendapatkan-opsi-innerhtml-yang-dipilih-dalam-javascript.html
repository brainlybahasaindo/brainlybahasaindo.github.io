Apa itu Opsi InnerHTML?

Opsi InnerHTML adalah sebuah properti HTML yang berfungsi untuk mengakses atau menetapkan nilai yang terkandung di dalam tag HTML. Secara sederhana, Anda dapat menggunakan properti InnerHTML untuk mengakses atau menetapkan konten yang ada di dalam tag HTML. Ini juga dapat digunakan untuk mengubah properti HTML dengan mengubah nilai InnerHTML.

Opsi InnerHTML juga dapat digunakan dalam kode JavaScript dan Anda dapat menggunakannya untuk melakukan berbagai macam hal. Misalnya, Anda dapat menggunakannya untuk mendapatkan opsi yang dipilih dari sebuah tag HTML. Dalam artikel ini, kita akan melihat bagaimana cara mendapatkan opsi InnerHTML yang dipilih dalam Javascript.

Bagaimana Cara Mendapatkan Opsi InnerHTML yang Dipilih dalam Javascript?

Untuk mendapatkan opsi InnerHTML yang dipilih dalam Javascript, Anda dapat menggunakan sintaks berikut:

document.querySelector('#selectID').value

Di sini, #selectID adalah ID dari tag HTML yang dipilih. Anda dapat menggunakan properti value untuk mendapatkan nilai yang dipilih dari tag HTML tersebut. Untuk mendapatkan nilai InnerHTML yang dipilih, Anda dapat menggunakan sintaks berikut:

document.querySelector('#selectID').innerHTML

Di sini, #selectID adalah ID tag HTML yang dipilih. Anda dapat menggunakan properti innerHTML untuk mendapatkan nilai yang terkandung di dalam tag HTML tersebut. Anda juga dapat menggunakan properti innerHTML untuk mengubah konten di dalam tag HTML.
