Apa Itu Tipe Data?

Tipe data merupakan cara untuk mengelompokkan informasi. Misalnya, mengenali bahwa suatu nilai adalah nomor, teks, atau boolean. Di Python, kita mengenal beberapa tipe data, di antaranya string, float, dan boolean. Pemahaman tentang tipe data ini sangat penting agar kita bisa menggunakan Python dengan benar.

Penjelasan Singkat Tentang Tipe Data di Python

Tipe data di Python dapat diklasifikasikan menjadi dua; tipe data tunggal dan tipe data kompleks. Tipe data tunggal yang biasa kita temui adalah string, float, dan boolean. String merupakan tipe data yang berisi teks. Float merupakan tipe data yang berisi angka desimal, sedangkan boolean merupakan tipe data yang berisi nilai benar atau salah. Tipe data kompleks yang ada di Python di antaranya list, tuple, set, dan dictionary. List merupakan tipe data yang berisi daftar nilai, tuple merupakan tipe data yang berisi daftar nilai yang tidak dapat diubah, set merupakan tipe data yang berisi daftar nilai yang unik, dan dictionary merupakan tipe data yang berisi daftar kunci-nilai.

Contoh Tipe Data di Python

Di bawah ini adalah contoh dari tipe data yang ada di Python.

String

String adalah tipe data yang berisi teks. Contohnya, ‘saya sedang belajar Python’.

Float

Float adalah tipe data yang berisi angka desimal. Contohnya, 3.14.

Boolean

Boolean adalah tipe data yang berisi nilai benar atau salah. Contohnya, True atau False.

List

List adalah tipe data yang berisi daftar nilai. Contohnya, [1, 2, 3].

Tuple

Tuple adalah tipe data yang berisi daftar nilai yang tidak dapat diubah. Contohnya, (1, 2, 3).

Set

Set adalah tipe data yang berisi daftar nilai yang unik. Contohnya, {1, 2, 3}.

Dictionary

Dictionary adalah tipe data yang berisi daftar kunci-nilai. Contohnya, {'satu':1, 'dua':2, 'tiga':3}.

Kesimpulan Tipe Data di Python

Tipe data di Python dapat diklasifikasikan menjadi tipe data tunggal dan tipe data kompleks. Tipe data tunggal yang biasa kita temui adalah string, float, dan boolean. Tipe data kompleks yang ada di Python di antaranya list, tuple, set, dan dictionary. Dengan memahami tipe data di Python, kita dapat menggunakan Python dengan benar dan membuat program yang lebih efisien.
