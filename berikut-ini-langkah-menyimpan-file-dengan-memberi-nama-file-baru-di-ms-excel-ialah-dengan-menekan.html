Bagaimana Cara Menyimpan File Di Ms Excel Dengan Memberi Nama File Baru?

Terbaik Pelajaran bahasa indonesia tips dan trik website . Cari apa saja topik Pelajaran bahasa indonesia ide di website ini.

MS Excel adalah sebuah program yang banyak digunakan untuk mengolah data. Menyimpan file di MS Excel sangat mudah dilakukan. Namun, bagaimana jika Anda ingin menyimpan file dengan memberi nama file baru? Berikut ini adalah langkah-langkah yang harus Anda ikuti untuk menyimpan file dengan memberi nama file baru di MS Excel.

Langkah 1: Buka MS Excel dan Buka File yang Ingin Anda Simpan

Untuk memulai, Anda harus membuka program MS Excel terlebih dahulu. Kemudian, buka file yang ingin Anda simpan dengan memberi nama file baru. Jika Anda belum punya file Excel, Anda dapat membuatnya dari awal.

Langkah 2: Klik ikon Simpan

Setelah membuka file yang ingin Anda simpan, Anda harus mencari ikon Simpan. Biasanya ikon ini berbentuk seperti sebuah kotak dengan lingkaran di dalamnya. Klik ikon ini untuk melanjutkan proses penyimpanan file.

Langkah 3: Tentukan Nama File Baru

Setelah menekan ikon Simpan, Anda harus memberi nama baru untuk file yang ingin Anda simpan. Anda dapat memberi nama sesuai keinginan Anda, tetapi jangan lupa untuk menambahkan ekstensi file yang tepat (.xlsx atau .xls). Kemudian, tekan tombol Simpan.

Langkah 4: Pilih Lokasi Penyimpanan File

Setelah Anda menentukan nama baru untuk file, Anda harus memilih lokasi tempat file akan disimpan. Anda dapat memilih lokasi dari folder di komputer Anda atau menyimpannya di layanan penyimpanan file online seperti Dropbox atau Google Drive.

Langkah 5: Selesai

Setelah Anda memilih lokasi penyimpanan file, tekan tombol Simpan di jendela yang muncul. Tunggu beberapa detik hingga proses penyimpanan selesai. Jika proses telah selesai, Anda akan melihat file baru yang telah Anda simpan di lokasi yang telah Anda pilih.

Kesimpulan

Itulah cara menyimpan file di MS Excel dengan memberi nama file baru. Proses ini sangat mudah dan hanya membutuhkan beberapa langkah saja. Jadi, jika Anda ingin menyimpan file dengan memberi nama file baru di MS Excel, ikuti langkah-langkah di atas untuk memulainya.
