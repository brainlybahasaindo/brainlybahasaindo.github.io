Python adalah bahasa pemrograman yang populer dan banyak digunakan untuk membangun aplikasi web. Bahasa ini menyediakan berbagai fungsi yang memungkinkan Anda untuk bekerja dengan format tanggal/waktu yang berbeda. Di sini kita akan membahas cara menggunakan fungsi datetime RFC3339 di Python.

Apa yang Dimaksud dengan Datetime RFC3339?

Datetime RFC3339 adalah format standar yang ditetapkan oleh IETF untuk menyimpan tanggal dan waktu. Format ini juga dikenal sebagai Internet Date/Time Format atau ISO 8601:2004. Format ini memungkinkan Anda untuk dengan mudah membuat, menyimpan, dan membagikan tanggal dan waktu di antara sistem yang berbeda. Format ini juga menyediakan fleksibilitas untuk menyimpan tanggal dan waktu dalam format yang berbeda.

Format datetime RFC3339 didefinisikan sebagai berikut: YYYY-MM-DDThh:mm:ss.sssZ. Di sini, "T" adalah pemisah antara tanggal dan waktu, dan "Z" adalah zona waktu. Di sini, "YYYY" adalah tahun, "MM" adalah bulan, "DD" adalah hari, "hh" adalah jam, "mm" adalah menit, dan "ss.sss" adalah detik.

Cara Menggunakan Fungsi Datetime RFC3339 di Python

Python menyediakan fungsi datetime.strptime() yang dapat Anda gunakan untuk mengonversi string ke objek waktu yang dapat digunakan dalam aplikasi Anda. Fungsi ini mengambil dua argumen: string yang ingin Anda konversi dan format string yang menyatakan format string. Untuk format datetime RFC3339, Anda harus menggunakan string format berikut: %Y-%m-%dT%H:%M:%S.%fZ.

Mari kita lihat contoh berikut ini:

 import datetime
 
 # String yang ingin konversi
 str_date ="2023-03-17T21:07:36.123Z"
 
 # Format string
 format_str ="%Y-%m-%dT%H:%M:%S.%fZ"
 
 # Konversi string ke objek waktu
 date_obj = datetime.datetime.strptime(str_date, format_str)
 
 print(date_obj) # Output: 2023-03-17 21:07:36.123000

Di sini kita menggunakan fungsi datetime.strptime() untuk mengkonversi string ke objek waktu. Kami menyediakan string yang ingin kami konversi dan format string yang menyatakan format string. Output dari contoh di atas adalah objek waktu yang disimpan dalam variabel date_obj.

Selain fungsi strptime(), Python juga menyediakan fungsi datetime.strftime() yang dapat Anda gunakan untuk mengkonversi objek waktu ke string. Fungsi ini mengambil dua argumen: objek waktu yang ingin Anda konversi dan format string yang menyatakan format string. Berikut adalah contoh penggunaannya:

 import datetime
 
 # Objek waktu
 date_obj = datetime.datetime(2023, 3, 17, 21, 7, 36, 123000)
 
 # Format string
 format_str ="%Y-%m-%dT%H:%M:%S.%fZ"
 
 # Konversi objek waktu ke string 
 str_date = date_obj.strftime(format_str)
 
 print(str_date) # Output: 2023-03-17T21:07:36.123000Z

Di sini kita menggunakan fungsi datetime.strftime() untuk mengkonversi objek waktu ke string. Kami menyediakan objek waktu yang ingin kami konversi dan format string yang menyatakan format string. Output dari contoh di atas adalah string yang disimpan dalam variabel str_date.

Kesimpulan

Datetime RFC3339 adalah format standar untuk menyimpan tanggal dan waktu. Format ini menggunakan sintaks RFC 3339 yang diterima secara luas di seluruh web. Python menyediakan fungsi datetime.strptime() dan datetime.strftime() yang dapat Anda gunakan untuk mengonversi string ke objek waktu dan sebaliknya. Dengan menggunakan format ini, Anda dapat dengan mudah membuat, menyimpan, dan membagikan tanggal dan waktu di antara sistem yang berbeda.
