Cara Mengubah Versi Php Default Di Whm

Terbaik Pelajaran bahasa indonesia tips dan trik website . Cari apa saja topik Pelajaran bahasa indonesia ide di website ini.

Ketika Anda memiliki hosting web dengan cPanel, Anda akan memiliki akses ke WHM (Web Host Manager). Ini adalah antarmuka yang digunakan untuk mengelola dan memperbarui server web Anda. Hal yang paling penting untuk dikonfigurasi di WHM adalah versi PHP default yang digunakan untuk situs web Anda. Jika PHP Anda tidak diperbarui selama beberapa tahun, maka Anda akan kehilangan banyak fitur, kemampuan, dan keamanan.

Ketika Anda menggunakan WHM, Anda dapat dengan mudah mengubah versi PHP default Anda. Ini akan mengubah versi PHP yang digunakan oleh semua situs web yang Anda miliki di server web Anda. Ini adalah cara yang bagus untuk memastikan bahwa semua situs web Anda berjalan dengan baik dan memiliki akses ke fitur terbaru. Di bawah ini adalah panduan langkah demi langkah tentang cara mengubah versi PHP default di WHM.

Langkah 1: Masuk ke WHM

Pertama, Anda harus masuk ke WHM. Buka browser web Anda dan masukkan alamat IP server web Anda atau URL WHM Anda. Anda akan melihat antarmuka WHM yang berbeda. Masukkan nama pengguna dan kata sandi Anda untuk masuk ke WHM.

Langkah 2: Cari Pengaturan PHP

Setelah masuk ke WHM, cari bagian "Pengaturan PHP". Ini berada di bagian bawah layar. Anda akan melihat daftar berbagai pengaturan PHP yang dapat Anda ubah.

Langkah 3: Ubah Versi PHP

Setelah Anda menemukan pengaturan PHP, temukan opsi yang mengatakan "Versi PHP Default". Di sini, Anda dapat memilih versi PHP yang ingin Anda gunakan. Pilih versi yang paling baru yang tersedia.

Langkah 4: Simpan Perubahan

Setelah Anda memilih versi PHP yang baru, Anda harus menyimpan perubahan. Anda akan melihat tombol "Simpan" di bagian bawah layar. Klik tombol ini untuk menyimpan perubahan.

Langkah 5: Perbarui Situs Web Anda

Setelah Anda menyimpan perubahan, Anda harus memastikan bahwa semua situs web Anda berjalan dengan aman dengan versi PHP yang baru. Untuk melakukan ini, Anda harus memperbarui situs web Anda. Anda dapat melakukan ini dengan mengunjungi situs web Anda satu per satu dan memeriksa apakah mereka berfungsi dengan benar.

Langkah 6: Periksa Semua Situs Web Anda

Setelah Anda memperbarui semua situs web Anda, Anda harus memeriksa semua situs web Anda untuk memastikan bahwa mereka berjalan dengan benar. Jika Anda menemukan masalah, Anda harus mencari tahu apa yang salah dan memperbaikinya. Jika semuanya berjalan dengan benar, Anda dapat melanjutkan dengan menggunakan versi PHP default baru Anda.

Kesimpulan

Mengubah versi PHP default di WHM adalah cara yang bagus untuk memastikan bahwa semua situs web Anda berjalan dengan baik dan memiliki akses ke fitur terbaru. Anda dapat melakukan ini dengan mengikuti panduan langkah demi langkah di atas. Pastikan Anda selalu memeriksa semua situs web Anda setelah mengubah versi PHP default untuk memastikan bahwa semuanya berjalan dengan benar.
