Cara Menggunakan Upper Pada Python

Terbaik Pelajaran bahasa indonesia tips dan trik website . Cari apa saja topik Pelajaran bahasa indonesia ide di website ini.

Python merupakan bahasa pemrograman yang populer, dikarenakan kemudahannya dalam pembuatan aplikasi web dan juga mobile. Salah satu fungsi dari bahasa pemrograman ini adalah upper. Upper merupakan fungsi yang dapat mengubah semua karakter yang ditulis menjadi huruf besar. Fungsi ini bisa diaplikasikan pada string, sehingga memudahkan dalam proses pembuatan program.

Apa Itu Upper Pada Python?

Upper merupakan fungsi yang dapat mengubah semua karakter menjadi huruf besar. Hal ini berguna ketika kamu ingin membuat sebuah program yang menghasilkan output berupa huruf besar. Dengan menggunakan fungsi upper, kamu tidak perlu menuliskan karakter satu per satu menjadi huruf besar. Fungsi ini juga dapat memudahkan dalam membuat program yang membutuhkan kasus huruf besar.

Bagaimana Cara Menggunakan Upper Pada Python?

Untuk menggunakan upper pada Python, kamu bisa memanfaatkan syntax berikut ini: string.upper(). Dimana string merupakan variabel yang berisi string yang ingin diubah menjadi huruf besar. Contoh kode programnya seperti ini:

 # Contoh kode
 
 # Contoh string
 string ="Hallo World!"
 
 # Menggunakan upper
 string = string.upper()
 
 # Output
 print(string)
 
 # Hasil output
 HALLO WORLD!

Kode di atas akan menghasilkan output berupa huruf besar. Fungsi upper akan mengubah semua karakter yang dituliskan menjadi huruf besar.

Kelebihan Menggunakan Upper Pada Python

Berikut adalah beberapa kelebihan yang bisa kamu dapatkan saat menggunakan upper pada Python:

  • Mudah diterapkan dalam program.
  • Memudahkan dalam menuliskan kode program.
  • Membuat program menjadi lebih ringkas dan efisien.
  • Membantu dalam mencari kata dan kalimat yang dituliskan dalam huruf besar.

Kekurangan Menggunakan Upper Pada Python

Selain kelebihan, ada juga beberapa kekurangan yang bisa kamu temui saat menggunakan upper pada Python:

  • Tidak bisa mengubah karakter spesial menjadi huruf besar.
  • Tidak bisa mengubah karakter spesial yang disertakan dengan string.
  • Tidak bisa mengubah karakter spesial di dalam string.

Contoh Program Menggunakan Upper Pada Python

Berikut adalah contoh program yang menggunakan upper pada Python:

 # Contoh Program
 
 # Membuat variabel
 string ="Hallo World!"
 
 # Mengubah karakter menjadi huruf besar
 string = string.upper()
 
 # Menampilkan output
 print(string)
 
 # Hasil output
 HALLO WORLD!

Program di atas akan menghasilkan output berupa huruf besar. Dengan menggunakan upper, kamu bisa membuat sebuah program yang menghasilkan output berupa kalimat yang dituliskan dalam huruf besar.

Kesimpulan

Upper pada Python bisa menjadi salah satu fungsi yang berguna dalam membuat sebuah program. Fungsi ini dapat mengubah semua karakter menjadi huruf besar.
