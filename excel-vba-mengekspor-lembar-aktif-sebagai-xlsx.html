Cara Ekspor Lembar Aktif Excel Ke Xlsx Dengan Vba

Terbaik Pelajaran bahasa indonesia tips dan trik website . Cari apa saja topik Pelajaran bahasa indonesia ide di website ini.

Dalam pembuatan sebuah laporan di Microsoft Excel, para pengguna sering menggunakan Microsoft Visual Basic for Applications (VBA) untuk membuat laporan yang mudah dibaca. VBA memungkinkan pengguna untuk menyimpan laporan sebagai file Excel (.xlsx) yang dapat diubah menjadi file lain. Untuk mengekspor lembar aktif Excel ke file XLSX, Anda perlu menggunakan kode VBA berikut.

Langkah-Langkah untuk Mengekspor Lembar Aktif Excel ke XLSX

Langkah 1: Buka Visual Basic Editor

Buka Visual Basic Editor (VBE) dengan menekan tombol Alt+F11. VBE adalah editor kode yang digunakan untuk membuat laporan Excel yang lebih mudah dibaca. Setelah Anda membuka VBE, kemudian buka lembar aktif yang ingin Anda ekspor. Kemudian buka Visual Basic Editor (VBE) lewat tab Developer dalam Ribbon Excel. Jika tab Developer tidak ditampilkan, buka opsi “Ribbon Display Options” dari tab File dan pilih “Show Developer Tab in the Ribbon”.

Langkah 2: Buat Sub Routine

Setelah Anda membuka Visual Basic Editor, Anda perlu membuat sub routine baru. Sub routine adalah bagian dari macro yang mengandung kode yang akan dijalankan oleh VBA. Anda dapat membuat sub routine dengan menekan tombol F5 atau dengan memilih menu Insert > Module. Setelah membuat sub routine, Anda perlu menambahkan kode VBA untuk mengekspor lembar aktif ke file XLSX.

Langkah 3: Tambahkan Kode VBA

Untuk mengekspor lembar aktif ke file XLSX, Anda perlu menambahkan kode VBA berikut. Kode ini menyimpan lembar aktif yang terbuka ke file XLSX dengan nama yang telah Anda tentukan.

ActiveWorkbook.SaveAs Filename:="C:\Documents\MyFile.xlsx", FileFormat:=xlOpenXMLWorkbook

Langkah 4: Jalankan Kode VBA

Setelah Anda menambahkan kode VBA, Anda dapat menjalankan kode tersebut dengan menekan tombol F5 atau dengan memilih menu Run > Run Sub/UserForm. Setelah menjalankan kode, lembar aktif yang terbuka akan disimpan ke file XLSX dengan nama yang telah Anda tentukan. Anda juga dapat mengubah kode VBA sesuai kebutuhan.

Kesimpulan

Mengekspor lembar aktif Excel ke file XLSX dapat dilakukan dengan menggunakan kode VBA. Kode ini dapat Anda jalankan dengan menekan tombol F5 atau dengan memilih menu Run > Run Sub/UserForm. Dengan begitu, Anda dapat dengan mudah mengekspor lembar aktif Excel ke file XLSX yang dapat diubah menjadi file lain.
