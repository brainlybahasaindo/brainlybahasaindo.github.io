Cara Mudah Membaca Dictionary Di Python

Terbaik Pelajaran bahasa indonesia tips dan trik website . Cari apa saja topik Pelajaran bahasa indonesia ide di website ini.

Python adalah bahasa pemrograman yang banyak digunakan untuk berbagai macam aplikasi. Salah satu yang paling populer adalah membaca dictionary. Dictionary adalah kumpulan data yang berisi pasangan key dan value. Setiap pasangan ini dinamakan item. Nilai dari setiap item dapat diakses dengan menggunakan key.

Cara membaca dictionary di Python tidak terlalu sulit. Anda cukup membuat dict objek dan menggunakan looping untuk mengakses nilai dari masing-masing item. Terkadang, Anda juga dapat menggunakan method built-in untuk membaca dictionary. Inilah cara membaca dictionary di Python.

1. Looping

Looping adalah cara yang paling umum digunakan untuk membaca dictionary. Anda dapat menggunakan for loop untuk mengakses item satu per satu. Dalam looping, Anda harus mendefinisikan loop variable yang berisi key dari item yang akan Anda baca. Anda dapat menggunakan loop variable untuk mengakses item dan mendapatkan nilai dari item tersebut.

Berikut adalah contoh kode untuk membaca dictionary menggunakan looping:

 dict = {'apple': 1, 'banana': 2, 'orange': 3}
 
 for key in dict:
 print(key, ':', dict[key])

Kode di atas akan menghasilkan output berikut:

 apple : 1
 banana : 2
 orange : 3

Dalam contoh di atas, loop variable adalah 'key', yang berisi key dari item yang akan Anda baca. Anda juga dapat menggunakan loop variable untuk mengakses nilai dari item tersebut.

2. Built-in Method

Selain looping, Anda juga dapat menggunakan method built-in untuk membaca dictionary. Salah satu method yang paling populer adalah get(). Method ini digunakan untuk mendapatkan nilai dari suatu item yang ditentukan oleh key. Metode ini akan mengembalikan nilai default jika key tidak ditemukan di dalam dictionary. Berikut adalah contoh kode untuk menggunakan method get():

 dict = {'apple': 1, 'banana': 2, 'orange': 3}
 
 value = dict.get('apple')
 print(value)

Kode di atas akan menghasilkan output berikut:

 1

3. Iteritems() Method

Selain method get(), Anda juga dapat menggunakan method iteritems() untuk membaca dictionary. Metode ini digunakan untuk mengiterasi setiap item di dalam dictionary. Metode ini akan mengembalikan pasangan key dan value dari setiap item. Berikut adalah contoh kode untuk menggunakan method iteritems():

 dict = {'apple': 1, 'banana': 2, 'orange': 3}
 
 for key, value in dict.iteritems():
 print(key, ':', value)

Kode di atas akan menghasilkan output berikut:

 apple : 1
 banana : 2
 orange : 3

4. Items() Method

Selain iteritems(), Anda juga dapat menggunakan method items() untuk membaca dictionary. Metode ini sama dengan iteritems(), tapi items() mengembalikan list yang berisi pasangan key dan value dari setiap item. Berikut adalah contoh kode untuk menggunakan method items():

 dict = {'apple': 1, 'banana': 2, 'orange': 3}
 
 for key, value in dict.items():
 print(key, ':', value)

Kode di atas akan menghasilkan output berikut:

 apple : 1
 banana : 2
 orange : 3

5. Keys() Method

Selain iteritems() dan items(), Anda juga dapat menggunakan method keys() untuk membaca dictionary. Metode ini digunakan untuk mendapatkan list yang berisi key dari setiap item di dalam dictionary. Berikut adalah contoh kode untuk menggunakan method keys():

 dict = {'apple': 1, 'banana': 2, 'orange': 3}
 
 for key in dict.keys():
 print(key)

Kode di atas akan menghasilkan output berikut:

 apple
 banana
 orange

Kesimpulan

Itulah cara membaca dictionary di Python. Anda dapat menggunakan looping, method built-in, atau method lainnya untuk membaca dictionary. Penting untuk diingat bahwa cara yang paling umum digunakan adalah looping. Selain itu, Anda juga dapat menggunakan method built-in untuk mendapatkan nilai dari suatu item.
