Cara Memisahkan Lembar Excel Menjadi Beberapa File Berdasarkan Kolom Menggunakan Vba

Terbaik Pelajaran bahasa indonesia tips dan trik website . Cari apa saja topik Pelajaran bahasa indonesia ide di website ini.

Saat bekerja di Excel, Anda mungkin menemukan situasi di mana Anda harus memisahkan lembar kerja yang berisi banyak data ke dalam beberapa file berdasarkan kolom tertentu. Pada saat seperti itu, Anda dapat menggunakan Macro VBA untuk memisahkan lembar kerja ke dalam beberapa file. Ini akan membuat pekerjaan Anda lebih mudah dan hemat waktu. Dalam tutorial ini, kami akan menunjukkan cara memisahkan lembar kerja Excel menjadi beberapa file berdasarkan kolom menggunakan VBA.

Apa yang Dibutuhkan untuk Memisahkan Lembar Excel?

Untuk memisahkan lembar Excel menjadi beberapa file berdasarkan kolom menggunakan VBA, Anda harus memenuhi persyaratan berikut:

  • Microsoft Excel versi 2007 atau yang lebih baru.
  • Pengetahuan VBA dasar.
  • Koneksi internet.

Apa Itu Macro VBA?

Macro VBA adalah kumpulan perintah yang dapat Anda gunakan untuk menjalankan tugas-tugas tertentu di Microsoft Excel. Anda dapat menggunakan Macro VBA untuk memisahkan lembar kerja, mengubah ukuran kolom, menghapus baris, mengambil data dari beberapa lembar kerja, dan banyak lagi. Macro VBA juga dapat membantu Anda menyelesaikan pekerjaan yang berulang dengan cepat dan efisien. Anda dapat membuat Macro VBA dengan menggunakan Editor Macro (ALT + F11).

Cara Membuat Macro VBA Untuk Memisahkan Lembar Excel

Berikut adalah langkah-langkah yang harus Anda ikuti untuk membuat Macro VBA untuk memisahkan lembar Excel menjadi beberapa file berdasarkan kolom:

  1. Buka lembar Excel yang ingin Anda pisahkan.
  2. Tekan tombol ALT + F11 untuk membuka Editor Macro.
  3. Pada jendela Editor Macro, buat modul baru dengan mengklik tombol New pada toolbar.
  4. Ketik kode berikut di dalam modul baru: Sub SplitSheet()
  5. Ketik kode berikut untuk mendeklarasikan variabel: Dim WorksheetName As String, FileName As String, ColumnName As String, LastRow As Long, LastColumn As Long, StartRow As Long, EndRow As Long, StartColumn As Long, EndColumn As Long
  6. Ketik kode berikut untuk mendapatkan nama lembar yang akan dipisahkan: WorksheetName = ActiveSheet.Name
  7. Ketik kode berikut untuk mendapatkan nama kolom yang akan dipisahkan: ColumnName = InputBox("Masukkan nama kolom yang akan dipisahkan:")
  8. Ketik kode berikut untuk mendapatkan jumlah baris terakhir: LastRow = Cells(Rows.Count, ColumnName).End(xlUp).Row
  9. Ketik kode berikut untuk mendapatkan jumlah kolom terakhir: LastColumn = Cells(1, Columns.Count).End(xlToLeft).Column
  10. Ketik kode berikut untuk memulai pembuatan file baru: For StartRow = 2 To LastRow
  11. Ketik kode berikut untuk mendapatkan nama file: FileName = Cells(StartRow, ColumnName).Value
  12. Ketik kode berikut untuk menentukan kolom mulai dan kolom akhir: StartColumn = 1 EndColumn = LastColumn
  13. Ketik kode berikut untuk menentukan baris mulai dan baris akhir: StartRow = StartRow EndRow = StartRow
  14. Ketik kode berikut untuk membuat file baru: ActiveWorkbook.Sheets.Add(After:=Sheets(Sheets.Count)).Name = FileName Range(Cells(StartRow, StartColumn), Cells(EndRow, EndColumn)).Select Selection.Copy Sheets(FileName).Select Range("A1").Select ActiveSheet.Paste
  15. Ketik kode berikut untuk menyimpan file baru: ActiveWorkbook.SaveAs FileName & ".xlsx"
  16. Ketik kode berikut untuk menutup file baru: ActiveWorkbook.Close Sheets(WorksheetName).Select
  17. Ketik kode berikut untuk menyelesaikan prosedur: Next StartRow End Sub

Cara Menjalankan Macro VBA Untuk Memisahkan Lembar Excel

Setelah Anda selesai membuat Macro VBA, Anda dapat menjalankannya dengan cara berikut:

  1. Tekan tombol ALT + F11 untuk membuka Editor Macro.
  2. Pilih modul yang telah Anda buat.
  3. Klik tombol Run pada toolbar atau tekan tombol F5 untuk menjalankan Macro.
  4. Anda akan diminta untuk memasukkan nama kolom yang akan dipisahkan. Masukkan nama kolom dan tekan tombol OK.
  5. Macro akan membuat file baru berdasarkan data di kolom tersebut.

Kesimpulan

Dengan Macro VBA, Anda dapat memisahkan lembar kerja Excel menjadi beberapa file berdasarkan kolom. Ini akan membuat pekerjaan Anda lebih mudah dan hemat waktu. Anda dapat dengan mudah membuat Macro VBA untuk memisahkan lembar Excel dengan mengikuti langkah-langkah di atas. Semoga tutorial ini bermanfaat.
