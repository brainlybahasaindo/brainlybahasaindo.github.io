HTML adalah singkatan dari HyperText Markup Language, merupakan bahasa pemrograman yang diterapkan di dalam web browser. HTML ini berfungsi untuk memformat dan mengatur layout halaman web, menampilkan berbagai jenis konten, dan membuat navigasi di dalam suatu website.

Fungsi HTML

Pada dasarnya, HTML memiliki beberapa fungsi, di antaranya adalah untuk:

  • Menyusun dan mengatur struktur halaman web
  • Mendefinisikan tata letak halaman web
  • Mengontrol format, warna, dan desain halaman web
  • Membuat navigasi di dalam suatu website
  • Membuat link antar halaman web
  • Menambahkan gambar, video, dan animasi
  • Menambahkan suara dan musik
  • Memasukkan form dan text box
  • Membuat tabel
  • Membuat daftar dan drop-down menu
  • Membuat aplikasi interaktif
  • Membuat halaman web responsive

Cara Menulis Kode HTML

Kode HTML ditulis menggunakan tag HTML. Setiap tag HTML memiliki tujuan dan fungsi yang berbeda. Tag HTML memiliki atribut tambahan yang dapat memodifikasi penampilan konten yang ditampilkan di halaman web. Tag HTML ditulis di dalam kurung siku. Misalnya, adalah tag HTML yang digunakan untuk memulai halaman web.

Untuk membuat elemen HTML, Anda harus menggunakan tag HTML yang berpasangan. Tag HTML berpasangan berisi tag pembuka dan penutup. Tag pembuka berisi konten yang akan dimasukkan di dalam halaman web, sedangkan tag penutup berisi informasi tentang bagaimana elemen HTML harus ditampilkan di halaman web. Misalnya,
