Setelah Anda berhasil mendownload dan menginstal Python Log2, Anda harus mengatur lingkungan pemrograman. Setelah Anda berhasil menulis kode Log2, Anda dapat mulai menjalankan aplikasi dengan Log2. Anda dapat menggunakan utilitas python-config untuk memulai aplikasi Anda.
