Cara Menghapus Php Dari Url Htaccess Di Tahun 2023

Terbaik Pelajaran bahasa indonesia tips dan trik website . Cari apa saja topik Pelajaran bahasa indonesia ide di website ini.

PHP adalah bahasa pemrograman yang digunakan untuk membuat situs web statis dan dinamis. Salah satu alasan utama menggunakan PHP adalah karena ia adalah salah satu bahasa pemrograman yang populer dan mudah untuk digunakan. Dengan banyaknya situs web yang dibuat menggunakan PHP, ada beberapa masalah yang mungkin terjadi, salah satunya adalah ketika PHP muncul di URL. Ini adalah masalah yang mungkin Anda alami dan untuk menghapusnya dari URL Anda, Anda harus menggunakan htaccess.

Apa itu htaccess?

htaccess adalah berkas teks yang dapat Anda temukan di folder root website Anda. Berkas ini dapat digunakan untuk mengatur berbagai pengaturan yang berhubungan dengan server web, termasuk mengatur URL. Karena berkas ini dapat Anda edit, maka Anda dapat menggunakannya untuk menghapus PHP dari URL Anda. Jika Anda ingin memahami cara kerjanya, berikut adalah penjelasan singkatnya:

Bagaimana htaccess Bekerja?

htaccess berfungsi untuk mengatur bagaimana server web harus merespons kepada sebuah permintaan. Jika Anda mengakses sebuah URL pada situs web Anda, server akan merespons sesuai dengan instruksi yang diberikan oleh htaccess. Jika Anda mengatur htaccess dengan benar, Anda dapat mencegah server untuk menampilkan PHP dalam URL Anda. Dengan kata lain, Anda dapat mengontrol bagaimana server merespons permintaan Anda dan menghapus PHP dari URL Anda.

Cara Menghapus PHP dari URL htaccess

Untuk menghapus PHP dari URL Anda, Anda harus mengedit berkas htaccess Anda. Jika Anda belum mengetahui cara mengedit berkas htaccess, berikut adalah langkah-langkahnya:

Langkah 1:

Pertama, Anda harus masuk ke folder root website Anda. Folder ini biasanya bernama “public_html” atau “www”. Setelah Anda masuk ke folder ini, carilah berkas htaccess. Jika Anda tidak menemukan berkas ini, Anda harus mengaktifkan “Hidden Files” di folder Anda. Setelah Anda menemukan berkas htaccess, Anda dapat mengeditnya.

Langkah 2:

Setelah Anda mengedit berkas htaccess, Anda harus menambahkan baris kode berikut ke berkas tersebut: RewriteEngine On RewriteCond %{REQUEST_URI} \.php$ RewriteRule (.*) $1 [R=301,L] Ini adalah kode yang akan membuat server menyembunyikan PHP dari URL Anda. Setelah Anda menambahkannya, Anda harus menyimpan berkas htaccess Anda.

Langkah 3:

Setelah Anda menyimpan berkas htaccess, Anda harus mereload situs web Anda. Setelah Anda mereload situs web Anda, Anda dapat melihat bahwa PHP tidak lagi muncul di URL Anda. Sekarang Anda sudah berhasil menghapus PHP dari URL Anda.
