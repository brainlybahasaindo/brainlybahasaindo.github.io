Bagaimana Cara Menambahkan Kunci Utama Komposit Di Postgresql?

PostgreSQL adalah salah satu sistem manajemen basis data relasional (RDBMS) yang paling populer dan digunakan oleh banyak perusahaan. Ini adalah sistem manajemen basis data open source yang memungkinkan Anda untuk mengakses, menyimpan, dan memperbarui informasi di dalamnya. PostgreSQL menyediakan berbagai alat untuk mengatur basis data Anda, termasuk penggunaan kunci utama komposit.

Kunci utama komposit adalah kunci yang terdiri dari lebih dari satu kolom. Mereka digunakan untuk mengidentifikasi baris unik dalam tabel dan memastikan bahwa tidak ada baris yang memiliki nilai yang sama di dalamnya. Dengan kata lain, kunci utama komposit adalah cara efisien untuk menjamin integritas data dalam tabel Anda. Dalam tutorial ini, Anda akan belajar bagaimana cara menambahkan kunci utama komposit ke tabel PostgreSQL Anda.

1. Siapkan Tabel Anda

Sebelum Anda dapat menambahkan kunci utama komposit ke tabel Anda, Anda harus membuat tabel itu terlebih dahulu. Anda dapat melakukan ini dengan menggunakan perintah CREATE TABLE. Perintah ini memungkinkan Anda untuk membuat tabel baru dengan kolom yang telah Anda tentukan. Anda juga dapat menambahkan konstrain, seperti NOT NULL, UNIQUE, dan PRIMARY KEY. Konstrain ini membantu menjamin integritas data di dalam tabel Anda.

Sebagai contoh, Anda dapat membuat tabel dengan kolom nama, kelas, dan nilai dengan perintah berikut:

CREATE TABLE mahasiswa (
 nama VARCHAR(50) NOT NULL,
 kelas VARCHAR(20) NOT NULL,
 nilai INT NOT NULL
 );

2. Tambahkan Kolom Kunci Utama

Setelah Anda membuat tabel, Anda dapat menambahkan kolom kunci utama dengan menggunakan perintah ALTER TABLE. Perintah ini memungkinkan Anda untuk mengubah struktur tabel yang telah ada. Anda dapat menggunakan perintah ini untuk menambahkan kolom baru, mengelola hak akses, dan menetapkan konstrain. Dalam contoh ini, Anda akan menggunakan perintah ini untuk menambahkan kolom kunci utama.

Untuk menambahkan kunci utama komposit ke tabel Anda, Anda harus menentukan kolom yang akan menjadi bagian dari kunci utama. Dalam contoh ini, kami akan menggunakan kolom nama dan kelas. Anda dapat menambahkan kolom kunci utama dengan perintah berikut:

ALTER TABLE mahasiswa
 ADD PRIMARY KEY (nama, kelas);

3. Periksa Tabel Anda

Setelah Anda menambahkan kunci utama komposit ke tabel Anda, Anda harus memeriksa tabel untuk memastikan bahwa konstrain telah diterapkan dengan benar. Anda dapat melakukan ini dengan menggunakan perintah DESCRIBE. Perintah ini akan menampilkan informasi tentang tabel, termasuk nama kolom, tipe data, dan konstrain yang diterapkan. Anda dapat menjalankan perintah ini dengan memasukkan nama tabel sebagai parameter.

DESCRIBE mahasiswa;

4. Uji Kunci Utama

Untuk memastikan bahwa kunci utama komposit berfungsi dengan benar, Anda harus menguji tabel dengan menambahkan beberapa baris baru. Anda dapat melakukan ini dengan menggunakan perintah INSERT. Perintah ini memungkinkan Anda untuk menambahkan baris ke tabel. Dalam contoh ini, Anda dapat menambahkan beberapa baris data dengan perintah berikut:

INSERT INTO mahasiswa (nama, kelas, nilai)
 VALUES ('John', 'A', 90),
 ('John', 'B', 85),
 ('Jane', 'A', 95);

5. Periksa Data Anda

Setelah menambahkan data ke tabel, Anda harus memeriksa tabel untuk memastikan bahwa data telah ditambahkan dengan benar. Anda dapat melakukan ini dengan menggunakan perintah SELECT. Perintah ini memungkinkan Anda untuk mengambil baris dari tabel. Dalam contoh ini, Anda dapat menggunakan perintah berikut untuk mengambil semua baris dari tabel:

SELECT * FROM mahasiswa;

Hasil yang dihasilkan harus menampilkan baris data yang telah Anda tambahkan ke tabel. Jika Anda mencoba menambahkan baris yang memiliki nilai yang sama untuk kolom nama dan kelas, maka perintah akan gagal. Ini menunjukkan bahwa kunci utama komposit berfungsi dengan benar dan melindungi integritas data di dalam tabel Anda.

Kesimpulan

Kunci utama komposit adalah cara yang efektif untuk menjamin integritas data di dalam tabel PostgreSQL Anda. Dengan menggunakan kunci utama komposit, Anda dapat memastikan bahwa tidak ada baris yang memiliki nilai yang sama di dalam tabel. Dalam tutorial ini, Anda telah belajar bagaimana cara menambahkan kunci utama komposit ke tabel PostgreSQL Anda. Semoga tutorial ini membantu Anda meningkatkan pengalaman menggunakan PostgreSQL.
