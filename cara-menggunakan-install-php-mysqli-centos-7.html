Cara Menggunakan Install Php Mysqli Di Centos 7

Terbaik Pelajaran bahasa indonesia tips dan trik website . Cari apa saja topik Pelajaran bahasa indonesia ide di website ini.

PHP adalah bahasa pemrograman yang populer saat ini. Banyak situs web menggunakan bahasa ini untuk menampilkan konten. Salah satu alasan utama mengapa bahasa ini begitu populer adalah karena kemudahan instalasi dan konfigurasi yang disediakan. Centos 7 adalah salah satu sistem operasi terkenal yang digunakan untuk menjalankan berbagai jenis aplikasi web. Mari kita lihat bagaimana Anda dapat menggunakan PHP MySQL di Centos 7.

Instalasi PHP MySQL di Centos 7

Pertama, Anda harus menginstal paket yang dibutuhkan di Centos 7. Untuk mengunduh paket, Anda harus menggunakan perintah yum:

yum install php-mysqlnd php-mbstring

Ini akan mengunduh semua paket yang diperlukan untuk menjalankan PHP MySQL di Centos 7. Setelah itu, Anda harus mengaktifkan modul php yang dibutuhkan dengan menggunakan perintah berikut:

phpenmod mysqli mbstring

Ini akan mengaktifkan modul yang Anda butuhkan. Setelah itu, Anda harus menyimpan semua konfigurasi dengan menggunakan perintah berikut:

systemctl restart httpd

Ini akan menyimpan semua konfigurasi yang Anda lakukan sebelumnya dan mengaktifkan PHP MySQL di Centos 7.

Mengatur Akses MySQL di Centos 7

Setelah itu, Anda harus mengatur akses MySQL di Centos 7. Untuk melakukan ini, Anda harus menjalankan perintah berikut:

mysql_secure_installation

Ini akan meminta Anda untuk mengatur kata sandi root. Anda harus menggunakan kata sandi yang aman. Kemudian Anda harus mengatur akses pengguna yang dapat mengakses MySQL. Anda harus menggunakan perintah berikut untuk mengatur akses pengguna:

grant all privileges on *.* to 'username'@'localhost' identified by 'password';

Anda harus mengganti username dengan nama pengguna yang ingin Anda gunakan untuk mengakses MySQL. Anda harus mengganti kata sandi dengan kata sandi yang Anda inginkan. Setelah itu, Anda harus menyimpan semua perubahan yang telah Anda lakukan dengan menggunakan perintah berikut:

flush privileges;

Ini akan menyimpan semua perubahan yang Anda lakukan dan Anda akan dapat mengakses MySQL dengan pengguna yang telah Anda buat.

Menggunakan PHP MySQL di Centos 7

Setelah itu, Anda dapat mulai menggunakan PHP MySQL di Centos 7. Untuk melakukan ini, Anda harus membuat skrip PHP yang akan menghubungkan ke database MySQL. Script ini harus memiliki baris berikut:

$conn = mysqli_connect('hostname','username','password','database');

Anda harus mengganti hostname dengan nama host yang digunakan untuk mengakses MySQL. Anda harus mengganti username dengan nama pengguna yang telah Anda buat sebelumnya. Anda harus mengganti kata sandi dengan kata sandi yang telah Anda buat sebelumnya. Anda harus mengganti database dengan nama database yang ingin Anda gunakan. Setelah itu, Anda harus menjalankan skrip dengan menggunakan perintah berikut:

php script.php

Ini akan menjalankan skrip Anda dan Anda akan dapat menggunakan PHP MySQL di Centos 7.

Menggunakan PHPMyAdmin di Centos 7

Anda juga dapat menggunakan PHPMyAdmin untuk mengelola database MySQL di Centos 7. Anda harus menginstal PHPMyAdmin dengan menggunakan perintah berikut:

yum install phpmyadmin

Ini akan mengunduh semua paket yang diperlukan untuk menjalankan PHPMyAdmin. Setelah itu, Anda harus mengkonfigurasi PHPMyAdmin dengan menggunakan perintah berikut:

phpenmod mysql mysqli mbstring

Ini akan mengkonfigurasi semua modul yang Anda butuhkan untuk menjalankan PHPMyAdmin. Setelah itu, Anda harus menyimpan semua konfigurasi dengan menggunakan perintah berikut:

systemctl restart httpd

Ini akan menyimpan semua konfigurasi yang Anda lakukan sebelumnya dan mengaktifkan PHPMyAdmin di Centos 7. Anda dapat mengakses PHPMyAdmin dengan menggunakan URL http://localhost/phpmyadmin.

Kesimpulan

Anda dapat dengan mudah menggunakan PHP MySQL di Centos 7. Anda harus menginstal paket yang dibutuhkan, mengaktifkan modul yang dibutuhkan, mengatur akses MySQL, dan menggunakan PHPMyAdmin. Ini adalah cara yang mudah dan efisien untuk menggunakan PHP MySQL di Centos 7. Harap diingat bahwa Anda harus menggunakan kata sandi yang aman untuk mengakses MySQL. Jika Anda melakukannya dengan benar, Anda dapat dengan mudah menggunakan PHP MySQL di Centos 7.
