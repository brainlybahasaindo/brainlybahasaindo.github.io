Tutorial Cara Menyusun Daftar Berdasarkan Urutan Dengan Python

Terbaik Pelajaran bahasa indonesia tips dan trik website . Cari apa saja topik Pelajaran bahasa indonesia ide di website ini.

Python merupakan bahasa pemrograman yang banyak digunakan untuk berbagai keperluan. Salah satunya adalah untuk menyusun daftar berdasarkan urutan tertentu. Python memiliki banyak fungsi yang bisa digunakan untuk mempermudah proses pengurutan daftar. Dalam tutorial ini, kami akan menunjukkan cara menggunakan fungsi Python untuk mengurutkan daftar secara efektif.

Apa Itu Pengurutan?

Pengurutan adalah proses menempatkan item atau objek dalam urutan tertentu. Ada beberapa metode pengurutan yang dapat digunakan untuk mengurutkan daftar. Metode pengurutan yang paling umum yaitu pengurutan berdasarkan abjad, angka, tanggal, dan lainnya. Dengan menggunakan fungsi Python, kita bisa membuat kode yang dapat secara efektif mengurutkan daftar berdasarkan kategori yang kita inginkan.

Fungsi Python Untuk Pengurutan Daftar

Python memiliki banyak fungsi yang bisa digunakan untuk mengurutkan daftar. Kita dapat menggunakan fungsi sort() untuk mengurutkan daftar berdasarkan urutan. Fungsi ini dapat menerima argumen seperti daftar, tuple, set, atau dictionary. Kita juga dapat menggunakan fungsi sorted() jika kita ingin mengurutkan daftar dalam urutan tertentu tanpa memodifikasi daftar aslinya. Kita juga dapat menggunakan fungsi reverse() untuk membalikkan urutan daftar.

Menggunakan Fungsi sort()

Fungsi sort() merupakan cara yang paling sederhana untuk mengurutkan daftar. Kita dapat menggunakan fungsi ini untuk mengurutkan daftar berdasarkan abjad, angka, tanggal, dan lainnya. Kita dapat menggunakan fungsi ini dengan menentukan jenis urutan yang kita inginkan. Sebagai contoh, jika kita ingin mengurutkan daftar berdasarkan abjad, kita bisa menggunakan fungsi sort() dengan urutan alfabet.

Menggunakan Fungsi sorted()

Fungsi sorted() merupakan fungsi yang bisa digunakan untuk mengurutkan daftar tanpa perlu memodifikasi daftar aslinya. Fungsi ini mengembalikan daftar yang telah tersusun dalam urutan tertentu. Sebagai contoh, jika kita ingin mengurutkan daftar berdasarkan tanggal, kita dapat menggunakan fungsi sorted() dengan urutan tanggal.

Menggunakan Fungsi reverse()

Fungsi reverse() merupakan fungsi yang dapat digunakan untuk membalikkan urutan daftar. Fungsi ini akan membalikkan daftar dari urutan yang paling awal di daftar ke urutan terakhir di daftar. Sebagai contoh, jika kita ingin membalikkan urutan daftar berdasarkan abjad, kita dapat menggunakan fungsi reverse() untuk mencapai tujuan tersebut.

Contoh Penggunaan Fungsi

Berikut ini adalah contoh penggunaan fungsi Python untuk mengurutkan daftar. Pertama, kita akan mencoba mengurutkan daftar berdasarkan abjad dengan menggunakan fungsi sort(). Berikut adalah contoh kodenya:

daftar = ['Honda','Toyota','Nissan','Mitsubishi']
daftar.sort()
print(daftar)

Berikut adalah outputnya:
['Honda', 'Mitsubishi', 'Nissan', 'Toyota']

Selanjutnya, kita akan mencoba mengurutkan daftar berdasarkan tanggal dengan menggunakan fungsi sorted(). Berikut adalah contoh kodenya:

daftar = [('2021-04-20', 'Beli Mobil'), ('2021-04-21', 'Beli Ban'), ('2021-04-22', 'Beli Pengereman')]
daftarBaru = sorted(daftar, key=lambda x: x[0])
print(daftarBaru)

Berikut adalah outputnya:
[('2021-04-20', 'Beli Mobil'), ('2021-04-21', 'Beli Ban'), ('2021-04-22', 'Beli Pengereman')]

Kesimpulan

Dari tutorial di atas, kita bisa melihat bahwa Python memiliki fungsi yang bisa digunakan untuk mengurutkan daftar. Kita dapat menggunakan fungsi sort() untuk mengurutkan daftar berdasarkan urutan tertentu. Kita juga dapat menggunakan fungsi sorted() dan reverse() untuk mengurutkan dan membalikkan urutan daftar. Dengan menggunakan fungsi Python, kita dapat dengan mudah mengurutkan daftar berdasarkan kategori yang kita inginkan.
