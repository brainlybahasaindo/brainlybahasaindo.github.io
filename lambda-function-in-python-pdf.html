Apa Itu Fungsi Lambda Di Python?

Pada dasarnya, fungsi lambda adalah fungsi yang didefinisikan tanpa nama yang dapat digunakan dalam kode Python Anda. Fungsi ini lebih pendek daripada fungsi biasa yang didefinisikan dengan menggunakan kata kunci def. Fungsi lambda juga dikenal sebagai fungsi anonim, karena mereka tidak memiliki nama. Pada artikel ini, kita akan melihat bagaimana cara menggunakan fungsi lambda di Python dan memahami beberapa contoh dari fungsi lambda.

Mengapa Kita Butuh Fungsi Lambda?

Fungsi lambda tidak hanya berguna untuk membuat kode Anda lebih singkat dan lebih mudah dibaca. Fungsi lambda juga berguna untuk membuat kode Anda lebih fleksibel dan mudah dimodifikasi. Fungsi lambda memungkinkan Anda untuk menggunakan fungsi sebagai parameter dalam fungsi lain. Selain itu, fungsi lambda dapat digunakan sebagai pengembali nilai dari fungsi lain. Dengan kata lain, fungsi lambda memungkinkan Anda untuk menulis kode yang lebih fleksibel dan lebih bisa dipahami.

Bagaimana Cara Membuat Fungsi Lambda?

Fungsi lambda didefinisikan dengan menggunakan kata kunci lambda seperti yang ditunjukkan di bawah ini. Fungsi lambda biasanya memiliki satu argument, tetapi Anda dapat menggunakan lebih dari satu argument. Setelah argument, Anda harus menuliskan ekspresi yang akan menghasilkan nilai yang akan dikembalikan oleh fungsi. Berikut adalah contoh dari fungsi lambda yang dapat digunakan untuk menambahkan dua angka.

lambda x, y: x + y

Apa Fungsi Lambda yang Bisa Digunakan?

Fungsi lambda dapat digunakan untuk menyederhanakan banyak tugas yang berbeda. Fungsi lambda dapat digunakan untuk menghitung nilai yang dikembalikan oleh fungsi lain. Fungsi lambda juga dapat digunakan untuk melakukan operasi pemfilteran, pengurutan, dan banyak lagi. Fungsi lambda juga dapat digunakan untuk menggabungkan dua fungsi menjadi satu. Berikut adalah contoh dari fungsi lambda yang dapat digunakan untuk menggabungkan dua fungsi.

lambda f, g: lambda x: f(g(x))

Contoh Aplikasi Fungsi Lambda

Fungsi lambda sangat berguna untuk menyederhanakan kode yang rumit. Berikut adalah contoh dari fungsi lambda yang dapat digunakan untuk mencari nilai minimum dari suatu daftar. Fungsi ini mengambil daftar sebagai parameter dan mengembalikan nilai minimum dari daftar tersebut.

lambda l: min(l)

Cara Menggunakan Fungsi Lambda di Python PDF

Python memiliki dukungan untuk fungsi lambda yang dapat digunakan untuk membuat kode yang lebih singkat dan lebih mudah dibaca. Fungsi lambda dapat digunakan dalam bentuk yang sama di Python PDF. Berikut adalah contoh dari fungsi lambda yang dapat digunakan untuk menghitung nilai minimum dari daftar.

lambda l: min(l)

Kesimpulan

Fungsi lambda adalah fungsi yang mudah digunakan dan berguna untuk menyederhanakan kode yang rumit. Fungsi ini dapat digunakan untuk menghitung nilai, melakukan operasi pemfilteran, dan banyak lagi. Fungsi ini dapat digunakan di Python PDF dan dapat membuat kode Anda lebih singkat dan lebih mudah dimodifikasi. Dengan demikian, fungsi lambda berguna untuk membuat kode Anda lebih fleksibel dan mudah dipahami.
