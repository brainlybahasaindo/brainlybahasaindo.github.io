Cara Menggunakan Xor Di Python Dengan Contoh

XOR atau eksklusif atau adalah salah satu operator logis yang digunakan untuk menghasilkan output berdasarkan logika boolean. Operator XOR didefinisikan sebagai satu-satunya operator yang dapat menghasilkan hasil TRUE ketika dua masukan yang diberikan bertentangan satu sama lain. Operator XOR merupakan buah dari keluarga operator logika yang lain, yang termasuk operator AND, OR, NOT, dan NAND.

XOR dapat digunakan untuk menyelesaikan banyak masalah logika yang rumit dan sering digunakan dalam berbagai bidang teknik. Dalam Python, XOR dapat digunakan dengan mudah menggunakan operator ^. Python menyederhanakan penggunaan XOR dengan cara yang sama seperti operator logika lainnya. Dalam tutorial ini, kita akan melihat bagaimana cara menggunakan XOR di Python dengan beberapa contoh.

Bagaimana XOR Bekerja?

Sebelum kita melihat contoh cara menggunakan XOR di Python, kita harus melihat bagaimana XOR bekerja. Operator XOR bekerja dengan cara yang sama seperti operator logis lainnya. Namun, bedanya adalah output yang dihasilkan oleh operator XOR hanya TRUE jika kedua masukan yang diberikan bertentangan satu sama lain. Jika kedua masukan yang diberikan sama, maka output adalah FALSE.

Untuk memahami lebih lanjut bagaimana XOR bekerja, mari kita lihat tabel berikut yang menunjukkan output yang dihasilkan oleh operator XOR berdasarkan dua masukan yang diberikan.

Cara Menggunakan XOR di Python

Setelah kita mengerti bagaimana XOR bekerja, mari kita lihat bagaimana cara menggunakan XOR di Python. Di Python, operator XOR disederhanakan dengan menggunakan operator ^. Ini berarti bahwa untuk melakukan operasi XOR, Anda hanya perlu menggunakan operator ^ antara dua masukan. Berikut adalah contoh sederhana bagaimana cara menggunakan operator XOR di Python.

 # Contoh Sederhana Cara Menggunakan XOR di Python
 x = True
 y = False
 
 # Menggunakan Operator XOR (^)
 z = x ^ y
 
 # Output
 print(z)
 # Output: True

Dalam contoh di atas, kita menggunakan operator XOR (^) untuk menghasilkan output berdasarkan logika boolean. Dalam contoh ini, kita menggunakan operator XOR untuk menghasilkan output TRUE dari dua masukan yang bertentangan satu sama lain. Dengan demikian, kita dapat melihat bahwa operasi XOR berhasil dilakukan dengan benar.

Contoh Penerapan XOR di Python

Selain menggunakan operator XOR untuk menghasilkan output berdasarkan logika boolean, operator XOR juga dapat digunakan untuk menyelesaikan banyak masalah logika yang rumit. Berikut adalah contoh sederhana mengenai cara menggunakan XOR untuk menyelesaikan masalah logika yang rumit.

 # Contoh Penerapan XOR di Python
 # Masalah:
 # Apakah salah satu dari dua buah bunga berwarna merah?
 
 bunga1 ='merah'
 bunga2 ='biru'
 
 # Menggunakan Operator XOR (^)
 hasil = (bunga1 =='merah') ^ (bunga2 =='merah')
 
 # Output
 print(hasil)
 # Output: True

Dalam contoh di atas, kita menggunakan operator XOR untuk menyelesaikan masalah logika yang rumit. Masalah yang kita hadapi adalah apakah salah satu dari dua buah bunga berwarna merah. Kita dapat melihat bahwa dengan menggunakan operator XOR, kita dapat dengan mudah menyelesaikan masalah logika yang rumit.
