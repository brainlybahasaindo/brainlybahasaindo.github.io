Persimpangan Python Dari Dua Daftar Dengan Duplikat

Terbaik Pelajaran bahasa indonesia tips dan trik website . Cari apa saja topik Pelajaran bahasa indonesia ide di website ini.

Persimpangan Python adalah suatu teknik yang digunakan untuk menemukan nilai yang sama antara dua daftar. Pada pemrograman, hal ini berguna ketika Anda ingin menemukan nilai-nilai yang muncul dalam kedua daftar. Ini dapat digunakan untuk menyaring data yang dikumpulkan dari berbagai sumber. Dalam artikel ini, kami akan membahas bagaimana menggunakan persimpangan Python dari dua daftar dengan duplikat.

Apa itu Persimpangan?

Persimpangan adalah suatu proses yang menggabungkan dua buah daftar menjadi satu. Hasil persimpangan adalah daftar yang berisi nilai yang muncul dalam kedua daftar yang digabungkan. Misalnya, jika Anda memiliki dua daftar yang berisi angka 1, 2, 3 dan 4, 5, 6, maka hasil persimpangan adalah daftar berisi angka 1, 2, 3, 4, 5, 6.

Cara Menggunakan Persimpangan Python

Python memiliki banyak fungsi yang dapat digunakan untuk menggabungkan dua daftar. Salah satunya adalah fungsi set (). Fungsi ini menghasilkan set dari nilai yang ada di kedua daftar. Setiap nilai hanya akan muncul satu kali di hasil persimpangan. Untuk menggunakan fungsi ini, masukkan dua daftar ke dalam fungsi. Misalnya, jika Anda memiliki dua daftar dengan nama a dan b, maka Anda dapat menggabungkannya menggunakan kode berikut:

c = set(a).intersection(set(b))

Hasil persimpangan akan disimpan dalam variabel c. Anda dapat menggunakan perintah print untuk menampilkan hasilnya. Misalnya, jika Anda ingin menampilkan hasil persimpangan, gunakan perintah berikut:

print(c)

Mengatasi Masalah Duplikat

Ketika menggabungkan dua daftar, Anda mungkin menemukan bahwa ada nilai yang sama muncul dalam kedua daftar. Ini disebut duplikat, dan dapat menyebabkan masalah. Misalnya, jika Anda memiliki dua daftar yang berisi angka 1, 2, 3 dan 1, 4, 5, maka hasil persimpangan akan menjadi daftar berisi angka 1, 2, 3, 4, 5. Fungsi set () tidak menghilangkan nilai duplikat, jadi Anda harus menyelesaikan masalah ini dengan cara lain.

Untuk menghilangkan nilai duplikat, Anda dapat menggunakan fungsi list (). Fungsi ini mengubah set menjadi daftar. Dengan menggunakan fungsi ini, Anda dapat menghilangkan nilai duplikat dengan cara berikut:

c = list(set(a).intersection(set(b)))

Ini akan menghasilkan daftar berisi angka 1, 2, 3, 4, 5 tanpa nilai duplikat.

Menggabungkan Lebih dari Dua Daftar

Python juga memiliki fungsi untuk menggabungkan lebih dari dua daftar sekaligus. Fungsi yang digunakan disebut reduce (), dan dapat digunakan dengan cara berikut:

from functools import reduce

c = reduce(lambda x, y: set(x).intersection(set(y)), [a, b, c, d])

Di sini, Anda harus mengganti variabel a, b, c, dan d dengan nama daftar yang ingin Anda gabungkan. Fungsi ini akan menghasilkan daftar berisi nilai yang muncul dalam kedua daftar.

Kesimpulan

Dalam artikel ini, kami telah membahas bagaimana menggunakan persimpangan Python dari dua daftar dengan duplikat. Kami telah membahas cara menggabungkan dua daftar menggunakan fungsi set () dan bagaimana menghilangkan nilai duplikat dengan fungsi list (). Kami juga telah membahas bagaimana menggabungkan lebih dari dua daftar menggunakan fungsi reduce (). Dengan menggunakan teknik-teknik ini, Anda dapat dengan mudah menggabungkan dua daftar dan menghilangkan nilai duplikat.
