Korelasi Antara Dua Time Series Python

Korelasi adalah hubungan matematis antara dua variabel atau lebih. Dalam dunia data, korelasi memungkinkan Anda untuk menentukan hubungan antara dua time series. Time series adalah urutan data waktu yang membentuk pola dan berkembang. Pada artikel ini, kita akan menggunakan Python untuk menghitung korelasi antara dua time series.

Apa itu Time Series?

Time series adalah himpunan data yang diurutkan berdasarkan waktu. Time series biasanya berbentuk grafik yang menggambarkan perubahan nilai atau data. Data time series dapat terdiri dari data yang diambil secara periodik atau data acak.

Time series dapat digunakan untuk menganalisis berbagai fenomena, seperti cuaca, harga saham, atau pengeluaran konsumen. Data time series juga dapat digunakan untuk memprediksi masa depan.

Apa itu Korelasi?

Korelasi adalah hubungan matematis antara dua variabel atau lebih. Korelasi bisa menunjukkan hubungan positif atau negatif antara variabel. Jika korelasi adalah positif, variabel akan bertambah atau berkurang secara bersamaan. Namun, jika korelasi adalah negatif, satu variabel akan bertambah ketika yang lain berkurang.

Bagaimana Menghitung Korelasi antara Dua Time Series?

Python adalah bahasa yang populer yang banyak digunakan untuk menganalisis time series. Python memiliki banyak pustaka yang dapat digunakan untuk menghitung korelasi antara dua time series.

Salah satu cara untuk menghitung korelasi antara dua time series menggunakan Pandas. Pandas adalah pustaka Python yang banyak digunakan untuk menganalisis data. Dengan Pandas, Anda dapat menghitung korelasi antara dua time series dengan menggunakan metode korelasi pearson.

Kode Python untuk Menghitung Korelasi antara Dua Time Series

Berikut adalah kode Python yang dapat Anda gunakan untuk menghitung korelasi antara dua time series:

import pandas as pd
 
 # Memuat Data Time Series
 ts1 = pd.read_csv('time_series_1.csv')
 ts2 = pd.read_csv('time_series_2.csv')
 
 # Menghitung Korelasi
 correlation = ts1.corrwith(ts2)
 
 # Menampilkan Hasil
 print(correlation)

Kode di atas akan menghitung korelasi antara dua time series dan menampilkan hasilnya. Hasil korelasi akan ditampilkan sebagai nilai antara -1 dan +1. Nilai yang lebih mendekati +1 berarti korelasi positif, sedangkan nilai yang lebih mendekati -1 berarti korelasi negatif.

Kesimpulan

Dengan Python, Anda dapat menghitung korelasi antara dua time series dengan menggunakan Pandas. Dengan Pandas, Anda dapat menghitung korelasi antara dua time series dengan menggunakan metode korelasi pearson.
