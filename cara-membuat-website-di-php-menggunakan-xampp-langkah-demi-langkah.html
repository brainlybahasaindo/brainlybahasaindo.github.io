Cara Membuat Website Di Php Menggunakan Xampp Langkah Demi Langkah

Apa Yang Harus Dilakukan Sebelum Membuat Website di PHP?

Membuat website di PHP tidaklah mudah. Sebelum membuat website, kamu harus mempersiapkan beberapa hal yang harus dilakukan. Hal-hal yang harus dipersiapkan antara lain:

  • Memiliki domain dan hosting
  • Menginstal XAMPP di komputer
  • Membuat database
  • Mengedit kode PHP dan HTML

Dengan mempersiapkan hal-hal di atas, maka kamu sudah siap untuk membuat website di PHP.

Menginstal XAMPP di Komputer

XAMPP adalah sebuah aplikasi open-source yang dibuat oleh Apache Friends yang berfungsi untuk mengatur server lokal di komputer. Aplikasi ini dapat digunakan untuk membuat website di PHP. Untuk menginstalnya, ikuti langkah-langkah berikut ini:

  1. Kunjungi situs resmi XAMPP di https://www.apachefriends.org/download.html dan download versi terbaru.
  2. Klik file .exe yang telah diunduh untuk menginstal aplikasi.
  3. Pilih bahasa yang akan digunakan saat menginstal aplikasi, kemudian klik tombol ‘Next’.
  4. Klik tombol ‘I Agree’ untuk menyetujui Lisensi Penggunaan.
  5. Pilih folder tempat aplikasi akan diinstal, kemudian klik tombol ‘Install’.
  6. Setelah selesai menginstal, klik tombol ‘Finish’ untuk melihat halaman utama XAMPP.

Setelah menginstal XAMPP, maka kamu sudah siap untuk membuat website di PHP.

Membuat Database

Database merupakan tempat untuk menyimpan data dan informasi yang akan ditampilkan di website. Untuk membuat database, kamu dapat menggunakan tool yang tersedia di XAMPP. Berikut ini cara membuat database dengan XAMPP:

  1. Buka halaman utama XAMPP, kemudian klik tombol ‘Admin’ di bagian MySQL.
  2. Kamu akan diarahkan ke halaman login phpMyAdmin. Login dengan menggunakan username dan password yang tersedia.
  3. Klik tab ‘Database’ untuk membuat database baru. Masukkan nama database yang akan dibuat, lalu klik tombol ‘Create’.
  4. Kamu akan melihat daftar tabel di database yang telah dibuat. Klik tab ‘SQL’ untuk membuat tabel baru.
  5. Kamu dapat membuat tabel baru dengan menggunakan query SQL. Masukkan query yang diinginkan, lalu klik tombol ‘Go’.

Setelah selesai membuat database, maka kamu sudah siap untuk membuat website di PHP.

Mengedit Kode PHP dan HTML

Sebelum membuat website, kamu harus mengedit kode PHP dan HTML untuk menampilkan konten yang diinginkan. Kode-kode tersebut dapat diedit dengan menggunakan aplikasi text editor. Aplikasi text editor yang dapat digunakan antara lain Notepad++, Sublime Text, dan Atom. Berikut ini cara mengedit kode PHP dan HTML dengan menggunakan Notepad++:

  1. Buka Notepad++, kemudian buka file yang akan diedit.
  2. Edit kode sesuai dengan kebutuhan.
  3. Simpan file dengan mengklik menu ‘File’ > ‘Save’.
  4. Kamu dapat mengecek hasil edit kode dengan mengklik menu ‘View’ > ‘Browser’.

Setelah selesai mengedit kode, maka kamu sudah siap untuk membuat website di PHP.

Membuat Website di PHP

Setelah mempersiapkan semuanya, kamu dapat mulai membuat website di PHP. Berikut ini cara membuat website di PHP dengan menggunakan XAMPP:

  1. Copy folder website ke dalam folder ‘htdocs’ di XAMPP.
  2. Buka halaman utama XAMPP, kemudian klik tombol ‘Admin’ pada bagian Apache.
  3. Kamu akan diarahkan ke halaman login phpMyAdmin. Login dengan menggunakan username dan password yang tersedia.
  4. Klik tab ‘Database’ untuk membuat database baru. Masukkan nama database yang akan dibuat, lalu klik tombol ‘Create’.
  5. Kamu akan melihat daftar tabel di database yang telah dibuat. Klik tab ‘SQL’ untuk membuat tabel baru.
  6. Kamu dapat membuat tabel baru dengan menggunakan query SQL. Masukkan query yang diinginkan, lalu klik tombol ‘Go’.
  7. Masukkan kode PHP dan HTML yang telah diedit sebelumnya ke dalam folder website.
  8. Kamu dapat mengecek hasil kode dengan mengakses alamat website di browser.

Setelah selesai membuat website, maka kamu sudah berhasil membuat website di PHP.

Kesimpulan

Membuat website di PHP tidaklah sulit jika kamu mengikuti semua langkah yang telah disebutkan di atas. Kamu harus mempersiapkan domain dan hosting, menginstal XAMPP di komputer, membuat database, dan mengedit kode PHP dan HTML. Setelah itu, kamu dapat mulai membuat website di PHP dengan menggunakan XAMPP. Dengan XAMPP, kamu dapat membuat website dengan mudah dan cepat.
