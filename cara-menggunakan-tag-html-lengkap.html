Apa itu Tag HTML?

Tag HTML adalah sebuah kode yang digunakan untuk menandai sebuah teks, gambar, atau element lainnya agar terlihat lebih menarik. Tag HTML juga berfungsi untuk memberi informasi kepada browser tentang bagaimana teks, gambar, atau element lainnya harus ditampilkan. Tag HTML biasanya ditulis dalam dua kurung siku berbeda - misalnya,

. Sintaks dari tag HTML diwakili oleh nama tag, biasanya ditulis dalam huruf kecil, dan kurung siku yang menutup.

Apa Fungsi Tag HTML?

Fungsi dari tag HTML adalah untuk menandai bagian-bagian dari halaman web dan memberi informasi kepada browser tentang bagaimana masing-masing bagian tersebut harus ditampilkan. Tag HTML juga berfungsi sebagai mekanisme untuk mengatur bagaimana teks dan gambar ditampilkan, dan juga untuk membuat link ke halaman web lain atau menyertakan elemen media seperti video dan audio. Tag HTML juga membantu dalam membuat struktur dasar halaman web, sehingga memudahkan pengunjung untuk menavigasi di dalamnya.

Berbagai Jenis Tag HTML

Tag HTML dibagi menjadi dua jenis utama, yaitu tag yang menandai bagian dari halaman web (contohnya , , dan ) dan tag yang menandai bagian-bagian dari isi halaman (contohnya <h1>, <p>, dan <img>). Tag-tag yang menandai isi halaman lebih banyak jenis dan lebih kompleks dibandingkan tag yang menandai struktur dasar halaman web. </p><h2>Tag Heading</h2><p>Tag heading adalah tag HTML yang digunakan untuk menandai judul atau sub-judul dalam halaman web. Terdapat enam jenis tag heading yang berbeda, mulai dari <h1> sampai <h6>. Tag <h1> digunakan untuk judul utama, sedangkan tag <h2> digunakan untuk sub-judul. Tag <h3>, <h4>, <h5>, dan <h6> digunakan untuk menandai tingkat sub-judul yang lebih rendah. </p><h2>Tag Paragraf</h2><p>Tag paragraf adalah tag HTML yang digunakan untuk menandai paragraf teks dalam halaman web. Tag paragraf ditulis dengan <p> dan ditutup dengan </p>. Semua teks yang ditulis di antara tag paragraf akan ditampilkan sebagai paragraf. </p><h2>Tag Link</h2><p>Tag link adalah tag HTML yang digunakan untuk menyertakan link ke halaman web lain di dalam halaman web yang sedang dibuka. Tag link ditulis dengan <a> dan ditutup dengan </a>. Di antara kedua tag tersebut, penulis harus menuliskan URL yang akan dihubungkan, contohnya <a href="http://example.com">Link</a>. </p><h2>Tag Gambar</h2><p>Tag gambar adalah tag HTML yang digunakan untuk menyertakan gambar dalam halaman web. Tag gambar ditulis dengan <img> dan ditutup dengan </img>. Tag gambar juga memerlukan beberapa atribut untuk menandai sumber gambar dan ukurannya, contohnya <img src="example.jpg" width="200" height="200">. </p><h2>Tag List</h2><p>Tag list adalah tag HTML yang digunakan untuk menampilkan daftar berbagai item. Ada dua jenis list yang dapat dibuat dengan tag HTML, yaitu daftar terurut (ordered list) dan daftar tak terurut (unordered list). Tag list ditulis dengan <ul> atau <ol> dan ditutup dengan </ul> atau </ol>. Tag list juga memerlukan tag item list (<li>) untuk menandai masing-masing item dalam daftar. </p><h2>Tag Tabel</h2><p>Tag tabel adalah tag HTML yang digunakan untuk menampilkan data dalam bentuk tabel. Tag tabel ditulis dengan <table> dan ditutup dengan </table>. Setelah itu, penulis harus menyertakan tag baris (<tr>) dan tag kolom (<td>) untuk menandai masing-masing baris dan kolom tabel. </p><h2>Tag Formulir</h2><p>Tag formulir adalah tag HTML yang digunakan untuk membuat formulir interaktif di dalam halaman web. Tag formulir ditulis dengan <form> dan ditutup dengan </form>. Setelah itu, penulis harus menyertakan tag input (<input>) untuk menandai setiap elemen dalam formulir. Tag input juga memerlukan atribut tambahan untuk menentukan jenis input dan nilai default. </p><h2>Tag Audio dan Video</h2><p>Tag audio dan video adalah tag HTML yang digunakan untuk menyertakan media audio dan video di dalam halaman web. Tag audio ditulis dengan <audio> dan ditutup dengan </audio>. Setelah itu, penulis harus menyertakan tag source (<source>) untuk menandai sumber media audio. Tag video juga ditulis dengan <video> dan ditutup dengan </video>.
