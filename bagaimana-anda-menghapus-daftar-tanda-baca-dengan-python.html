Bagaimana Anda Menghapus Daftar Tanda Baca Dengan Python

Terbaik Pelajaran bahasa indonesia tips dan trik website . Cari apa saja topik Pelajaran bahasa indonesia ide di website ini.

Python adalah salah satu bahasa pemrograman populer yang digunakan oleh banyak orang di dunia. Bahasa ini mudah dipelajari karena sintaks-nya yang sederhana serta karena banyaknya sumber daya yang tersedia. Dengan menggunakan Python, Anda dapat menghapus daftar tanda baca dari string atau file teks. Inilah cara melakukannya.

Pengantar

Tanda baca adalah simbol yang digunakan untuk menghubungkan suatu kalimat atau untuk menyatakan pengaturan yang spesifik. Tanda baca biasanya berupa tanda koma, titik, tanda tanya, dan tanda seru. Kebanyakan bahasa pemrograman memiliki fungsi yang dapat membantu Anda memanipulasi string untuk menghapus tanda baca. Python adalah salah satu dari bahasa pemrograman ini. Berikut adalah cara menghapus daftar tanda baca dengan Python.

Menggunakan Fungsi strip ()

Fungsi strip () adalah fungsi bawaan Python yang berguna untuk membuang karakter tertentu dari string. Fungsi ini dapat menerima sebuah karakter atau daftar karakter. Jika Anda tidak menyediakan karakter, maka fungsi akan menghapus whitespace dari string. Untuk menghapus daftar tanda baca, Anda harus menyediakan daftar karakter tanda baca kedalam fungsi ini. Berikut adalah contoh penggunaan fungsi strip ():

string ='Hello, world!'
 string = string.strip ('!,.')
 print (string) #Hasilnya akan menjadi 'Hello world'

Dalam contoh di atas, fungsi strip () akan menghapus semua tanda baca yang tercantum dalam daftar karakter. Jika Anda hanya ingin menghapus koma, Anda hanya perlu menyediakan koma dalam daftar karakter. Contoh berikut akan menghapus semua koma dari string:

string ='Hello, world!'
 string = string.strip (',')
 print (string) #Hasilnya akan menjadi 'Hello world!'

Menggunakan Fungsi replace ()

Fungsi replace () adalah fungsi bawaan Python yang berguna untuk mengganti suatu karakter atau string dengan string lainnya. Fungsi ini dapat menerima sebuah karakter atau string sebagai argumen pertama, dan sebuah string sebagai argumen kedua. Untuk menghapus daftar tanda baca, Anda dapat menggunakan fungsi ini dengan menyediakan daftar tanda baca sebagai argumen pertama dan string kosong ('') sebagai argumen kedua. Berikut adalah contoh penggunaan fungsi replace ():

string ='Hello, world!'
 string = string.replace ('!,.', '')
 print (string) #Hasilnya akan menjadi 'Hello world'

Dalam contoh di atas, fungsi replace () akan mengganti semua tanda baca yang tercantum dalam daftar karakter dengan string kosong. Jika Anda hanya ingin menghapus koma, Anda hanya perlu menyediakan koma dalam daftar karakter. Contoh berikut akan menghapus semua koma dari string:

string ='Hello, world!'
 string = string.replace (',', '')
 print (string) #Hasilnya akan menjadi 'Hello world!'

Menggunakan Regex

Selain fungsi bawaan, Anda juga dapat menggunakan ekspresi reguler (regex) untuk menghapus daftar tanda baca. Regex adalah sebuah bahasa yang digunakan untuk membuat pola string. Anda dapat menggunakan regex untuk menemukan string yang sesuai dengan pola yang ditentukan. Dalam hal ini, Anda dapat menggunakan regex untuk menemukan dan menghapus tanda baca dari string. Berikut adalah contoh penggunaan regex:

import re
 string ='Hello, world!'
 string = re.sub ('[!,.]', '', string)
 print (string) #Hasilnya akan menjadi 'Hello world'

Dalam contoh di atas, fungsi sub () digunakan untuk mengganti semua tanda baca yang tercantum dalam regex dengan string kosong. Jika Anda hanya ingin menghapus koma, Anda hanya perlu menyediakan koma dalam regex. Contoh berikut akan menghapus semua koma dari string:

import re
 string ='Hello, world!'
 string = re.sub (',', '', string)
 print (string) #Hasilnya akan menjadi 'Hello world!'

Kesimpulan

Dengan menggunakan Python, Anda dapat dengan mudah menghapus daftar tanda baca dari string atau file teks. Anda dapat menggunakan fungsi strip (), replace (), atau regex untuk melakukan ini. Semua metode memiliki kelebihan dan kekurangannya masing-masing, jadi Anda harus mempertimbangkan mana yang lebih sesuai dengan kebutuhan Anda. Semoga artikel ini bermanfaat.
