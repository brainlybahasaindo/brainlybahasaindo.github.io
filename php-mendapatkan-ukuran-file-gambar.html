Cara Mendapatkan Ukuran File Gambar Dengan Php

Terbaik Pelajaran bahasa indonesia tips dan trik website . Cari apa saja topik Pelajaran bahasa indonesia ide di website ini.

Salah satu kemampuan yang dimiliki oleh bahasa pemrograman PHP adalah mengetahui ukuran file gambar. Ini berguna ketika Anda mengintegrasikan gambar dalam proyek Anda, karena Anda dapat dengan mudah mengetahui ukuran gambar. Ini juga memungkinkan Anda untuk melakukan manipulasi lebih lanjut, seperti mengubah ukuran gambar atau menghapus file yang berukuran lebih besar dari yang diperlukan.

Berikut adalah cara mendapatkan ukuran file gambar dengan PHP:

1. Buat Fungsi

Untuk memulai, buatlah sebuah fungsi PHP untuk mengambil ukuran gambar. Fungsi ini akan mengambil parameter sebagai nama file gambar, dan mengembalikan ukuran gambar dalam bentuk array. Berikut adalah contoh fungsi:

function getImageSize($filename) { $imageSize = getimagesize($filename); return $imageSize; }

2. Panggil Fungsi

Selanjutnya, Anda dapat memanggil fungsi dengan parameter nama file gambar yang akan Anda dapatkan ukurannya. Berikut adalah contoh yang dapat Anda gunakan:

$imageSize = getImageSize("image.jpg");

3. Ukur File Gambar

Setelah menjalankan fungsi, Anda dapat menggunakan array $imageSize untuk mengambil informasi ukuran gambar. Berikut adalah contoh penggunaan array ini:

$width = $imageSize[0]; $height = $imageSize[1];

Variabel $width akan berisi lebar gambar, sedangkan variabel $height akan berisi tinggi gambar. Anda juga dapat menggunakan variabel lain dari array ini untuk mengambil informasi lebih lanjut tentang gambar, seperti tipe file, dll.

4. Cetak Ukuran Gambar

Setelah menggunakan array $imageSize untuk mengambil informasi ukuran gambar, Anda dapat menggunakan kode PHP berikut untuk mencetak ukuran gambar:

echo "Ukuran gambar: " . $width . "x" . $height;

Ini akan menghasilkan output berikut:

Ukuran gambar: 250x250

5. Ubah Ukuran Gambar

Jika Anda ingin mengubah ukuran gambar, Anda dapat menggunakan fungsi PHP imagecopyresized(). Fungsi ini memiliki parameter yang sama dengan fungsi getImageSize() yang telah Anda buat di atas. Berikut adalah contoh penggunaan fungsi ini:

imagecopyresized($dst_image, $src_image, 0, 0, 0, 0, $dst_width, $dst_height, $src_width, $src_height);

Fungsi ini akan mengubah ukuran gambar dari $src_width x $src_height ke $dst_width x $dst_height. Ini berarti Anda dapat mengubah ukuran gambar dengan mudah dengan menggunakan fungsi ini.

Kesimpulan

Menggunakan PHP, Anda dapat dengan mudah mendapatkan ukuran file gambar dan menggunakan informasi untuk melakukan manipulasi gambar. Ini berguna ketika Anda bekerja dengan gambar di proyek Anda, dan dapat membantu Anda menyelesaikan pekerjaan lebih cepat dan efisien.
