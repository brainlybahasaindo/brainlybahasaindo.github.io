Mengenal &Amp; Menggunakan Datetime Postgresql Di Python

Pemrograman Python dan PostgreSQL merupakan kombinasi yang sangat menarik. Salah satu fitur yang menarik dari PostgreSQL adalah penanganan datetime. Terutama ketika Anda menggunakan bahasa pemrograman Python untuk memanipulasi data Anda. Terutama ketika Anda menggunakan paket Python yang disebut psycopg2, yang menyediakan antarmuka untuk koneksi database PostgreSQL di Python.

Datetime PostgreSQL sangat penting ketika Anda membangun aplikasi berbasis data. Ketika Anda bekerja dengan data yang memiliki tanggal atau waktu, Anda mungkin ingin melakukan berbagai hal. Anda mungkin ingin menggabungkan data berdasarkan tanggal tertentu, mengurutkan data berdasarkan waktu, atau menyaring data berdasarkan jangka waktu tertentu. Semuanya dapat diakses dengan mudah dengan bantuan bahasa pemrograman Python.

Panduan Cepat Datetime PostgreSQL di Python

Untuk memulai, Anda perlu menginstal paket psycopg2 di Python. Anda dapat menggunakan perintah berikut di Command Prompt untuk menginstal paket ini:

python -m pip install psycopg2

Setelah paket diinstal, Anda dapat memulai menggunakan PostgreSQL di Python. Anda dapat menghubungkan ke database Anda dengan perintah berikut:

import psycopg2
conn = psycopg2.connect(&#39;dbname=mydb user=postgres password=mypassword&#39;)

Sekarang, Anda dapat membuat kueri SQL untuk mengambil data dari database Anda. Anda dapat menggunakan kueri berikut untuk mengambil semua data dari tabel Anda yang memiliki kolom datetime:

SELECT * FROM mytable WHERE mydatetime &gt;='2020-01-01'

Mengubah Tipe Data di Python

Ketika Anda mendapatkan data dari PostgreSQL, itu biasanya dalam format string. Jadi jika data Anda berupa tanggal atau jam, Anda mungkin ingin mengubahnya ke tipe data datetime di Python sebelum Anda menggunakannya. Anda dapat menggunakan fungsi datetime.strptime() di Python untuk mengubah string ke tipe data datetime.

Misalnya, jika Anda memiliki string berikut:

'2020-01-01 12:00:00'

Anda dapat mengubahnya menjadi tipe data datetime dengan perintah berikut:

import datetime
mydatetime = datetime.datetime.strptime('2020-01-01 12:00:00', '%Y-%m-%d %H:%M:%S')

Selanjutnya, Anda dapat menggunakan tipe data datetime untuk melakukan berbagai manipulasi data. Misalnya, Anda dapat menambahkan beberapa hari ke tipe data datetime menggunakan fungsi datetime.timedelta():

import datetime
future_datetime = mydatetime + datetime.timedelta(days=7)

Mengubah Tipe Data PostgreSQL

Selain mengubah tipe data di Python, Anda juga dapat mengubah tipe data di PostgreSQL. Misalnya, jika Anda memiliki kolom datetime di PostgreSQL, Anda dapat mengubahnya menjadi tipe data tanggal dengan kueri berikut:

ALTER TABLE mytable ALTER COLUMN mydatetime TYPE date;

Ini akan mengubah kolom datetime menjadi tipe data tanggal. Anda juga dapat mengubah tipe data waktu, jam, dan tipe data lainnya di PostgreSQL.

Fungsi Datetime PostgreSQL

PostgreSQL menyediakan berbagai fungsi untuk memanipulasi data datetime. Misalnya, Anda dapat mengambil tahun, bulan, tanggal, dan jam dari suatu data dengan menggunakan fungsi berikut:

SELECT YEAR(mydatetime) AS year, MONTH(mydatetime) AS month, DAY(mydatetime) AS day, HOUR(mydatetime) AS hour FROM mytable;

Anda juga dapat menambahkan atau mengurangi jangka waktu dari data datetime dengan menggunakan fungsi ADD_MONTHS(), ADD_DAYS(), dan ADD_HOURS().

Kesimpulan

Dalam artikel ini, Anda telah belajar tentang menggunakan datetime PostgreSQL di Python. Anda telah belajar cara menghubungkan ke database PostgreSQL, mengubah tipe data, dan menggunakan fungsi datetime PostgreSQL. Semoga artikel ini membantu Anda memanipulasi data datetime dengan mudah.
