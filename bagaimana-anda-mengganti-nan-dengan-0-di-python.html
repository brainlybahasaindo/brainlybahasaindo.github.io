Bagaimana Anda Mengganti Nan Dengan 0 Di Python

Apa itu Nan?

Nan adalah akronim dari Not a Number. Ini adalah sebuah kata yang digunakan untuk menggambarkan suatu nilai yang tidak dapat diidentifikasi sebagai angka. Nilai ini dapat berupa string, boolean, atau objek. Biasanya, Nan digunakan untuk menandai data yang hilang atau tidak valid. Di Python, Nan dapat ditentukan dengan menggunakan syntax berikut:

np.nan

Mengapa Anda Harus Mengganti Nan Dengan 0?

Karena Nan adalah nilai yang tidak valid, ia akan menimbulkan masalah saat Anda menjalankan operasi aritmatika. Beberapa fungsi bahkan dapat mengembalikan hasil yang tidak valid ketika Nan ditemukan. Misalnya, fungsi np.mean dapat mengembalikan nilai Nan ketika ia diberikan sebagai argumen. Untuk menghindari masalah ini, Anda harus mengganti Nan dengan 0. Dengan mengganti Nan dengan 0, Anda dapat menjamin bahwa semua operasi aritmatika yang Anda lakukan akan berjalan dengan benar.

Bagaimana Cara Mengganti Nan Dengan 0 di Python?

Untuk mengganti Nan dengan 0, Anda dapat menggunakan fungsi pandas.replace. Fungsi ini akan mengambil argumen Nan dan menggantinya dengan nilai yang Anda tentukan. Berikut adalah contoh bagaimana fungsi ini dapat diterapkan pada data:

import pandas as pd
data = pd.DataFrame(['a', np.nan, 'b', np.nan, 'c'])
data.replace(np.nan, 0)

Hasilnya adalah sebagai berikut:

0 a
1 0
2 b
3 0
4 c

Cara Lain Mengganti Nan Dengan 0

Selain menggunakan fungsi pandas.replace, Anda juga dapat menggunakan syntax berikut untuk mengganti Nan dengan 0:

data.fillna(0)

Fungsi ini akan menggantikan semua Nan dengan 0, dan hasilnya akan sama dengan yang dihasilkan oleh pandas.replace. Namun, jika Anda ingin mengganti hanya Nan tertentu, Anda dapat menggunakan syntax berikut:

data.fillna({'col1': 0, 'col2': 0})

Cara Mengganti Nan Dengan Nilai Lain

Sebagian besar teknik yang disebutkan di atas dapat digunakan untuk mengganti Nan dengan nilai lain. Anda hanya perlu mengganti argumen 0 dengan nilai yang diinginkan. Berikut adalah contoh bagaimana Anda dapat mengganti Nan dengan nilai lain:

data.replace(np.nan, -1)
data.fillna({'col1': -1, 'col2': -1})

Kesimpulan

Nan adalah kata yang digunakan untuk menggambarkan suatu nilai yang tidak dapat diidentifikasi sebagai angka. Karena Nan adalah nilai yang tidak valid, ia akan menimbulkan masalah saat Anda menjalankan operasi aritmatika. Untuk menghindari masalah ini, Anda harus mengganti Nan dengan 0. Anda dapat menggunakan fungsi pandas.replace atau syntax data.fillna untuk mengganti Nan dengan 0. Anda juga dapat mengganti Nan dengan nilai lain dengan mengganti argumen 0 dengan nilai yang Anda tentukan.
