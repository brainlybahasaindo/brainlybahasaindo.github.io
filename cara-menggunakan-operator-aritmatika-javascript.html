Cara Menggunakan Operator Aritmatika Javascript

Operator Aritmatika adalah salah satu cara menentukan nilai dalam JavaScript. Operator ini bisa digunakan untuk menghitung berbagai macam operasi matematika seperti penjumlahan, pengurangan, perkalian, pembagian, dan banyak lagi. Ini bisa digunakan untuk membuat kode yang lebih mudah dipahami dan lebih cepat untuk dijalankan. Dalam tutorial ini, kita akan melihat cara menggunakan operator aritmatika JavaScript.

Penjumlahan

Penjumlahan adalah cara paling sederhana untuk menggunakan operator aritmatika. Caranya sangat sederhana, kita cukup menggunakan tanda "+" untuk menjumlahkan dua buah nilai. Sebagai contoh, jika kita ingin menjumlahkan 3 dan 4, kita dapat menggunakan kode berikut:

var jumlah = 3 + 4;

Setelah kode di atas dieksekusi, nilai dari variabel jumlah akan menjadi 7. Penjumlahan juga bisa dilakukan dengan menambahkan nilai dari sebuah variabel. Sebagai contoh, jika kita memiliki variabel bernama "nilai1" dengan nilai 3 dan variabel bernama "nilai2" dengan nilai 4, kita dapat menggunakan kode berikut untuk menjumlahkan kedua nilai tersebut:

var jumlah = nilai1 + nilai2;

Pengurangan

Pengurangan dapat dilakukan dengan menggunakan tanda "-". Ini bisa digunakan untuk mengurangi dua buah nilai atau mengurangi nilai dari sebuah variabel. Sebagai contoh, jika kita ingin mengurangi 3 dari 4, kita dapat menggunakan kode berikut:

var hasil = 4 - 3;

Setelah kode di atas dieksekusi, nilai dari variabel hasil akan menjadi 1. Pengurangan juga bisa dilakukan dengan mengurangi nilai dari sebuah variabel. Sebagai contoh, jika kita memiliki variabel bernama "nilai1" dengan nilai 3 dan variabel bernama "nilai2" dengan nilai 4, kita dapat menggunakan kode berikut untuk mengurangi kedua nilai tersebut:

var hasil = nilai1 - nilai2;

Perkalian

Perkalian dapat dilakukan dengan menggunakan tanda "*". Ini bisa digunakan untuk memperkirakan dua buah nilai atau memperkirakan nilai dari sebuah variabel. Sebagai contoh, jika kita ingin memperkirakan 3 dengan 4, kita dapat menggunakan kode berikut:

var hasil = 3 * 4;

Setelah kode di atas dieksekusi, nilai dari variabel hasil akan menjadi 12. Perkalian juga bisa dilakukan dengan memperkirakan nilai dari sebuah variabel. Sebagai contoh, jika kita memiliki variabel bernama "nilai1" dengan nilai 3 dan variabel bernama "nilai2" dengan nilai 4, kita dapat menggunakan kode berikut untuk memperkirakan kedua nilai tersebut:

var hasil = nilai1 * nilai2;

Pembagian

Pembagian dapat dilakukan dengan menggunakan tanda "/". Ini bisa digunakan untuk membagi dua buah nilai atau membagi nilai dari sebuah variabel. Sebagai contoh, jika kita ingin membagi 3 dengan 4, kita dapat menggunakan kode berikut:

var hasil = 3 / 4;

Setelah kode di atas dieksekusi, nilai dari variabel hasil akan menjadi 0.75. Pembagian juga bisa dilakukan dengan membagi nilai dari sebuah variabel. Sebagai contoh, jika kita memiliki variabel bernama "nilai1" dengan nilai 3 dan variabel bernama "nilai2" dengan nilai 4, kita dapat menggunakan kode berikut untuk membagi kedua nilai tersebut:

var hasil = nilai1 / nilai2;

Modulus

Modulus dapat dilakukan dengan menggunakan tanda "%". Ini bisa digunakan untuk menghitung sisa hasil pembagian dari dua buah nilai atau menghitung sisa hasil pembagian dari sebuah variabel. Sebagai contoh, jika kita ingin menghitung sisa hasil pembagian 3 dari 4, kita dapat menggunakan kode berikut:

var hasil = 3 % 4;

Setelah kode di atas dieksekusi, nilai dari variabel hasil akan menjadi 3. Modulus juga bisa dilakukan dengan menghitung sisa hasil pembagian nilai dari sebuah variabel. Sebagai contoh, jika kita memiliki variabel bernama "nilai1" dengan nilai 3 dan variabel bernama "nilai2" dengan nilai 4, kita dapat menggunakan kode berikut untuk menghitung sisa hasil pembagian kedua nilai tersebut:

var hasil = nilai1 % nilai2;

Kesimpulan

Dengan mengetahui cara menggunakan operator aritmatika JavaScript, kita dapat membuat kode yang lebih mudah dipahami dan lebih cepat untuk dijalankan. Operator aritmatika JavaScript dapat digunakan untuk menjumlahkan, mengurangi, memperkirakan, membagi, dan menghitung sisa hasil pembagian dua buah nilai atau nilai dari sebuah variabel. Sekarang kamu sudah tahu bagaimana cara menggunakan operator aritmatika JavaScript!
