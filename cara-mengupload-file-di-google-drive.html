Cara Mudah Dan Cepat Mengupload File Ke Google Drive

Terbaik Pelajaran bahasa indonesia tips dan trik website . Cari apa saja topik Pelajaran bahasa indonesia ide di website ini.

Google Drive merupakan layanan cloud storage gratis yang disediakan oleh Google. Dengan Google Drive, kamu bisa menyimpan file-file penting, seperti dokumen, gambar, video, dan lain-lain. Selain itu, file-file yang tersimpan di Google Drive bisa diakses dari berbagai perangkat, baik melalui komputer, laptop, ataupun smartphone.

Untuk menggunakan Google Drive, kamu harus terlebih dahulu membuat akun Google (Google Account). Jika sudah punya akun Google, maka kamu bisa memulai mengupload file ke Google Drive. Berikut ini cara mengupload file ke Google Drive yang bisa kamu lakukan.

1. Login ke Akun Google

Pertama, buka browser kamu dan masuk ke halaman google.com. Setelah itu, klik tombol "Masuk" di pojok kanan atas. Akan muncul kolom untuk memasukkan alamat email atau nomor telepon dan password. Isi kolom tersebut dengan data akun Google yang kamu miliki. Jika sudah, klik tombol "Masuk" untuk login.

2. Buka Google Drive

Setelah kamu berhasil login ke akun Google, kamu bisa langsung membuka Google Drive. Caranya, klik ikon aplikasi Google di pojok kiri atas layar. Setelah itu, pilih aplikasi Google Drive. Akan muncul halaman Google Drive yang bisa kamu gunakan untuk mengupload file.

3. Klik Tombol Upload

Di halaman Google Drive, kamu akan melihat sebuah tombol berwarna hijau bertuliskan "Unggah" atau "Upload" (tergantung bahasa yang kamu gunakan). Klik tombol tersebut untuk mulai mengupload file. Akan muncul jendela pop up untuk memilih file yang akan diupload.

4. Pilih File yang Ingin di Upload

Di jendela pop up, kamu bisa memilih file yang akan diupload. File yang bisa diupload adalah file gambar, video, dokumen, dan lain-lain. Setelah itu, klik tombol "Buka" untuk mengirimkan file yang telah dipilih ke Google Drive. Tunggu beberapa saat hingga proses upload selesai.

5. Lihat File yang Sudah di Upload

Setelah proses upload selesai, maka file yang telah kamu upload akan muncul di halaman Google Drive. Di sini, kamu bisa melihat file-file yang telah tersimpan di Google Drive. Jika ingin membuka file, kamu bisa klik file tersebut dan kamu akan diarahkan ke halaman selanjutnya.

6. Bagikan File ke Orang Lain

Jika ingin berbagi file yang telah kamu upload ke Google Drive, kamu bisa menggunakan fitur bagikan. Caranya, klik file yang ingin kamu bagikan ke orang lain. Setelah itu, klik ikon titik tiga di pojok kanan atas layar. Akan muncul menu "Bagikan", klik menu tersebut. Akan muncul jendela pop up, masukkan alamat email orang yang ingin kamu bagikan file, lalu klik tombol "Kirim". File yang telah kamu bagikan akan dikirim ke alamat email orang yang telah kamu masukkan.

7. Gunakan Link Publik

Selain mengirim file lewat email, kamu juga bisa menggunakan link publik untuk berbagi file. Caranya, klik file yang ingin kamu bagikan, lalu klik ikon titik tiga di pojok kanan atas layar. Akan muncul menu "Bagikan", klik menu tersebut. Akan muncul jendela pop up, pastikan opsi "Bagikan" diaktifkan, lalu klik tombol "Obrolan". Setelah itu, klik tombol "Tautan Saya". Akan muncul link publik yang bisa kamu gunakan untuk berbagi file ke orang lain.

8. Simpan Link Publik

Setelah mendapatkan link publik yang bisa kamu gunakan untuk berbagi file, kamu bisa menyimpan link tersebut. Copy link tersebut dan simpan di mana pun, seperti di catatan teks, dokumen, ataupun di aplikasi notes. Setelah itu, kamu bisa berbagi link tersebut ke orang lain melalui chat, email, ataupun sosial media. Orang lain yang mengakses link tersebut bisa langsung mengunduh file yang telah kamu bagikan.

9. Set Permission

Selain itu, kamu juga bisa mengatur permission file yang kamu bagikan. Caranya, pilih file yang ingin kamu bagikan, lalu klik ikon titik tiga di pojok kanan atas layar. Akan muncul menu "Bagikan", klik menu tersebut. Akan muncul jendela pop up, klik tombol "Perizinan". Di sini, kamu bisa memilih permission yang akan diberikan ke orang lain yang membuka file yang kamu bagikan. Permission yang bisa kamu berikan adalah "Bisa Mengunduh", "Bisa Mengedit", ataupun "Bisa Menyunting". Pilih permission yang diinginkan dan klik tombol "Simpan".

10. Hapus File yang Sudah Di Bagi

Jika sudah tidak diperlukan lagi, kamu bisa menghapus file yang telah kamu bagikan. Caranya, pilih file yang ingin kamu hapus, lalu klik ikon titik tiga di pojok kanan atas layar. Akan muncul menu "Bagikan", klik menu tersebut. Akan muncul jendela pop up, klik tombol "Perizinan". Di sini, kamu bisa menghapus permission yang diberikan ke orang lain. Pilih orang yang ingin kamu hapus permissionnya, lalu klik tombol "Hapus" di pojok kanan bawah. Setelah itu, klik tombol "Simpan" untuk menyimpan perubahan.

Itulah cara mengupload file ke Google Drive yang bisa kamu lakukan. Dengan menggunakan Google Drive, kamu bisa dengan mudah dan cepat menyimpan file penting dan berbagi file ke orang lain. Selamat mencoba!
