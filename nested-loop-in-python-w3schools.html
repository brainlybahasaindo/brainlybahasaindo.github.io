Apa itu Nested Loop?

Nested Loop adalah pengulangan dalam pengulangan. Dengan begitu, kita bisa melakukan pengulangan pada struktur data yang telah dibuat. Di Python, kita bisa menggunakan nested loop untuk mengulangi suatu variabel sebanyak yang diinginkan. Nested loop bisa membantu dalam penyelesaian berbagai masalah.

Penggunaan Nested Loop di Python

Untuk menggunakan nested loop, kita bisa menggunakan syntax berikut: 

for x in range(a,b):
 for y in range(c,d):
 # code
Di syntax di atas, x adalah variabel yang akan diulangi sebanyak jumlah yang ditentukan di antara a dan b, sementara y adalah variabel yang akan diulangi sebanyak jumlah yang ditentukan di antara c dan d. Kode yang ada di dalam loop akan diulangi berkali-kali sampai semua variabel selesai.

Contoh Penggunaan Nested Loop di Python

Berikut adalah contoh penggunaan nested loop di Python: 

for i in range(1,11):
 for j in range(1,11):
 print(i*j)
Di contoh di atas, kita akan melakukan loop pada variabel i antara 1 sampai dengan 11, dan untuk setiap loop kita akan melakukan loop lagi pada variabel j antara 1 sampai 11. Setelah itu, kita akan mencetak hasil dari perkalian kedua variabel tersebut. Sehingga, jika loop selesai, kita akan memiliki hasil dari perkalian 1 sampai 10.

Keuntungan Menggunakan Nested Loop di Python

Nested loop bisa memberikan keuntungan dalam banyak situasi. Misalnya, dengan membuat nested loop, kita bisa dengan mudah melakukan pengulangan pada struktur data. Selain itu, nested loop juga bisa membantu dalam penyelesaian berbagai masalah. Dengan nested loop, kita bisa dengan mudah membuat kode yang lebih efisien dan lebih mudah dibaca.
