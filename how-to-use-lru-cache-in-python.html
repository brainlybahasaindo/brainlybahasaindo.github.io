Cara Menggunakan Lru Cache Di Python

Apa itu LRU Cache?

LRU Cache adalah algoritma pengganti memori yang digunakan untuk mengurangi kompleksitas waktu pencarian data. LRU Cache menggunakan prinsip Least Recently Used (LRU) untuk menentukan data mana yang harus dikeluarkan dari cache ketika cache penuh. Prinsip LRU ini berarti bahwa data yang paling lama tidak digunakan akan dikeluarkan dari cache. Ini berarti bahwa ketika data baru ingin dimasukkan ke cache, data yang sudah ada akan dikeluarkan dari sana.

Kenapa LRU Cache Berguna

LRU Cache sangat berguna karena memungkinkan aplikasi untuk mengembalikan data yang diinginkan tanpa harus melakukan query ke basis data. Hal ini membuat aplikasi lebih cepat dan responsif. Selain itu, jika seseorang perlu mengakses data yang sudah ada, ia tidak perlu mengirimkan query ke basis data lagi. Dengan demikian, dapat mengurangi beban pada server basis data dan meningkatkan kinerja aplikasi.

Bagaimana Cara Menggunakan LRU Cache di Python?

LRU Cache dapat dengan mudah diimplementasikan di Python dengan menggunakan library bawaan Python yaitu collections. Untuk menggunakannya, Anda harus mengimport library collections. Anda dapat melakukan ini dengan cara berikut:

import collections

Anda juga perlu menentukan jumlah elemen yang dapat disimpan dalam cache. Ini dapat dilakukan dengan menggunakan fungsi LRU Cache sebagai berikut:

my_cache = collections.LRU_Cache(maxsize = 100)

Di sini, maxsize adalah jumlah elemen yang dapat disimpan dalam cache. Dengan menggunakan fungsi ini, Anda dapat mengatur ukuran maksimum cache Anda.

Mengakses Data dalam LRU Cache

Untuk mengakses data dalam cache, Anda dapat menggunakan metode get () yang disediakan oleh collections. Ini akan mengembalikan nilai data jika data tersebut ada di dalam cache, atau None jika tidak ada. Contohnya, jika Anda ingin mengakses data dengan kunci 'foo', Anda dapat menggunakan metode get () sebagai berikut:

data = my_cache.get('foo')

Memasukkan Data ke cache

Untuk memasukkan data ke dalam cache, Anda harus menggunakan metode set () yang disediakan oleh collections. Metode ini mengambil dua argumen, yaitu kunci dan nilai. Ini akan menambahkan data baru ke cache atau memperbarui nilai data yang sudah ada. Contohnya, jika Anda ingin memasukkan data dengan kunci 'foo' dan nilai 'bar', Anda dapat menggunakan metode set () sebagai berikut:

my_cache.set('foo', 'bar')

Menghapus Data dari Cache

Untuk menghapus data dari cache, Anda dapat menggunakan metode delete () yang disediakan oleh collections. Metode ini mengambil satu argumen yaitu kunci. Ini akan menghapus entri data yang ditentukan dari cache. Contohnya, jika Anda ingin menghapus data dengan kunci 'foo', Anda dapat menggunakan metode delete () sebagai berikut:

my_cache.delete('foo')

Mengosongkan Cache

Untuk mengosongkan cache, Anda dapat menggunakan metode clear () yang disediakan oleh collections. Metode ini tidak membutuhkan argumen. Ini akan menghapus semua entri data dari cache. Contohnya, jika Anda ingin mengosongkan cache, Anda dapat menggunakan metode clear () sebagai berikut:

my_cache.clear()

Kesimpulan
