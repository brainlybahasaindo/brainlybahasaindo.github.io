Cara Membuat Situs Wordpress Offline

Terbaik Pelajaran bahasa indonesia tips dan trik website . Cari apa saja topik Pelajaran bahasa indonesia ide di website ini.

Berikut adalah cara mudah membuat situs WordPress offline. Dengan cara ini, Anda dapat menonaktifkan situs WordPress Anda secara sementara tanpa harus menghapus semua konten di dalamnya. Proses ini juga dapat membantu Anda menyembunyikan situs dari pengunjung dan mengurangi beban pemrosesan server.

Langkah-Langkah Membuat Situs WordPress Offline

Langkah 1: Masuk Ke Akun Hosting Anda

Langkah pertama adalah masuk ke akun hosting Anda. Setelah Anda masuk, Anda akan melihat beberapa pilihan yang bisa Anda gunakan. Cari dan klik tombol yang bertuliskan 'Cpanel'. Anda harus menggunakan pilihan ini untuk membuat situs WordPress Anda offline.

Langkah 2: Masuk Ke Cpanel

Setelah Anda masuk ke Cpanel, Anda akan melihat berbagai pilihan yang bisa Anda gunakan. Cari dan klik pilihan yang bertuliskan 'File Manager'. Pilihan tersebut dapat Anda temukan di sebelah kanan layar.

Langkah 3: Masuk Ke File Manager

Setelah Anda masuk ke File Manager, Anda harus mencari file yang bernama '.htaccess'. File ini berada di dalam folder utama situs WordPress Anda. Setelah Anda menemukan file tersebut, Anda bisa langsung membukanya.

Langkah 4: Tambahkan Kode Ke File .htaccess

Setelah Anda membuka file '.htaccess', Anda bisa langsung menambahkan beberapa baris kode. Kode yang Anda tambahkan harus sesuai dengan yang dituliskan di bawah ini:

Options -Indexes
ErrorDocument 403 /maintenance.html

Setelah Anda menambahkan kode tersebut, Anda harus menyimpan file '.htaccess' dengan menekan tombol 'Save'.

Langkah 5: Buat File Maintenance.html

Langkah selanjutnya adalah membuat file baru yang bernama 'maintenance.html'. Anda bisa membuat file tersebut di dalam folder utama situs WordPress Anda. File ini akan menampilkan pesan yang akan diberikan kepada pengunjung ketika situs Anda dalam mode offline.

Langkah 6: Tambahkan Konten Ke File Maintenance.html

Setelah Anda membuat file 'maintenance.html', Anda harus menambahkan beberapa konten di dalamnya. Anda bisa menambahkan pesan atau gambar untuk menampilkan informasi kepada pengunjung. Misalnya, Anda bisa menambahkan pesan berikut:

Maaf, situs ini sedang dalam perbaikan. Silahkan coba lagi nanti. Terima kasih.

Anda juga bisa menambahkan gambar yang menyatakan bahwa situs Anda sedang dalam perbaikan.

Langkah 7: Simpan File Maintenance.html

Setelah Anda menambahkan konten ke dalam file 'maintenance.html', Anda harus menyimpan file tersebut dengan menekan tombol 'Save'. Setelah Anda menyimpan file tersebut, maka situs WordPress Anda akan secara otomatis berada dalam mode offline.

Langkah 8: Aktifkan Kembali Situs WordPress Anda

Setelah Anda selesai melakukan perbaikan situs WordPress Anda, Anda bisa mengaktifkan kembali situs Anda dengan cara menghapus file 'maintenance.html'. Anda juga bisa menghapus kode yang ada di file '.htaccess' agar situs Anda dapat kembali online.

Kesimpulan

Membuat situs WordPress Anda offline adalah hal yang cukup mudah. Anda hanya perlu mengikuti langkah-langkah yang disebutkan di atas. Proses ini dapat membantu Anda untuk menyembunyikan situs Anda dari pengunjung dan mengurangi beban pemrosesan server. Jadi, jika Anda ingin membuat situs WordPress Anda offline, Anda bisa mengikuti langkah-langkah di atas.
