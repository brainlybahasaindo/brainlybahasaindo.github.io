HTML (Hypertext Markup Language) adalah bahasa markup yang digunakan untuk mengkode halaman web. Tag HTML digunakan untuk menandai bagian-bagian tertentu dari halaman web, dengan menyertakan konten dalam elemen. Salah satu tag yang paling umum adalah tag div. Tag div adalah tag dasar yang digunakan untuk membagi konten pada halaman web.

Apa Fungsi Tag Div di HTML?

Fungsi utama tag div adalah untuk membagi konten pada halaman web. Tag div digunakan untuk membuat struktur konten yang lebih rapi dan mudah untuk dibaca. Tag div juga dapat digunakan untuk menambahkan gaya dan efek, seperti menetapkan warna latar belakang, tata letak teks, dan gaya warna, untuk meningkatkan tampilan halaman web. Tag div juga dapat digunakan untuk menambahkan konten yang tersembunyi dan membuat elemen yang dapat disesuaikan.

Apa Itu Tag Span?
