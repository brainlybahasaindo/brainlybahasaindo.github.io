Cara Menyalin Tabel Dari Html Ke Excel

Pengenalan

Ketika kita sedang membuat laporan atau presentasi, kita biasanya menggunakan data dalam bentuk tabel. Excel adalah salah satu aplikasi yang dapat digunakan untuk membuat tabel. Namun, jika kita memiliki tabel dalam format HTML, maka kita harus menyalin tabel tersebut ke dalam Excel agar dapat diubah sesuai kebutuhan. Pada artikel ini, kita akan membahas tentang bagaimana cara menyalin tabel dari HTML ke Excel.

Cara Menyalin Tabel HTML ke Excel

Ada beberapa cara yang dapat digunakan untuk menyalin tabel HTML ke Excel. Berikut adalah cara-caranya:

Cara 1: Menyalin dan Tempel Tabel

Untuk melakukan cara ini, kita dapat menyalin tabel HTML dengan menggunakan kombinasi tombol Ctrl + C (untuk Windows) atau Command + C (untuk Mac). Kemudian, buka aplikasi Excel dan tempel tabel tersebut dengan menggunakan kombinasi tombol Ctrl + V (untuk Windows) atau Command + V (untuk Mac). Selesai.

Cara 2: Menggunakan Fitur “Paste Special”

Fitur “Paste Special” juga dapat digunakan untuk menyalin tabel HTML ke Excel. Pertama, kita harus menyalin tabel HTML dengan menggunakan kombinasi tombol Ctrl + C (untuk Windows) atau Command + C (untuk Mac). Setelah itu, buka aplikasi Excel dan tempel tabel tersebut dengan menggunakan kombinasi tombol Ctrl + V (untuk Windows) atau Command + V (untuk Mac). Lalu, klik menu “Edit”, kemudian pilih “Paste Special”. Setelah itu, pilih opsi “HTML” dan klik tombol “OK”. Selesai.

Cara 3: Menggunakan Fitur “Data”

Selain menggunakan fitur “Paste Special”, kita juga dapat menggunakan fitur “Data” untuk menyalin tabel HTML ke Excel. Untuk melakukan cara ini, kita harus menyalin tabel HTML dengan menggunakan kombinasi tombol Ctrl + C (untuk Windows) atau Command + C (untuk Mac). Setelah itu, buka aplikasi Excel dan pilih tab “Data”. Kemudian, klik tombol “From Web” dan masukkan URL tabel HTML yang akan kita salin. Setelah itu klik tombol “Import” dan pilih opsi “Table”, kemudian klik tombol “OK”. Selesai.

Cara 4: Menggunakan Fitur “Web Query”

Fitur “Web Query” juga dapat digunakan untuk menyalin tabel HTML ke Excel. Untuk melakukan cara ini, kita harus menyalin URL tabel HTML yang akan kita salin. Setelah itu, buka aplikasi Excel dan pilih tab “Data”. Kemudian, klik tombol “From Web” dan masukkan URL tabel HTML yang tadi telah kita salin. Setelah itu, klik tombol “Import” dan pilih opsi “Table”, kemudian klik tombol “OK”. Selesai.

Kesimpulan

Mengubah tabel dari HTML ke Excel memang dapat dilakukan dengan berbagai cara. Namun, cara yang paling mudah adalah dengan menggunakan fitur “Paste Special” atau “Data”. Selain itu, kita juga dapat menggunakan fitur “Web Query” untuk menyalin tabel HTML ke Excel. Dengan mengetahui cara-cara di atas, kita dapat dengan mudah menyalin tabel HTML ke Excel tanpa harus membuat tabel dari awal.
