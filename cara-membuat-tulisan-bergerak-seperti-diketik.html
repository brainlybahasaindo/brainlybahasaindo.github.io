Tahun 2023, teknologi berkembang sangat pesat. Bahkan, sekarang, Anda dapat membuat tulisan bergerak seperti diketik di layar Anda. Ini adalah teknik yang terkenal di antara para pemrogram. Cara membuat tulisan bergerak seperti diketik adalah dengan menggunakan kode HTML dan JavaScript. Berikut adalah panduan langkah demi langkah untuk membuat tulisan bergerak seperti diketik.

Langkah 1: Buat File HTML

Pertama-tama, Anda harus membuat file HTML untuk menampilkan tulisan bergerak seperti diketik. Buat sebuah file HTML dan beri nama "index.html". Dalam file ini, tulis kode HTML dasar. Anda dapat menggunakan kode berikut:

Langkah 2: Buat File JavaScript

Setelah membuat file HTML, Anda harus membuat file JavaScript untuk mengontrol proses mengetik. Buat sebuah file JavaScript dan beri nama "script.js". Dalam file ini, tulis kode JavaScript untuk mengetik tulisan secara otomatis. Anda dapat menggunakan kode berikut: 

var i = 0;
 var speed = 25;
 var text ="Halo, ini adalah contoh tulisan yang bergerak seperti diketik.";
 
 function typeWriter() {
 if (i < text.length) {
 document.getElementById("text").innerHTML += text.charAt(i);

 setTimeout(typeWriter, speed);

 }

Langkah 5: Tambahkan Estetika

Selain menampilkan tulisan bergerak seperti diketik, Anda juga dapat menambahkan estetika. Anda dapat menggunakan kode CSS untuk mengubah warna, font, dan gaya tulisan. Anda juga dapat menggunakan efek animasi untuk menambahkan estetika.
