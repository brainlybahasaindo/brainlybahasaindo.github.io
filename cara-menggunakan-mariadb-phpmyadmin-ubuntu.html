Apa Itu Mariadb, Phpmyadmin Dan Ubuntu?

Terbaik Pelajaran bahasa indonesia tips dan trik website . Cari apa saja topik Pelajaran bahasa indonesia ide di website ini.

Mariadb, Phpmyadmin dan Ubuntu merupakan tiga perangkat lunak yang digunakan untuk membangun dan mengelola basis data. Mariadb merupakan sebuah sistem manajemen basis data relasional (RDBMS) yang dikembangkan oleh MariaDB Corporation AB. Ini adalah versi yang ditingkatkan dari MySQL, yang merupakan basis data open source yang populer. PhpMyAdmin adalah sebuah aplikasi web open source yang memungkinkan Anda mengelola basis data MySQL melalui antarmuka grafis. Ubuntu adalah sistem operasi Linux open source yang dipromosikan oleh Canonical Ltd. Ini adalah sistem operasi yang populer untuk server dan desktop yang digunakan oleh banyak organisasi dan individu. Ini adalah platform yang kuat dan fleksibel yang menyediakan segala hal yang Anda butuhkan untuk mengelola basis data Anda.

Cara Menginstal Mariadb dan Phpmyadmin di Ubuntu

Sebelum Anda dapat menggunakan Mariadb dan PhpMyAdmin di Ubuntu, Anda harus menginstal keduanya. Mariadb dan PhpMyAdmin dapat dengan mudah diinstal dengan menggunakan perintah apt. Langkah-langkah yang perlu Anda ikuti untuk menginstal Mariadb dan PhpMyAdmin di Ubuntu adalah sebagai berikut:

Langkah 1: Memperbarui Repositori

Pertama, Anda perlu memperbarui repositori yang tersedia di Ubuntu. Ini dapat dilakukan dengan menjalankan perintah berikut di terminal:

$ sudo apt-get update

Langkah 2: Menginstal Mariadb

Setelah Anda selesai memperbarui repositori, Anda dapat menginstal Mariadb dengan menjalankan perintah berikut di terminal:

$ sudo apt-get install mariadb-server

Langkah 3: Konfigurasi Mariadb

Setelah Mariadb berhasil diinstal, Anda dapat melakukan konfigurasi yang diperlukan dengan menjalankan perintah berikut di terminal:

$ sudo mysql_secure_installation

Langkah 4: Menginstal PhpMyAdmin

Setelah Mariadb berhasil dikonfigurasi dengan benar, Anda dapat melanjutkan dengan menginstal PhpMyAdmin dengan menjalankan perintah berikut di terminal:

$ sudo apt-get install phpmyadmin

Langkah 5: Konfigurasi PhpMyAdmin

Setelah PhpMyAdmin berhasil diinstal, Anda perlu melakukan konfigurasi yang diperlukan dengan menjalankan perintah berikut di terminal:

$ sudo php5enmod mcrypt

$ sudo service apache2 restart

Langkah 6: Mengakses PhpMyAdmin

Setelah PhpMyAdmin berhasil dikonfigurasi, Anda dapat melakukan akses ke antarmuka web PhpMyAdmin dengan mengunjungi alamat berikut di browser web Anda:

http://localhost/phpmyadmin

Cara Menggunakan Mariadb dan PhpMyAdmin di Ubuntu

Setelah Anda berhasil menginstal Mariadb dan PhpMyAdmin di Ubuntu, Anda dapat mulai menggunakannya. Mariadb dan PhpMyAdmin dapat dengan mudah digunakan dengan menggunakan antarmuka web PhpMyAdmin. Berikut adalah beberapa cara untuk menggunakannya:

Membuat Database

Anda dapat membuat database baru dengan mengakses antarmuka web PhpMyAdmin. Anda hanya perlu mengklik tombol “Create database” di antarmuka web dan memasukkan nama database yang diinginkan. Setelah itu, database baru akan dibuat.

Menambahkan Tabel

Setelah database baru dibuat, Anda dapat mulai menambahkan tabel baru. Anda dapat melakukannya dengan mengklik tombol “Create table” di antarmuka web PhpMyAdmin. Anda akan diminta untuk memasukkan nama tabel dan jumlah kolom. Setelah itu, tabel akan dibuat.

Menambahkan Data

Setelah tabel baru dibuat, Anda dapat mulai menambahkan data ke tabel. Anda dapat melakukannya dengan mengklik tombol “Insert” di antarmuka web PhpMyAdmin. Anda dapat memasukkan data yang diinginkan ke dalam tabel. Setelah itu, data akan disimpan.

Menjalankan Query

Anda juga dapat menjalankan query SQL di antarmuka web PhpMyAdmin. Anda hanya perlu menuliskan query di jendela query dan mengklik tombol “Go”. Query akan segera dijalankan dan hasilnya akan ditampilkan dalam bentuk tabel.

Menyimpan Perubahan

Setelah Anda selesai menjalankan query atau melakukan perubahan pada data, Anda perlu menyimpan perubahan yang Anda lakukan. Anda dapat melakukannya dengan mengklik tombol “Save” di antarmuka web PhpMyAdmin. Perubahan akan segera disimpan ke dalam basis data.

Kesimpulan

Mariadb, Phpmyadmin dan Ubuntu adalah tiga perangkat lunak yang digunakan untuk membangun dan mengelola basis data. Anda dapat dengan mudah menginstal dan menggunakan ketiga perangkat lunak ini di Ubuntu dengan menggunakan perintah apt di terminal. Anda dapat mengelola basis data dengan mudah menggunakan antarmuka web PhpMyAdmin. Dengan menggunakan antarmuka web ini, Anda dapat melakukan berbagai macam operasi seperti membuat database, menambahkan tabel, menambahkan data, menjalankan query, dan menyimpan perubahan.
