Cara Mudah Merubah Tanggal Iso Ke Tanggal Normal Di Javascript

Terbaik Pelajaran bahasa indonesia tips dan trik website . Cari apa saja topik Pelajaran bahasa indonesia ide di website ini.

Tahun 2023 ini, format tanggal ISO sudah menjadi standar untuk tanggal di seluruh dunia. Format tanggal ISO adalah format tanggal yang dituliskan dalam bentuk YYYY-MM-DD atau YYYYMMDD. Format ini sangat mudah dimengerti oleh mesin, namun agak sulit untuk dipahami oleh manusia. Oleh karena itu, penting untuk mengetahui cara merubah tanggal ISO menjadi tanggal yang lebih mudah dimengerti.

Dalam bahasa JavaScript, ada beberapa cara mudah yang bisa Anda gunakan untuk merubah tanggal ISO menjadi tanggal normal. Anda bisa menggunakan metode toLocaleDateString() untuk mengubah tanggal ISO menjadi tanggal yang dapat dibaca oleh manusia. Metode ini juga bisa digunakan untuk merubah tanggal ISO menjadi format tanggal yang berbeda sesuai dengan preferensi Anda. Selain itu, Anda juga bisa menggunakan metode Date.parse() untuk merubah tanggal ISO menjadi nilai timestamp yang dapat dibaca oleh manusia.

Menggunakan Metode toLocaleDateString()

Metode toLocaleDateString() adalah metode yang paling umum digunakan untuk merubah tanggal ISO menjadi tanggal normal. Metode ini dalam bentuk sederhana dapat digunakan sebagai berikut:

let isoDate ='2023-08-12'; let normalDate = new Date(isoDate).toLocaleDateString();

Pada contoh di atas, kita telah mendeklarasikan variabel isoDate dengan nilai tanggal ISO. Kemudian, kita menggunakan metode toLocaleDateString() untuk mengubah tanggal ISO menjadi tanggal normal. Metode ini akan mengembalikan tanggal dalam format DD-MM-YYYY.

Selain itu, Anda juga dapat menggunakan metode toLocaleDateString() untuk merubah tanggal ISO menjadi bentuk tanggal yang berbeda. Anda hanya perlu menambahkan parameter tambahan ke metode ini. Misalnya, jika Anda ingin mengubah tanggal ISO menjadi bentuk MM-DD-YYYY, Anda dapat menggunakan metode ini seperti ini:

let isoDate ='2023-08-12'; let normalDate = new Date(isoDate).toLocaleDateString('en-US', {day: 'numeric', month: 'numeric', year: '2-digit'});

Pada contoh di atas, kita telah menambahkan parameter tambahan ke metode toLocaleDateString(). Parameter ini akan memastikan bahwa metode ini mengembalikan tanggal dalam format MM-DD-YYYY.

Menggunakan Metode Date.parse()

Selain metode toLocaleDateString(), Anda juga dapat menggunakan metode Date.parse() untuk merubah tanggal ISO menjadi timestamp. Metode ini bisa digunakan seperti ini:

let isoDate ='2023-08-12'; let timestamp = Date.parse(isoDate);

Pada contoh di atas, kita telah mendeklarasikan variabel isoDate dengan nilai tanggal ISO. Kemudian, kita menggunakan metode Date.parse() untuk mengubah tanggal ISO menjadi nilai timestamp. Nilai timestamp ini dapat dibaca oleh manusia dan digunakan untuk berbagai keperluan.

Kesimpulan

Dalam artikel ini, kita telah membahas cara mudah merubah tanggal ISO ke tanggal normal di JavaScript. Kita telah membahas bagaimana menggunakan metode toLocaleDateString() untuk merubah tanggal ISO menjadi format tanggal yang dapat dibaca oleh manusia, serta bagaimana menggunakan metode Date.parse() untuk merubah tanggal ISO menjadi nilai timestamp. Dengan pengetahuan ini, Anda sekarang dapat dengan mudah merubah tanggal ISO menjadi tanggal yang lebih mudah dimengerti.
