Cara Mengambil Input Pengguna Dari Array Di Python

Pengantar

Python merupakan salah satu bahasa pemrograman yang banyak digunakan oleh para programmer. Pemrograman Python menyediakan banyak fitur yang menarik dan menguntungkan seperti mengambil input pengguna dari array. Fitur ini memungkinkan program kita untuk mengambil data dari array yang berisi input pengguna. Pada artikel ini, kita akan membahas cara mengambil input pengguna dari array di Python.

Apa Itu Array?

Array adalah struktur data yang digunakan untuk menyimpan data dalam urutan tertentu. Array terdiri dari beberapa elemen yang terurut berdasarkan indeks. Indeks adalah nilai yang digunakan untuk mengakses elemen dari array. Array dapat berisi berbagai jenis tipe data seperti integer, karakter, string, dan objek. Array juga dapat berisi array lain yang disebut sebagai array multidimensi.

Cara Mengambil Input Pengguna dari Array di Python

Ketika kita ingin mengambil input pengguna dari array di Python, kita harus menggunakan loop untuk mengakses elemen array satu per satu. Ada beberapa loop yang dapat kita gunakan untuk mengakses elemen array, seperti for loop, while loop, dan nested loop. Kita juga dapat menggunakan built-in function seperti map(), filter(), dan reduce() untuk mengambil input pengguna dari array. Mari kita lihat cara mengambil input pengguna dari array di Python.

Menggunakan For Loop

Kita dapat menggunakan for loop untuk mengambil input pengguna dari array. For loop akan mengulangi blok kode yang ada di dalamnya berdasarkan jumlah elemen dalam array. Setiap kali loop dijalankan, kita dapat mengakses elemen array yang berbeda. Kode berikut menunjukkan cara menggunakan for loop untuk mengambil input pengguna dari array:

 user_input = []
 for i in range(len(array)):
 user_input.append(input('Masukkan nilai: '))

Kode di atas akan mengulangi sebanyak jumlah elemen array dan mengambil input pengguna untuk setiap iterasi. Hasilnya akan disimpan dalam array baru yang disebut user_input.

Menggunakan While Loop

Kita juga dapat menggunakan while loop untuk mengambil input pengguna dari array. While loop akan mengulangi blok kode yang ada di dalamnya selama kondisi tertentu terpenuhi. Kondisi ini biasanya menggunakan counter dan akan diperiksa setiap kali loop dijalankan. Kode berikut menunjukkan cara menggunakan while loop untuk mengambil input pengguna dari array:

 user_input = []
 i = 0
 while i < len(array):
 user_input.append(input('Masukkan nilai: '))
 i += 1

Kode di atas akan mengulangi sebanyak jumlah elemen array dan mengambil input pengguna untuk setiap iterasi. Hasilnya akan disimpan dalam array baru yang disebut user_input.

Menggunakan Built-in Function

Selain menggunakan loop, kita juga dapat menggunakan built-in function seperti map(), filter(), dan reduce() untuk mengambil input pengguna dari array. Built-in function dapat membantu kita untuk mengambil input pengguna dari array dengan lebih cepat dan mudah. Kode berikut menunjukkan cara menggunakan built-in function untuk mengambil input pengguna dari array:

 user_input = list(map(input, array))

Kode di atas akan mengambil input pengguna dari array dan menyimpannya dalam array baru yang disebut user_input.

Kesimpulan

Kita telah membahas cara mengambil input pengguna dari array di Python. Kita dapat menggunakan loop seperti for loop dan while loop atau built-in function seperti map(), filter(), dan reduce(). Semua metode tersebut dapat membantu kita dalam mengambil input pengguna dari array dengan mudah. Sekian artikel tentang cara mengambil input pengguna dari array di Python.
