Bagaimana Cara Mengubah Kata Sandi Root Mysql Di Centos 7?

Terbaik Pelajaran bahasa indonesia tips dan trik website . Cari apa saja topik Pelajaran bahasa indonesia ide di website ini.

CentOS 7 adalah sistem operasi yang kuat dan terkenal karena keandalannya dan kemampuannya untuk melayani berbagai jenis layanan. MySQL adalah salah satu layanan yang banyak digunakan dan diinstal di CentOS 7. MySQL memiliki root user yang memiliki akses ke semua basis data. Karena alasan keamanan, Anda harus selalu menjaga kata sandi root user MySQL aman. Berikut adalah panduan bagaimana cara mengubah kata sandi root MySQL di CentOS 7.

Persyaratan

Sebelum Anda mulai mengubah kata sandi root MySQL, pastikan Anda memenuhi persyaratan berikut:

  • Akses root atau sudo ke sistem CentOS 7.
  • Kata sandi root MySQL saat ini.

Cara Mengubah Kata Sandi Root MySQL di CentOS 7

Ada dua cara yang dapat Anda gunakan untuk mengubah kata sandi root MySQL di CentOS 7. Berikut adalah cara-cara tersebut:

Cara 1: Menggunakan Perintah mysql_secure_installation

mysql_secure_installation adalah alat yang disertakan dengan MySQL untuk membantu mengamankan instalasi. Ini akan mengubah kata sandi root MySQL dan membuat beberapa pengaturan lainnya yang diperlukan untuk mengamankan MySQL. Berikut adalah cara menggunakan mysql_secure_installation untuk mengubah kata sandi root MySQL di CentOS 7:

  1. Mulai dengan menjalankan perintah berikut untuk memulai prosedur mysql_secure_installation:
    $ sudo mysql_secure_installation
  2. Anda akan diminta untuk memasukkan kata sandi root MySQL saat ini. Ketikkan kata sandi dan tekan Enter untuk melanjutkan.
  3. Anda akan diminta untuk memasukkan kata sandi baru untuk root MySQL. Masukkan kata sandi baru dan tekan Enter untuk melanjutkan.
  4. Anda akan diminta untuk memasukkan kata sandi baru lagi untuk memastikan bahwa Anda benar-benar mengingatnya. Masukkan kata sandi baru lagi dan tekan Enter untuk melanjutkan.
  5. Anda akan diminta untuk mengkonfirmasi pilihan lainnya. Gunakan pilihan default dan tekan Enter untuk melanjutkan.
  6. Prosedur mysql_secure_installation selesai. Kata sandi root MySQL berhasil diubah.

Cara 2: Menggunakan Perintah UPDATE user

Anda juga dapat menggunakan perintah UPDATE user untuk mengubah kata sandi root MySQL. Berikut adalah cara menggunakan perintah UPDATE user untuk mengubah kata sandi root MySQL di CentOS 7:

  1. Mulai dengan menjalankan perintah berikut untuk masuk ke MySQL:
    $ mysql -u root -p
  2. Anda akan diminta untuk memasukkan kata sandi root MySQL. Ketikkan kata sandi dan tekan Enter untuk melanjutkan.
  3. Anda akan diminta untuk memasukkan kata sandi baru untuk root MySQL. Masukkan kata sandi baru dan tekan Enter untuk melanjutkan:
    mysql> UPDATE user SET Password = PASSWORD('kata_sandi_baru') WHERE User ='root';
  4. Jalankan perintah berikut untuk menerapkan perubahan:
    mysql> FLUSH PRIVILEGES;
  5. Keluar dari MySQL:
    mysql> EXIT;
  6. Kata sandi root MySQL berhasil diubah.

Kesimpulan

CentOS 7 sangat berguna untuk menjalankan berbagai jenis layanan. MySQL adalah salah satu layanan yang banyak digunakan dan diinstal di CentOS 7. Root user memiliki akses ke semua basis data MySQL, oleh karena itu, Anda harus selalu menjaga kata sandi root user MySQL aman. Di sini, kami telah menjelaskan bagaimana cara mengubah kata sandi root MySQL di CentOS 7 menggunakan mysql_secure_installation atau UPDATE user. Semoga artikel ini membantu Anda.
