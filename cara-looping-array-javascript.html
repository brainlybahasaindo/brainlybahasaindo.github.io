Apa Itu Looping Array?

Lopping array adalah teknik untuk mengeksekusi serangkaian perintah berulang pada sebuah array (kumpulan data). Ini memungkinkan Anda untuk mengulangi beberapa tugas dalam jumlah yang jauh lebih sedikit daripada jika Anda harus menulis kode yang sama berulang kali. Cara looping array dapat digunakan untuk membuat suatu proses menjadi lebih cepat dan lebih efisien.

Cara Loop Array Javascript

Cara looping array JavaScript berbeda-beda tergantung jenis loop yang Anda gunakan. Ada beberapa jenis loop yang bisa Anda gunakan dalam JavaScript, yaitu for loops, for-in loops, for-of loops, dan while loops. Berikut adalah cara melakukan looping array menggunakan masing-masing jenis loop:

For Loop

For loops digunakan untuk membuat loop yang dapat mengulangi perintah untuk jumlah tertentu kali. Anda dapat menggunakan for loop untuk melakukan looping pada array JavaScript dengan menggunakan parameter sebagai berikut:

  • i: Ini adalah variabel counter yang digunakan untuk menyimpan nilai saat loop berjalan. Ini harus dideklarasikan di luar loop.
  • Length: Ini adalah panjang array yang akan diloop. Anda dapat mendapatkan panjang array dengan menggunakan length property array.
  • Array: Ini adalah array yang akan diloop.

Contoh for loop array JavaScript adalah sebagai berikut: 

 let arr = [1,2,3,4,5]; for(let i = 0; i < arr.length; i++){ console.log(arr[i]); }

Di atas adalah contoh sederhana dari for loop yang digunakan untuk menampilkan nilai di sebuah array. Dalam contoh ini, variabel i disimpan ke variabel counter dan diterapkan ke panjang array. Loop ini akan berjalan sebanyak panjang array dan setiap loop akan mencetak nilai array ke console.

For-In Loop

For-in loop digunakan untuk looping melalui elemen di array. Ini memungkinkan Anda untuk mengakses nilai dan properti dari setiap elemen di array. Contoh for-in loop di bawah ini akan mencetak nilai dan properti dari setiap elemen array: 

 let arr = [1,2,3,4,5]; for (let i in arr) { console.log(arr[i]); console.log(i); }

Di atas adalah contoh sederhana dari for-in loop yang digunakan untuk menampilkan nilai dan properti dari setiap elemen di array. Dalam contoh ini, variabel i disimpan ke variabel counter dan diterapkan ke setiap elemen array. Loop ini akan berjalan sebanyak jumlah elemen array dan setiap loop akan mencetak nilai dan properti array ke console.

For-Of Loop

For-of loop digunakan untuk looping melalui elemen di array. Ini memungkinkan Anda untuk mengakses nilai dari setiap elemen di array. Contoh for-of loop di bawah ini akan mencetak nilai dari setiap elemen array: 

 let arr = [1,2,3,4,5]; for (let i of arr) { console.log(i); }

Di atas adalah contoh sederhana dari for-of loop yang digunakan untuk menampilkan nilai dari setiap elemen di array. Dalam contoh ini, variabel i disimpan ke variabel counter dan diterapkan ke setiap elemen array. Loop ini akan berjalan sebanyak jumlah elemen array dan setiap loop akan mencetak nilai array ke console.

While Loop

While loop digunakan untuk mengeksekusi sebuah perintah selama kondisi yang diberikan benar. Ini dapat digunakan untuk looping array dengan menggunakan parameter sebagai berikut:

  • i: Ini adalah variabel counter yang digunakan untuk menyimpan nilai saat loop berjalan. Ini harus dideklarasikan di luar loop.
  • Length: Ini adalah panjang array yang akan diloop. Anda dapat mendapatkan panjang array dengan menggunakan length property array.
  • Condition: Ini adalah kondisi yang harus dipenuhi agar loop berjalan.
  • Array: Ini adalah array yang akan diloop.

Contoh while loop array JavaScript adalah sebagai berikut: 

 let arr = [1,2,3,4,5]; let i = 0; while (i < arr.length) { console.log(arr[i]); i++; }

Di atas adalah contoh sederhana dari while loop yang digunakan untuk menampilkan nilai di sebuah array. Dalam contoh ini, variabel i disimpan ke variabel counter dan diterapkan ke panjang array. Loop ini akan berjalan selama kondisi yang diberikan benar dan setiap loop akan mencetak nilai array ke console.

Kesimpulan

Dapat disimpulkan bahwa looping array dapat dilakukan dengan berbagai jenis loop tergantung jenis loop yang Anda gunakan. Anda dapat memilih jenis loop yang paling cocok dengan kebutuhan Anda. Semoga artikel ini bermanfaat bagi Anda yang ingin mempelajari cara looping array JavaScript.
