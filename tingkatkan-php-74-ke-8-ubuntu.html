Cara Mengupdate Php 7.4 Ke 8 Di Ubuntu

Terbaik Pelajaran bahasa indonesia tips dan trik website . Cari apa saja topik Pelajaran bahasa indonesia ide di website ini.

PHP (acronym for Hypertext Preprocessor) adalah bahasa pemrograman yang paling populer digunakan untuk membangun aplikasi web. Mulai dari website besar hingga kecil, hampir semua menggunakan PHP. Di tahun 2023, PHP 8 telah dirilis dan ini adalah versi terbaru dari bahasa pemrograman ini. Jika Anda menggunakan Ubuntu dan ingin meningkatkan versi PHP Anda dari 7.4 ke 8, maka Anda berada di tempat yang tepat. Berikut adalah panduan lengkap untuk mengupdate PHP 7.4 ke 8 di Ubuntu.

1. Instal PPA

PPA (Personal Package Archive) adalah repositori yang berisi paket yang dikelola oleh pihak ketiga. Jadi, sebelum menginstal versi baru, Anda harus memasang PPA dahulu. Untuk menginstal PPA, Anda harus membuka terminal dan ketikkan perintah berikut:

sudo add-apt-repository ppa:ondrej/php

Setelah mengetikkan perintah, Anda harus mengkonfirmasi instalasi dengan menekan tombol "Enter". Proses ini akan memasang PPA, yang selanjutnya akan digunakan untuk mengupdate PHP.

2. Perbarui Repositori

Setelah PPA berhasil diinstal, Anda harus memperbarui repositori dengan mengetikkan perintah berikut di terminal:

sudo apt update

Perintah ini akan memperbarui semua paket yang tersedia di repositori. Ini akan membantu Anda untuk mengunduh versi terbaru PHP.

3. Pilih PHP yang Anda Inginkan

Sekarang, Anda harus menentukan versi PHP yang akan diinstal. Ini dapat dilakukan dengan mengetikkan perintah berikut di terminal:

sudo apt install php8.0

Perintah ini akan memulai proses instalasi. Anda harus mengikuti petunjuk yang diberikan untuk menyelesaikan proses instalasi.

4. Verifikasi Instalasi PHP 8

Setelah instalasi selesai, Anda harus memverifikasi bahwa PHP 8 berhasil diinstal. Untuk melakukan ini, Anda harus membuka terminal dan mengetikkan perintah berikut:

php -v

Perintah ini akan menampilkan versi PHP yang diinstal. Jika versi yang ditampilkan adalah 8.0, maka Anda berhasil mengupdate PHP Anda dari 7.4 ke 8.

5. Konfigurasi PHP 8

Setelah PHP 8 berhasil diinstal, Anda harus melakukan konfigurasi untuk memastikan bahwa semua pengaturan yang diperlukan telah dimasukkan. Ini dapat dilakukan dengan mengedit file php.ini. Anda harus membuka file ini dengan editor teks dan pastikan bahwa pengaturan berikut telah dimasukkan:

memory_limit = 128M upload_max_filesize = 64M max_input_time = 60 max_execution_time = 30

Setelah mengedit file php.ini, Anda harus menyimpan file dan memulai ulang web server. Ini akan membuat PHP 8 berfungsi dengan benar.

6. Instalasi Modul PHP 8

PHP 8 memiliki beberapa modul yang dapat membantu Anda dalam meningkatkan kinerja aplikasi web Anda. Untuk menginstal modul PHP 8, Anda harus membuka terminal dan mengetikkan perintah berikut:

sudo apt install php8.0-common php8.0-json php8.0-opcache php8.0-readline

Perintah ini akan mengunduh dan menginstal modul PHP 8 yang diperlukan. Setelah semua modul berhasil diinstal, Anda harus memulai ulang web server. Ini akan memastikan bahwa modul berfungsi dengan benar.

7. Uji Coba PHP 8

Setelah semuanya siap, Anda harus melakukan uji coba untuk memastikan bahwa semua berjalan dengan lancar. Anda dapat melakukan ini dengan membuat file yang disebut phpinfo.php di folder web Anda. File ini akan menampilkan informasi lengkap tentang PHP yang diinstal. Jika informasi yang ditampilkan adalah 8.0, maka Anda telah berhasil meningkatkan versi PHP Anda dari 7.4 ke 8.

Kesimpulan

Setelah menyelesaikan panduan ini, Anda harus dapat meningkatkan versi PHP Anda dari 7.4 ke 8 di Ubuntu. Meskipun ini cukup mudah, Anda harus mengikuti instruksi dengan hati-hati untuk menghindari masalah. Jika Anda mengalami masalah, Anda harus menghubungi tim dukungan. Dengan informasi ini, Anda sekarang dapat dengan mudah meningkatkan versi PHP Anda di Ubuntu.
