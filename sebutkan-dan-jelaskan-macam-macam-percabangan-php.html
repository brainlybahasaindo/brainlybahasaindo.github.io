Mengenal Macam-Macam Percabangan Php

PHP adalah bahasa pemrograman yang populer yang banyak digunakan untuk membangun aplikasi web dan digunakan oleh ribuan website di seluruh dunia. Percabangan PHP merupakan bagian penting dari bahasa ini, dimana percabangan memungkinkan Anda untuk membuat aplikasi berdasarkan logika. Dengan PHP, Anda dapat membuat kondisi yang dapat menentukan alur program yang akan Anda jalankan. Berikut ini adalah beberapa macam percabangan yang tersedia di PHP.

Percabangan IF-ELSE

Percabangan IF-ELSE adalah salah satu jenis percabangan yang paling umum digunakan dalam PHP. Di sini, kondisi yang ditentukan dalam pernyataan IF akan diuji dan jika kondisi tersebut benar, kode yang ada di blok IF akan dijalankan. Jika kondisi tersebut salah, maka blok ELSE akan dijalankan. Misalnya, jika Anda ingin memeriksa apakah suatu nilai adalah positif atau negatif, Anda dapat menggunakan percabangan IF-ELSE. Berikut adalah contoh sederhananya:

 if($nilai > 0) { 
 echo "Nilai positif"; 
 } else { 
 echo "Nilai negatif"; 
 }

Di atas, kita menguji kondisi apakah nilai lebih besar dari nol atau tidak. Jika nilai lebih besar dari nol, maka blok IF akan dijalankan dan jika nilai kurang dari atau sama dengan nol, maka blok ELSE akan dijalankan.

Percabangan IF-ELSEIF-ELSE

Percabangan IF-ELSEIF-ELSE adalah bentuk lebih lanjut dari percabangan IF-ELSE. Di sini, kita dapat menambahkan lebih dari satu kondisi untuk diuji dan berbeda blok kode untuk setiap kondisi. Jika kondisi yang ditentukan dalam pernyataan IF benar, maka blok IF akan dijalankan. Jika kondisi IF salah, maka kondisi ELSEIF akan diuji dan jika kondisi ELSEIF benar, maka blok ELSEIF akan dijalankan. Jika semua kondisi salah, maka blok ELSE akan dijalankan. Berikut adalah contohnya:

 if($nilai > 0) { 
 echo "Nilai positif"; 
 } elseif ($nilai < 0) {
 echo "Nilai negatif"; 
 } else { 
 echo "Nilai nol"; 
 }

Di atas, kita menguji kondisi apakah nilai lebih besar dari nol, kurang dari nol, atau sama dengan nol. Jika nilai lebih besar dari nol, maka blok IF akan dijalankan, jika nilai kurang dari nol, maka blok ELSEIF akan dijalankan, dan jika nilai sama dengan nol, maka blok ELSE akan dijalankan.

Percabangan SWITCH

Percabangan SWITCH adalah bentuk lain dari percabangan yang dapat digunakan di PHP. Di sini, kita dapat menguji lebih dari satu kondisi untuk satu variabel. Misalnya, jika Anda ingin memeriksa nilai suatu variabel, Anda dapat menggunakan percabangan SWITCH. Berikut adalah contohnya:

 switch($nilai) { 
 case 1: 
 echo "Nilai satu"; 
 break; 
 case 2: 
 echo "Nilai dua"; 
 break; 
 case 3: 
 echo "Nilai tiga"; 
 break; 
 default: 
 echo "Nilai lain"; 
 }

Di atas, kita menguji apakah nilai variabel yang ditentukan adalah satu, dua, atau tiga. Jika nilainya satu, maka blok CASE 1 akan dijalankan, jika nilainya dua, maka blok CASE 2 akan dijalankan, jika nilainya tiga, maka blok CASE 3 akan dijalankan, dan jika nilai variabel bukan satu, dua, atau tiga, maka blok DEFAULT akan dijalankan.

Percabangan GANDA

Percabangan GANDA adalah bentuk lain dari percabangan yang dapat Anda gunakan di PHP. Di sini, kita dapat menentukan lebih dari satu kondisi untuk menentukan alur program yang akan dijalankan. Misalnya, jika Anda ingin memeriksa apakah suatu nilai adalah positif atau negatif, Anda dapat menggunakan percabangan GANDA. Berikut adalah contohnya:

 if($nilai > 0) { 
 echo "Nilai positif"; 
 } elseif ($nilai < 0) {
 echo "Nilai negatif"; 
 } else { 
 echo "Nilai nol"; 
 }

Di atas, kita menguji kondisi apakah nilai lebih besar dari nol, kurang dari nol, atau sama dengan nol. Jika nilai lebih besar dari nol, maka blok IF akan dijalankan, jika nilai kurang dari nol, maka blok ELSEIF akan dijalankan, dan jika nilai sama dengan nol, maka blok ELSE akan dijalankan.

Percabangan BERULANG

Percabangan BERULANG adalah bentuk lain dari percabangan yang dapat digunakan di PHP. Di sini, kita dapat menentukan kondisi yang akan diulang selama kondisi tersebut benar. Misalnya, jika Anda ingin menampilkan beberapa baris data dari database, Anda dapat menggunakan percabangan BERULANG. Berikut adalah contohnya:

 while ($data = mysql_fetch_array($result)) { 
 echo "Nama: ".$data['nama']."; 
 echo "Umur: ".$data['umur']."; 
 echo "Alamat: ".$data['alamat']."; 
 }

Di atas, kita menggunakan percabangan BERULANG untuk menampilkan beberapa baris data dari database. Kode di dalam blok WHILE akan diulang selama kondisi yang ditentukan dalam pernyataan WHILE benar. Jadi, jika ada beberapa baris data dalam database, kode ini akan diulang sebanyak baris data tersebut.

Kesimpulan

Dengan PHP, Anda dapat membuat berbagai macam percabangan untuk menentukan alur program yang akan Anda jalankan. Beberapa jenis percabangan yang tersedia di PHP adalah percabangan IF-ELSE, percabangan IF-ELSEIF-ELSE, percabangan SWITCH, percabangan GANDA, dan percabangan BERULANG. Jadi, jika Anda ingin membuat aplikasi berdasarkan logika, Anda harus mengetahui mengenai macam-macam percabangan yang tersedia di PHP.
