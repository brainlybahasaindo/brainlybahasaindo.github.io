Cara Menambahkan Kunci Dan Nilai Dalam Array Di Php

Array adalah salah satu komponen yang paling penting yang dapat Anda temukan di PHP. Array dapat berisi berbagai macam data, mulai dari string, angka, boolean, objek, dan lainnya. Dalam artikel ini kita akan membahas bagaimana cara menambahkan kunci dan nilai ke array di PHP.

Apa itu Array?

Array adalah kumpulan dari nilai yang terkait dengan kunci yang terkait. Array dapat diwakili sebagai kotak yang berisi berbagai macam nilai. Array dapat berisi berbagai macam tipe data. Array dapat dianggap sebagai variabel yang dapat menampung lebih dari satu nilai sekaligus. Array dapat dibuat dengan menggunakan tanda kurung kurawal ( [ ] ) dan nilai-nilai dalam array dipisahkan dengan tanda koma ( , ).

Cara Menambahkan Kunci dan Nilai ke Array di PHP

Ada beberapa cara untuk menambahkan kunci dan nilai ke array di PHP. Kunci dan nilai dapat ditambahkan dengan menggunakan tanda titik dua ( : ) atau tanda sama dengan ( = ). Nilai yang ditambahkan ke array akan tersedia untuk diteruskan ke fungsi atau skrip lainnya.

Menambahkan Kunci dan Nilai Menggunakan Tanda Titik Dua ( : )

Anda dapat menggunakan tanda titik dua ( : ) untuk menambahkan kunci dan nilai ke array. Berikut ini contoh penggunaannya:

 $array = array("key" => "value");

Dalam contoh di atas, kunci adalah "key" dan nilai adalah "value". Kunci dan nilai bisa berupa string, angka, boolean, objek, dan lainnya. Anda juga dapat menambahkan lebih dari satu kunci dan nilai ke array menggunakan tanda titik dua ( : ). Berikut ini contoh penggunaannya:

 $array = array("key1" => "value1", "key2" => "value2");

Menambahkan Kunci dan Nilai Menggunakan Tanda Sama Dengan ( = )

Anda juga dapat menggunakan tanda sama dengan ( = ) untuk menambahkan kunci dan nilai ke array. Berikut ini contoh penggunaannya:

 $array["key"] ="value";

Dalam contoh di atas, kunci adalah "key" dan nilai adalah "value". Anda juga dapat menambahkan lebih dari satu kunci dan nilai ke array menggunakan tanda sama dengan ( = ). Berikut ini contoh penggunaannya:

 $array["key1"] ="value1";
 $array["key2"] ="value2";

Kesimpulan

Array adalah kumpulan dari nilai yang terkait dengan kunci yang terkait. PHP menyediakan berbagai cara untuk menambahkan kunci dan nilai ke array. Kunci dan nilai dapat ditambahkan dengan menggunakan tanda titik dua ( : ) atau tanda sama dengan ( = ). Kunci dan nilai yang ditambahkan ke array akan tersedia untuk diteruskan ke fungsi atau skrip lainnya.
