Cara Menggunakan Datatable Bootstrap 4

Pengertian Datatable Bootstrap 4

Datatable Bootstrap 4 adalah sebuah komponen yang menyediakan fitur untuk menampilkan data dalam bentuk tabel. Komponen ini memiliki beberapa fitur canggih yang tidak tersedia di tabel HTML biasa, seperti sorting, pagination, search dan lainnya. Komponen ini juga dapat diintegrasikan dengan beberapa library jQuery seperti jQuery UI dan DataTables.

Cara Menggunakan Datatable Bootstrap 4

Untuk menggunakan Datatable Bootstrap 4, Anda harus mendownload library yang tersedia di situs resmi Bootstrap 4. Anda juga perlu mendownload file CSS dan Javascript berikut untuk memastikan Datatable berfungsi dengan baik:

  • DataTables.min.js
  • DataTables.min.css
  • jQuery UI (jQuery UI.min.js dan jQuery UI.min.css)

Setelah itu, Anda harus menambahkan skrip Javascript berikut ke dalam halaman web Anda untuk menginisialisasi DataTable.

$(document).ready(function(){
 $('#example').DataTable();
 });

Membuat Tabel

Setelah Anda berhasil mendownload library dan menambahkan skrip inisialisasi, Anda dapat membuat tabel dengan mudah. Anda cukup membuat tabel HTML biasa dan menambahkan class "table" dan "dataTable" ke tag

. Berikut adalah contoh tabel yang sudah dikonfigurasi dengan Datatable Bootstrap 4:

NamePositionOfficeAgeStart dateSalary
Tiger NixonSystem ArchitectEdinburgh612011/04/25$320,800
Garrett WintersAccountantTokyo632011/07/25$170,750

Menambahkan Fitur Sorting

Untuk menambahkan fitur sorting, Anda hanya perlu menambahkan atribut "data-sortable" ke setiap tag yang ingin Anda sorot. Berikut adalah contohnya:

NamePositionOfficeAgeStart dateSalary

Menambahkan Fitur Pencarian

Untuk menambahkan fitur pencarian, Anda harus menambahkan skrip berikut ke halaman web Anda:

$('#example').DataTable({
 searching: true
 });

Menambahkan Fitur Pagination

Untuk menambahkan fitur pagination, Anda harus menambahkan skrip berikut ke halaman web Anda:

$('#example').DataTable({
 pageLength: 10
 });

Anda juga dapat menambahkan tombol navigasi dan jumlah item perhalaman di tabel dengan menambahkan skrip berikut:

$('#example').DataTable({
 lengthChange: true,
 lengthMenu: [10, 25, 50, 100]
 });

Menambahkan Fitur Responsive

Untuk menambahkan fitur responsive, Anda harus menambahkan skrip berikut ke halaman web Anda:

$('#example').DataTable({
 responsive: true
 });

Kesimpulan

Datatable Bootstrap 4 adalah sebuah komponen yang memungkinkan Anda untuk menampilkan data dalam bentuk tabel dengan berbagai fitur canggih. Fitur-fitur ini termasuk sorting, pagination, search dan responsive. Anda dapat mendownload library dan menambahkan skrip inisialisasi untuk mengaktifkan fitur-fitur ini. Dengan demikian, Anda dapat dengan mudah membuat tabel interaktif yang responsif.
