Cara Mengatasi "The Specified Network Name Is No Longer Available" Di Windows 10

Windows 10 adalah salah satu sistem operasi terbaik yang dirilis Microsoft. Namun, beberapa masalah yang muncul di Windows 10, termasuk "The Specified Network Name is No Longer Available" error. Error ini menyebabkan Anda tidak dapat menghubungkan ke jaringan nirkabel, atau Anda hanya dapat menghubungkan ke jaringan nirkabel untuk sementara waktu. Error ini bisa disebabkan oleh berbagai alasan, termasuk masalah pada driver jaringan, masalah keamanan, kesalahan konfigurasi jaringan, atau masalah lainnya. Berikut adalah beberapa cara untuk mengatasi "The Specified Network Name is No Longer Available" error di Windows 10.

Cara 1: Periksa Driver Jaringan Anda

Salah satu penyebab "The Specified Network Name is No Longer Available" error adalah driver jaringan yang tidak up to date. Anda harus memeriksa driver jaringan Anda untuk memastikan bahwa semuanya up to date. Anda dapat memeriksa driver jaringan Anda melalui Device Manager. Cukup buka Device Manager, cari dan buka "Network Adapters". Periksa semua adaptor jaringan yang terdaftar di sana. Klik kanan salah satu adaptor jaringan, lalu pilih "Update Driver". Anda dapat mengikuti petunjuk di layar untuk mengupdate driver jaringan Anda.

Cara 2: Periksa Pembatasan Akses Jaringan

Beberapa jaringan dapat membatasi akses ke jaringan mereka. Untuk mengecek apakah jaringan Anda membatasi akses, coba masuk ke router Anda. Anda dapat melakukannya dengan mengetikkan alamat IP router Anda ke dalam browser web. Jika Anda tidak tahu alamat IP router Anda, Anda dapat menemukannya dengan mengetikkan "ipconfig" di Command Prompt. Setelah masuk ke router Anda, cari pengaturan yang berkaitan dengan pembatasan akses. Periksa apakah ada pembatasan pada jaringan yang Anda gunakan.

Cara 3: Periksa Masalah Keamanan

Beberapa masalah keamanan juga dapat menyebabkan "The Specified Network Name is No Longer Available" error. Anda harus memeriksa firewall dan antivirus Anda untuk memastikan bahwa mereka tidak menyebabkan masalah. Jika Anda menggunakan firewall atau antivirus, cobalah menonaktifkannya sementara waktu untuk melihat apakah error hilang. Jika error hilang, Anda dapat mengaktifkan firewall dan antivirus Anda lagi.

Cara 4: Periksa Pengaturan Jaringan Anda

Kesalahan konfigurasi jaringan juga dapat menyebabkan "The Specified Network Name is No Longer Available" error. Anda harus memeriksa semua pengaturan jaringan Anda untuk memastikan bahwa semuanya diatur dengan benar. Untuk melakukan ini, buka Control Panel, lalu pilih "Network and Internet". Pilih "Network and Sharing Center". Di sana, Anda akan melihat beberapa pengaturan jaringan. Periksa semua pengaturan ini untuk memastikan bahwa semuanya diatur dengan benar.

Cara 5: Periksa DHCP Server

DHCP server juga dapat menyebabkan "The Specified Network Name is No Longer Available" error. Anda harus memeriksa DHCP server Anda untuk memastikan bahwa semuanya berjalan dengan benar. Untuk melakukan ini, buka Command Prompt, lalu ketik "ipconfig /all". Lihatlah hasil, dan pastikan bahwa DHCP Enabled adalah "Yes". Jika DHCP Enabled adalah "No", coba buka DHCP server Anda dan pastikan bahwa semuanya diatur dengan benar.

Cara 6: Uninstall dan Reinstall Adaptor Jaringan Anda

Jika Anda masih mengalami masalah, cobalah me-uninstall dan menginstal adaptor jaringan Anda. Untuk me-uninstall adaptor jaringan Anda, buka Device Manager, cari dan buka "Network Adapters". Klik kanan salah satu adaptor jaringan, lalu pilih "Uninstall". Ketika uninstall selesai, restart komputer Anda. Setelah restart, Windows akan mendeteksi adaptor jaringan Anda dan menginstal driver yang diperlukan.

Cara 7: Gunakan Kabel LAN

Jika Anda masih mengalami masalah, cobalah menggunakan kabel LAN untuk menghubungkan ke jaringan. Kabel LAN dapat menyelesaikan banyak masalah koneksi jaringan. Untuk menggunakan kabel LAN, hubungkan kabel LAN ke port LAN di router Anda. Kemudian, hubungkan kabel LAN lainnya ke port LAN di komputer Anda. Sekarang komputer Anda harus terhubung ke jaringan.

Cara 8: Reset Router Anda

Jika Anda masih mengalami masalah, cobalah me-reset router Anda. Reset router dapat menyelesaikan banyak masalah jaringan. Untuk me-reset router Anda, cari tombol reset di router Anda. Tekan dan tahan tombol reset selama beberapa detik. Setelah router me-reset, coba lagi untuk menghubungkan ke jaringan.

Cara 9: Gunakan Troubleshooter Jaringan

Windows 10 memiliki Troubleshooter Jaringan yang dapat Anda gunakan untuk mencari dan memperbaiki masalah jaringan. Untuk menggunakan troubleshooter jaringan, buka Control Panel, lalu pilih "Troubleshooting". Pilih "Network and Internet". Pilih "Network Adapter" dan ikuti petunjuk di layar untuk mencari dan memperbaiki masalah jaringan Anda.

Cara 10: Perbarui Windows 10 Anda

Terakhir, Anda harus memperbarui sistem operasi Anda jika Anda mengalami masalah. Microsoft sering me-release update untuk menyelesaikan masalah yang mungkin Anda alami. Untuk memperbarui Windows 10 Anda, buka Settings, lalu pilih "Update & Security". Pilih "Windows Update" dan ikuti petunjuk di layar untuk memperbarui Windows 10 Anda.

Itulah beberapa cara untuk mengatasi "The Specified Network Name is No Longer Available" error di Windows 10. Jika Anda masih mengalami masalah, Anda harus menghubungi Microsoft untuk mendapatkan bantuan lebih lanjut.
