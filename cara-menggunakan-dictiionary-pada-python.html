Cara Menggunakan Dictionary Pada Python - Panduan Lengkap 2023

Python adalah bahasa pemrograman yang populer dan banyak digunakan oleh para pengembang. Python juga termasuk salah satu bahasa pemrograman yang paling mudah dipelajari. Banyak komponen Python yang bisa digunakan untuk membuat berbagai aplikasi dan algoritma, salah satunya adalah dictionary.

Dictionary adalah kumpulan data yang disimpan dalam bentuk pasangan kunci-nilai. Jadi, setiap item dalam dictionary akan menyimpan dua bagian informasi, yaitu kunci dan nilai. Kunci adalah yang digunakan untuk mengakses nilai tersebut. Nilai adalah informasi yang disimpan dalam dictionary.

Dictionary pada Python adalah tipe data yang bisa digunakan untuk menyimpan kumpulan data. Ini sangat berguna jika Anda ingin menyimpan data yang terkait, misalnya detail produk, informasi pelanggan, dan lain-lain. Anda bisa menggunakan dictionary pada Python dengan menggunakan sintaks berikut:

Cara Membuat Dictionary Pada Python

Untuk membuat dictionary pada Python, Anda cukup menuliskan kata kunci-nilai berikut ini: 

dict = {key1:value1, key2:value2, …..}

Di mana kunci dan nilai dapat berupa tipe data apa pun, termasuk string, integer, float, boolean, list, tuple, dan set.

Cara Mengakses Nilai dari Dictionary Pada Python

Untuk mengakses nilai dari dictionary, Anda cukup menuliskan kunci yang ingin Anda akses. Berikut adalah contoh sintaksnya: 

dict[key]

Contoh berikut ini menunjukkan cara mengakses nilai dari dictionary: 

 dict = {“A”:1, “B”:2, “C”:3}
 nilai_dari_a = dict[“A”]
 print(nilai_dari_a)
 
 # Output
 1

Di sini, Anda ingin mengakses nilai dari kunci “A”. Anda bisa menggunakan sintaks di atas untuk mengakses nilai dari kunci tersebut.

Cara Menambah Nilai dalam Dictionary Pada Python

Untuk menambah nilai dalam dictionary, Anda dapat menggunakan sintaks berikut: 

dict[key] = value

Contoh berikut ini menunjukkan cara menambah nilai ke dalam dictionary: 

 dict = {“A”:1, “B”:2, “C”:3}
 dict[“D”] = 4
 print (dict)
 
 # Output
 {“A”:1, “B”:2, “C”:3, “D”:4}

Di sini, Anda ingin menambah nilai "4" dengan kunci “D”. Anda bisa menggunakan sintaks di atas untuk menambah nilai ke dalam dictionary.

Cara Mengubah Nilai dalam Dictionary Pada Python

Untuk mengubah nilai dalam dictionary, Anda dapat menggunakan sintaks berikut: 

dict[key] = value

Contoh berikut ini menunjukkan cara mengubah nilai dalam dictionary: 

 dict = {“A”:1, “B”:2, “C”:3}
 dict[“C”] = 4
 print (dict)
 
 # Output
 {“A”:1, “B”:2, “C”:4}

Di sini, Anda ingin mengubah nilai "3" dengan kunci “C” menjadi "4". Anda bisa menggunakan sintaks di atas untuk mengubah nilai dalam dictionary.

Cara Menghapus Nilai dalam Dictionary Pada Python

Untuk menghapus nilai dalam dictionary, Anda dapat menggunakan sintaks berikut: 

del dict[key]

Contoh berikut ini menunjukkan cara menghapus nilai dalam dictionary: 

 dict = {“A”:1, “B”:2, “C”:3}
 del dict[“C”]
 print (dict)
 
 # Output
 {“A”:1, “B”:2}

Di sini, Anda ingin menghapus nilai dengan kunci “C”. Anda bisa menggunakan sintaks di atas untuk menghapus nilai dalam dictionary.

Kesimpulan

Dari contoh di atas, Anda sudah tahu bagaimana cara menggunakan dictionary pada Python. Dengan menggunakan dictionary, Anda bisa menyimpan, mengakses, menambah, mengubah, dan menghapus data dalam Python dengan mudah. Semoga panduan ini bermanfaat bagi Anda.
