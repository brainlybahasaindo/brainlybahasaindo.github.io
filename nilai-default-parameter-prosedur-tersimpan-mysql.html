Pengertian Nilai Default Parameter Prosedur Tersimpan Di Mysql

Ketika Anda membuat sebuah prosedur tersimpan di MySQL, Anda dapat menetapkan nilai parameter default. Nilai parameter default adalah nilai yang digunakan jika Anda tidak menyediakan nilai parameter saat memanggil prosedur tersimpan. Parameter adalah variabel yang diteruskan ke prosedur tersimpan. Anda dapat menetapkan nilai parameter default saat membuat prosedur tersimpan. Nilai default parameter dapat ditentukan dalam tipe data yang berbeda seperti integer, karakter, dan tanggal.

Parameter default adalah nilai yang digunakan jika Anda melewatkan parameter ketika memanggil prosedur tersimpan. Parameter ini membuat lebih mudah untuk membuat prosedur tersimpan yang dapat dipanggil dengan mudah dan memberikan nilai default dari parameter ketika parameter itu tidak ditentukan. Parameter default juga dapat membantu Anda mengeksekusi prosedur tersimpan dengan cepat.

Membuat Nilai Default Parameter Prosedur Tersimpan di MySQL

Untuk membuat nilai default parameter prosedur tersimpan di MySQL, Anda harus menggunakan sintaks CREATE PROCEDURE. Sintaks CREATE PROCEDURE memiliki beberapa parameter seperti nama prosedur, nama parameter, tipe data parameter, dan nilai default.

Berikut adalah sintaks CREATE PROCEDURE dasar yang digunakan untuk membuat nilai default parameter prosedur tersimpan di MySQL:

CREATE PROCEDURE nama_prosedur 
 ( 
 nama_parameter datatype [DEFAULT nilai_default] 
 ) 
 BEGIN 
 -- prosedur tersimpan 
 END;

Sintaks di atas digunakan untuk membuat nilai default parameter prosedur tersimpan di MySQL. Anda dapat melihat bahwa parameter datatype dari parameter memiliki nilai default yang ditentukan. Nilai default akan digunakan jika Anda tidak menyediakan nilai parameter saat memanggil prosedur tersimpan.

Contoh Nilai Default Parameter Prosedur Tersimpan di MySQL

Berikut adalah contoh sederhana untuk menunjukkan cara menetapkan nilai default parameter prosedur tersimpan di MySQL:

CREATE PROCEDURE test 
 ( 
 nama VARCHAR(30) DEFAULT 'John Doe' 
 ) 
 BEGIN 
 SELECT nama; 
 END;

Dalam contoh di atas, Anda dapat melihat bahwa parameter nama memiliki nilai default 'John Doe'. Nilai default ini akan digunakan jika Anda tidak menyediakan nilai parameter saat memanggil prosedur tersimpan.

Menggunakan Nilai Default Parameter Prosedur Tersimpan di MySQL

Setelah Anda menetapkan nilai default parameter prosedur tersimpan di MySQL, Anda dapat menggunakannya. Untuk menggunakan nilai default parameter prosedur tersimpan, Anda harus menggunakan sintaks CALL. Sintaks CALL digunakan untuk memanggil prosedur tersimpan. Anda dapat memanggil prosedur tersimpan dengan atau tanpa parameter.

Berikut adalah contoh sederhana untuk menunjukkan cara menggunakan nilai default parameter prosedur tersimpan di MySQL:

CALL test();

Dalam contoh di atas, Anda dapat melihat bahwa kami memanggil prosedur tersimpan tanpa menyediakan nilai parameter. Karena kami tidak menyediakan nilai parameter, nilai default yang telah ditentukan akan digunakan.

Kesimpulan

Nilai default parameter prosedur tersimpan di MySQL adalah nilai yang digunakan jika Anda melewatkan parameter ketika memanggil prosedur tersimpan.
