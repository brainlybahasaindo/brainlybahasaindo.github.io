Bagaimana Cara Masuk Ke Mysql Di Mac

Apa itu MySQL?

MySQL adalah salah satu sistem manajemen basis data yang paling populer dan terkenal di dunia. Basis data MySQL digunakan di berbagai jenis perangkat lunak, termasuk CMS (Content Management System) populer seperti Wordpress, Joomla dan Drupal. MySQL juga merupakan salah satu komponen dari LAMP (Linux, Apache, MySQL, PHP) atau LEMP (Linux, Nginx, MySQL, PHP) yang sering digunakan untuk membangun dan menjalankan aplikasi web dengan cepat dan mudah. MySQL dikembangkan oleh Oracle Corporation dan dapat diinstal di berbagai jenis sistem operasi, termasuk Windows, Linux, MacOS dan Solaris.

Bagaimana Cara Masuk ke MySQL di Mac?

MySQL dapat diinstal di MacOS dengan beberapa cara yang berbeda, termasuk menggunakan paket biner, Homebrew, dan Docker. Berikut ini adalah panduan tentang cara memasang dan mengkonfigurasi MySQL di MacOS.

Instalasi Menggunakan Paket Biner

MySQL dapat dengan mudah diinstal di MacOS menggunakan paket biner yang disediakan oleh Oracle. Instalasi menggunakan paket biner cukup mudah dan cepat, jadi ini adalah cara yang disarankan jika Anda ingin menginstal MySQL di MacOS.

Untuk memulai, Anda harus mengunduh paket biner dari situs web Oracle. Anda akan menemukan link untuk mengunduh paket biner di halaman download MySQL. Setelah mengunduh paket biner, buka terminal dan jalankan perintah berikut untuk menginstal paket biner.

$ sudo installer -pkg mysql-5.7.21-macos10.14-x86_64.pkg -target /

Ini akan menginstal MySQL di folder /usr/local/mysql. Setelah menyelesaikan proses instalasi, jalankan perintah berikut untuk memulai server MySQL.

$ sudo /usr/local/mysql/support-files/mysql.server start

Untuk memastikan bahwa server berjalan dengan benar, Anda dapat menggunakan perintah berikut untuk memeriksa statusnya.

$ sudo /usr/local/mysql/support-files/mysql.server status

Instalasi Menggunakan Homebrew

Homebrew adalah package manager yang dapat digunakan untuk menginstal berbagai jenis aplikasi di MacOS dengan mudah. Untuk menginstal MySQL menggunakan Homebrew, Anda harus menjalankan perintah berikut di terminal.

$ brew install mysql

Setelah menyelesaikan instalasi, Anda dapat menjalankan server MySQL dengan menjalankan perintah berikut.

$ mysql.server start

Untuk memastikan bahwa server berjalan dengan benar, Anda dapat menggunakan perintah berikut untuk memeriksa statusnya.

$ mysql.server status

Instalasi Menggunakan Docker

Docker adalah container virtualisasi yang dapat digunakan untuk menjalankan berbagai jenis aplikasi dan basis data dengan mudah. Untuk menginstal MySQL menggunakan Docker, Anda harus menjalankan perintah berikut di terminal.

$ docker run --name=mysql -d -p 3306:3306 -e MYSQL_ROOT_PASSWORD=password mysql

Setelah menyelesaikan instalasi, Anda dapat menjalankan server MySQL dengan menjalankan perintah berikut.

$ docker start mysql

Untuk memastikan bahwa server berjalan dengan benar, Anda dapat menggunakan perintah berikut untuk memeriksa statusnya.

$ docker ps

Konfigurasi MySQL

Setelah menginstal MySQL, Anda harus mengkonfigurasi untuk menyesuaikan kebutuhan pengguna. Untuk memulai, Anda harus mengakses command-line shell dan masuk sebagai root. Anda dapat mengakses shell dengan menjalankan perintah berikut.

$ mysql -u root -p

Setelah masuk sebagai root, Anda dapat menggunakan perintah berikut untuk mengubah password root.

ALTER USER 'root'@'localhost' IDENTIFIED BY 'new_password';

Anda juga dapat menggunakan perintah berikut untuk menambahkan pengguna baru.

CREATE USER 'username'@'localhost' IDENTIFIED BY 'password';

Setelah menambahkan pengguna baru, Anda dapat menggunakan perintah berikut untuk memberikan hak akses ke pengguna baru.

GRANT ALL PRIVILEGES ON *.* TO 'username'@'localhost';

Dengan begitu, Anda sudah selesai mengkonfigurasi MySQL di MacOS. Anda dapat dengan mudah mengakses basis data MySQL dan menjalankan berbagai jenis operasi basis data dengan mudah.
