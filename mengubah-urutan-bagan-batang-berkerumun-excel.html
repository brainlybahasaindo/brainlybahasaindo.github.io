Cara Mengubah Urutan Bagan Batang Berkerumun Di Excel 2023

Terbaik Pelajaran bahasa indonesia tips dan trik website . Cari apa saja topik Pelajaran bahasa indonesia ide di website ini.

Tahun 2023, Microsoft Excel telah menjadi salah satu program yang paling populer. Excel telah menjadi program wajib dalam kehidupan sehari-hari karena banyaknya fitur yang telah dibangun di dalamnya. Salah satu fitur yang umum digunakan adalah menggambar bagan batang berkerumun. Bagan batang berkerumun dapat menyajikan informasi dengan cara yang visual dan mudah dimengerti.

Namun, ada kalanya Anda mungkin ingin mengubah urutan bagan batang berkerumun. Mungkin Anda ingin mengurutkan berdasarkan nilai tertinggi atau terendah, atau mungkin Anda ingin mengurutkannya berdasarkan abjad. Apapun alasan Anda, berikut adalah cara untuk mengubah urutan bagan batang berkerumun di Excel 2023.

1. Siapkan Data Anda

Sebelum Anda dapat mengubah urutan bagan batang berkerumun, Anda harus mempersiapkan data Anda dengan benar. Jika data Anda tidak tersusun dengan benar, Anda tidak akan dapat mengubah urutannya dengan tepat. Pastikan Anda menyusun data Anda dengan benar sebelum mulai mengubah urutannya.

2. Buka File Excel Anda

Selanjutnya, buka file Excel Anda dan pastikan Anda berada di jendela yang tepat. Jika Anda belum membuat bagan batang berkerumun, Anda dapat membuatnya sekarang. Pilih opsi “Grafik” dari menu “Insert”, lalu pilih “Grafik Bagan Batang”. Anda akan melihat opsi untuk membuat bagan batang berkerumun.

3. Pilih Data Yang Ingin Anda Ubah Urutannya

Setelah Anda membuat bagan batang berkerumun, Anda dapat memilih data yang ingin Anda ubah urutannya. Anda dapat memilih data dengan mengklik pada bagian yang berwarna biru di bawah bagan. Anda akan melihat daftar data yang tersedia di bagian samping jendela. Pilih data yang ingin Anda ubah urutannya.

4. Klik Tombol “Urutkan”

Selanjutnya, klik tombol “Urutkan” yang terletak di bagian atas jendela. Anda akan melihat opsi untuk mengurutkan data berdasarkan nilai tertinggi atau terendah, atau berdasarkan abjad. Pilih opsi yang sesuai dengan kebutuhan Anda.

5. Klik Tombol “OK”

Setelah Anda selesai mengurutkan data, klik tombol “OK” untuk menerapkan perubahan. Anda akan melihat bagan batang berkerumun telah diurutkan sesuai dengan pilihan Anda.

6. Simpan File Excel Anda

Setelah Anda selesai mengubah urutan bagan batang berkerumun, jangan lupa untuk menyimpan file Excel Anda. Anda dapat melakukannya dengan mengklik tombol “Save” di bagian atas jendela. Anda juga dapat menyimpan file Excel Anda sebagai file yang berbeda dengan mengklik tombol “Save As”.

7. Gunakan Bagan Batang Berkerumun Anda

Selesai! Anda sekarang dapat menggunakan bagan batang berkerumun yang telah Anda urutkan. Anda dapat menggunakan bagan ini untuk menampilkan data dengan cara yang visual dan mudah dimengerti. Selamat mencoba!
