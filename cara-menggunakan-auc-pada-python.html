Panduan Lengkap Cara Menggunakan Auc Pada Python

AUC adalah singkatan dari Area Under the Curve. Konsep ini biasanya digunakan dalam Machine Learning untuk mengukur kinerja model prediksi. AUC adalah metrik yang sangat populer, karena mudah dipahami dan dapat digunakan untuk model prediksi yang berbeda. Dalam tutorial ini, kita akan melihat cara menghitung AUC pada Python.

Apa itu AUC?

AUC adalah metrik kinerja yang digunakan untuk mengukur kemampuan model prediksi dalam membedakan antara label positif dan negatif. AUC adalah sebuah jumlah yang berkisar antara 0 dan 1, yang menjelaskan tingkat kemampuan model dalam membedakan antara label positif dan negatif. Semakin tinggi nilai AUC, semakin baik model dalam memprediksi label. Metrik ini paling sering digunakan untuk model klasifikasi biner, namun juga dapat digunakan untuk model klasifikasi multi-label.

Cara Menghitung AUC Pada Python

Untuk menghitung AUC pada Python, kita dapat menggunakan library scikit-learn. Library ini menyediakan fungsi roc_auc_score yang dapat digunakan untuk menghitung AUC. Fungsi ini mengambil dua argumen: label prediksi dan label aktual. Label prediksi adalah hasil prediksi dari model, sedangkan label aktual adalah label yang sebenarnya dari data uji. Fungsi ini kemudian menghitung AUC dari hasil prediksi dan label aktual dan mengembalikan nilai AUC.

Untuk menggunakan fungsi ini, kita harus memasukkan prediksi kita ke dalam bentuk array atau matriks. Kita juga harus memastikan bahwa prediksi dan label aktual berada dalam bentuk yang sama. Jika prediksi kita berupa array, maka label aktual juga harus berupa array. Jika prediksi kita berupa matriks, maka label aktual juga harus berupa matriks.

Contoh Penggunaan AUC Pada Python

Untuk menggunakan fungsi roc_auc_score, kita dapat melakukan hal berikut. Pertama, kita harus memasukkan prediksi kita dan label aktual ke dalam bentuk yang sesuai. Kedua, kita harus membuat array atau matriks dari prediksi kita dan label aktual. Terakhir, kita dapat menggunakan fungsi roc_auc_score untuk menghitung AUC dari prediksi dan label aktual kita.

Berikut adalah contoh sederhana tentang cara menggunakan AUC pada Python. Kita akan menggunakan prediksi kita yang berupa array dan label aktual yang juga berupa array. Kita juga akan menggunakan library scikit-learn untuk menghitung AUC dari prediksi dan label aktual kita.

from sklearn.metrics import roc_auc_score
 
 # Memasukkan prediksi kita dan label aktual ke dalam array
 predictions = [1, 0, 1, 0, 0, 1, 1, 0]
 true_labels = [1, 0, 0, 0, 0, 1, 0, 0]
 
 # Menghitung AUC
 auc = roc_auc_score(true_labels, predictions)
 
 print(auc) # Output: 0.625

Dalam contoh di atas, kita memasukkan prediksi kita dan label aktual ke dalam array. Kita kemudian menggunakan fungsi roc_auc_score untuk menghitung AUC. Fungsi ini kemudian mengembalikan nilai AUC yaitu 0.625.

Kesimpulan

AUC adalah metrik yang sering digunakan untuk mengukur kinerja model prediksi. Metrik ini dapat digunakan untuk model klasifikasi biner maupun multi-label. Untuk menghitung AUC pada Python, kita dapat menggunakan library scikit-learn. Library ini menyediakan fungsi roc_auc_score yang dapat digunakan untuk menghitung AUC. Ini adalah panduan singkat tentang cara menggunakan AUC pada Python.
