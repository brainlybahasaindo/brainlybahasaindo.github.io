Cara Membuat Shortcode Di Wordpress Tanpa Plugin

Apa itu Shortcode?

Shortcode adalah kode pendek yang Anda masukkan dalam artikel atau halaman Anda untuk menampilkan fungsi tertentu. Shortcode membantu Anda menampilkan fungsi yang berbeda tanpa harus menulis kode panjang. Dengan shortcode, Anda dapat membuat, misalnya, tabel, galeri gambar, formulir kontak, dan banyak lagi. Di WordPress, Anda dapat membuat dan menggunakan shortcode tanpa harus menggunakan plugin tambahan.

Cara Membuat Shortcode di WordPress Tanpa Plugin

Berikut adalah langkah-langkah untuk membuat shortcode di WordPress tanpa plugin:

1. Buat Fungsi Shortcode

Pertama, buat fungsi untuk membuat shortcode Anda. Anda akan menggunakan fungsi ini untuk menentukan bagaimana shortcode akan ditampilkan di halaman atau artikel Anda. Jika Anda menggunakan plugin, Anda dapat mengikuti langkah-langkah yang diberikan oleh plugin. Namun, jika Anda tidak menggunakan plugin, Anda harus menulis fungsi Anda sendiri.

Fungsi Anda akan menggunakan dua parameter. Fungsi pertama adalah parameter opsional, yang Anda gunakan untuk menentukan parameter tambahan untuk shortcode. Contohnya, jika Anda membuat galeri gambar, Anda dapat menggunakan parameter opsional untuk menentukan jumlah gambar yang akan ditampilkan. Fungsi kedua adalah parameter ikatan, yang akan menyimpan konten yang ditampilkan di antara tag shortcode.

2. Buat Shortcode

Kemudian, buat shortcode Anda. Anda dapat menggunakan fungsi WordPress add_shortcode untuk membuat shortcode. Anda dapat menggunakan nama bebas seperti yang Anda inginkan untuk shortcode Anda. Namun, pastikan Anda tidak menggunakan shortcode yang sudah ada. Contohnya, jika Anda membuat shortcode galeri, Anda dapat menggunakan nama “gallery” atau “galeri”.

3. Tambahkan Fungsi Ke Tema Anda

Setelah Anda membuat shortcode, Anda harus menambahkannya ke tema WordPress Anda. Anda dapat melakukannya dengan menambahkan fungsi Ke file functions.php dari tema Anda. Anda dapat menemukan file functions.php di folder tema Anda. Jika Anda menggunakan tema default WordPress, Anda dapat menemukannya di folder wp-content / themes / tema Anda.

4. Gunakan Shortcode di Halaman Anda

Ketika Anda selesai membuat shortcode Anda, Anda dapat mulai menggunakannya di halaman atau artikel Anda. Untuk menggunakannya, masukkan nama shortcode yang Anda buat di antara tag dua kurung kotak. Jika Anda membuat shortcode galeri, contohnya, Anda dapat menggunakan [gallery]. Jika Anda menambahkan parameter opsional, Anda dapat menggunakannya dengan menambahkan nama parameter dan nilainya ke dalam kurung kotak. Contohnya, [gallery num=10] akan menampilkan 10 gambar di halaman Anda.

Kesimpulan

Itulah cara membuat shortcode di WordPress tanpa plugin. Membuat shortcode adalah cara yang bagus untuk menambahkan fungsi yang berbeda ke situs WordPress Anda. Meskipun ada plugin yang membuatnya lebih mudah, Anda juga dapat membuat shortcode Anda sendiri tanpa plugin. Dengan mengikuti panduan di atas, Anda akan dapat membuat shortcode Anda sendiri dengan cepat dan mudah.
