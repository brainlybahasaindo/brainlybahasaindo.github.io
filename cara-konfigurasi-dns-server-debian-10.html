Panduan Konfigurasi Dns Server Debian 10

Debian 10 telah dirilis sejak 2019 dan memiliki banyak fitur baru yang menarik. Salah satu fitur yang menarik adalah konfigurasi DNS server. DNS server adalah sistem yang mengkonversi nama domain menjadi alamat IP. Ini membuat proses mencari dan mengakses website lebih mudah. Anda dapat mengkonfigurasi server DNS Debian 10 dengan cepat dan mudah.

Pengaturan DNS Server di Debian 10

Konfigurasi DNS server di Debian 10 sangat mudah. Berikut adalah langkah-langkah yang harus Anda ikuti:

Langkah 1: Instal Paket DNS

Pertama, Anda harus menginstal paket DNS. Paket ini berisi semua alat dan berkas yang diperlukan untuk mengkonfigurasi server DNS. Anda dapat menginstal paket DNS dengan menjalankan perintah berikut:

sudo apt-get install bind9 bind9utils bind9-doc

Setelah menginstall paket, Anda harus mengkonfigurasi server DNS. Untuk melakukan ini, Anda harus mengedit file konfigurasi Bind9. File ini dapat ditemukan di /etc/bind/named.conf.

Langkah 2: Konfigurasi File named.conf

File named.conf adalah file konfigurasi utama dari server DNS. File ini berisi informasi tentang nama domain, alamat IP, dan lokasi lainnya yang berhubungan dengan server DNS. Untuk mengkonfigurasi file, Anda harus mengeditnya dengan perintah berikut:

sudo nano /etc/bind/named.conf

Setelah mengedit file, Anda harus menyimpan dan menutupnya. Jika Anda telah menyimpan perubahan, Anda harus mengaktifkan ulang server DNS. Anda dapat melakukannya dengan menjalankan perintah berikut:

sudo /etc/init.d/bind9 restart

Langkah 3: Konfigurasi File Zone

Selanjutnya, Anda harus mengkonfigurasi file zona. File ini berisi informasi tentang nama domain, alamat IP, dan lokasi lainnya yang berhubungan dengan domain. Anda dapat mengkonfigurasi file zona dengan menjalankan perintah berikut:

sudo nano /etc/bind/named.conf.local

Setelah mengedit file, Anda harus menyimpan dan menutupnya. Jika Anda telah menyimpan perubahan, Anda harus mengaktifkan ulang server DNS. Anda dapat melakukannya dengan menjalankan perintah berikut:

sudo /etc/init.d/bind9 restart

Langkah 4: Menambahkan Domain

Setelah mengkonfigurasi file konfigurasi dan file zona, Anda dapat mulai menambahkan domain. Untuk menambahkan domain, Anda harus mengedit file zona yang Anda konfigurasi sebelumnya. File ini berisi informasi tentang nama domain, alamat IP, dan lokasi lainnya yang berhubungan dengan domain. Anda dapat mengedit file dengan menjalankan perintah berikut:

sudo nano /etc/bind/named.conf.local

Di dalam file, Anda harus menambahkan baris berikut untuk menambahkan domain baru:

zone "nama_domain" { type master; file "/etc/bind/zone/nama_domain.db"; };

Setelah menambahkan baris, Anda harus menyimpan dan menutup file. Jika Anda telah menyimpan perubahan, Anda harus mengaktifkan ulang server DNS. Anda dapat melakukannya dengan menjalankan perintah berikut:

sudo /etc/init.d/bind9 restart

Langkah 5: Menambahkan Alamat IP dan Nama Host

Setelah menambahkan domain, Anda harus menambahkan alamat IP dan nama host. Anda dapat melakukannya dengan mengedit file zone yang telah Anda buat. File ini berisi informasi tentang nama domain, alamat IP, dan lokasi lainnya yang berhubungan dengan domain. Anda dapat mengedit file dengan menjalankan perintah berikut:

sudo nano /etc/bind/zone/nama_domain.db

Anda harus menambahkan baris berikut untuk menambahkan alamat IP dan nama host:

A nama_host alamat_ip

Setelah menambahkan baris, Anda harus menyimpan dan menutup file. Jika Anda telah menyimpan perubahan, Anda harus mengaktifkan ulang server DNS. Anda dapat melakukannya dengan menjalankan perintah berikut:

sudo /etc/init.d/bind9 restart

Langkah 6: Melihat Hasil Konfigurasi DNS

Setelah mengkonfigurasi server DNS, Anda dapat melihat hasil konfigurasi dengan menjalankan perintah berikut:

dig nama_domain

Perintah ini akan menampilkan informasi tentang domain yang telah Anda konfigurasi. Jika Anda melihat informasi yang benar, itu berarti Anda telah berhasil mengkonfigurasi server DNS Anda.

Kesimpulan

Konfigurasi DNS server di Debian 10 sangat mudah. Anda hanya perlu menginstal paket yang diperlukan, mengkonfigurasi file konfigurasi dan file zona, menambahkan domain, menambahkan alamat IP dan nama host, dan melihat hasil konfigurasi. Dengan mengikuti langkah-langkah ini, Anda dapat dengan mudah mengkonfigurasi server DNS Debian 10.
