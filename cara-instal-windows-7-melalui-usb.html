Cara Instal Windows 7 Melalui Usb: Langkah-Langkah Mudah

Terbaik Pelajaran bahasa indonesia tips dan trik website . Cari apa saja topik Pelajaran bahasa indonesia ide di website ini.

Windows 7 merupakan sistem operasi yang dikeluarkan oleh Microsoft pada tahun 2009. Kini, Windows 7 masih banyak digunakan di seluruh dunia. Dengan Windows 7, kamu bisa memainkan beragam game, menjalankan berbagai aplikasi, serta melakukan banyak hal lainnya dengan mudah.

Namun, instalasi Windows 7 agak sedikit rumit. Jika kamu ingin menginstal Windows 7 di komputer atau laptop, kamu harus memasang Windows 7 melalui USB. Langkah-langkahnya cukup mudah, asalkan kamu tahu caranya. Berikut ini adalah cara instal Windows 7 melalui USB yang mudah.

1. Persiapkan USB

Langkah pertama adalah persiapan. USB yang digunakan harus memiliki kapasitas setidaknya 4GB. Selain itu, pastikan bahwa file ISO Windows yang akan digunakan sudah berada di USB. Jika belum, silakan buat file ISO Windows 7 dengan menggunakan aplikasi seperti UltraISO atau WinRAR.

2. Masuk ke Boot Setup

Setelah persiapan selesai, masuk ke Boot Setup. Caranya, kamu tinggal menekan tombol F10 atau F2 saat komputer atau laptop kamu sedang dinyalakan. Ini akan membuka menu Boot Setup. Di sini, kamu bisa mengatur boot device ke USB. Apabila sudah selesai, silakan tekan F10 untuk menyimpan pengaturan.

3. Jalankan Instalasi Windows 7

Setelah masuk ke Boot Setup, komputer atau laptop kamu akan secara otomatis me-reboot. Saat proses reboot, tekan tombol F12. Ini akan membuka menu boot device. Di sini, silakan pilih USB sebagai boot device. Setelah itu, Windows 7 akan berjalan dan mulai menginstal sistem operasinya.

4. Lengkapi Proses Instalasi

Setelah instalasi dimulai, kamu hanya perlu menunggu beberapa saat. Proses instalasi Windows 7 berlangsung cukup cepat. Selanjutnya, pada saat proses instalasi selesai, kamu hanya perlu mengikuti beberapa petunjuk yang diberikan oleh sistem. Setelah itu, Windows 7 pun akan berhasil diinstal di komputer atau laptop kamu.

Kesimpulan

Nah, itulah cara instal Windows 7 melalui USB. Prosesnya cukup mudah, asalkan kamu mengikuti langkah-langkah di atas dengan tepat. Selain itu, jangan lupa untuk membuat file ISO Windows 7 terlebih dahulu. Jika sudah, kamu bisa memasang Windows 7 dengan mudah. Selamat mencoba!
