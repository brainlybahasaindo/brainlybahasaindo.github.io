Cara Mendapatkan Argument Perintah Baris Di Php

Menggunakan bahasa PHP untuk menulis skrip? Jika iya, maka Anda pasti akan memerlukan cara untuk mendapatkan argumen berbasis perintah baris. Argumen ini adalah data yang diberikan kepada skrip ketika skrip dimulai dari baris perintah. Argumen ini dapat berupa teks, angka, atau kombinasi keduanya. Untuk menggunakan argumen ini, Anda harus mengetahui cara mendapatkan argumen perintah baris di PHP.

Mengapa Anda Memerlukan Argumen Perintah Baris?

Argumen perintah baris sangat berguna karena memungkinkan Anda untuk menyediakan data yang diperlukan oleh skrip PHP tanpa mengubah skrip itu sendiri. Hal ini juga memungkinkan Anda untuk melakukan tugas-tugas seperti menjalankan skrip secara berulang tanpa mengubah skrip itu sendiri. Contohnya, jika Anda memiliki skrip PHP untuk mengunduh data dari situs web, Anda dapat menggunakan argumen perintah baris untuk memberikan tautan situs web yang ingin Anda unduh. Ini berarti Anda dapat menjalankan skrip PHP tanpa harus mengubah skrip itu sendiri.

Bagaimana Cara Mendapatkan Argumen Perintah Baris di PHP?

Untuk mendapatkan argumen perintah baris, Anda harus menggunakan fungsi bawaan PHP yang disebut getopt(). Fungsi ini akan membaca argumen perintah baris dan mengembalikan array berisi argumen. Array tersebut akan berisi kunci-nilai, di mana kunci adalah argumen perintah baris, dan nilainya adalah nilai yang diberikan untuk argumen perintah baris tersebut. Contohnya, jika Anda menjalankan skrip PHP seperti ini:

php skrip.php -u http://example.com

Kemudian, array yang dikembalikan oleh getopt() akan berisi kunci-nilai seperti ini:

[ 'u'=>'http://example.com' ]

Kunci adalah argumen -u, dan nilainya adalah URL yang diberikan. Anda dapat menggunakan array ini untuk mengakses nilai argumen perintah baris.

Cara Menggunakan Argumen Perintah Baris di PHP

Setelah Anda mendapatkan argumen perintah baris di PHP, Anda dapat menggunakannya dalam skrip PHP Anda. Salah satu cara termudah untuk melakukannya adalah dengan menggunakan if statement. Contoh berikut menunjukkan cara menggunakan argumen perintah baris di PHP:

$options = getopt("u:");
 
 if(isset($options['u'])) {
 $url = $options['u'];
 echo "The URL is $url";
 }

Pada contoh di atas, fungsi getopt() dipanggil untuk membaca argumen perintah baris. Kemudian, if statement digunakan untuk memeriksa apakah argumen -u diberikan atau tidak. Jika argumen telah diberikan, maka nilainya akan disimpan ke dalam variabel $url dan dicetak ke layar. Dengan cara ini, Anda bisa menggunakan argumen perintah baris dalam skrip PHP Anda.

Cara Lain Mendapatkan Argumen Perintah Baris di PHP

Selain menggunakan fungsi getopt(), Anda juga dapat menggunakan fungsi bawaan PHP lainnya untuk mendapatkan argumen perintah baris. Fungsi ini adalah $_SERVER['argv']. Fungsi ini akan mengembalikan array berisi argumen perintah baris. Array tersebut akan berisi argumen perintah baris dalam urutan berurutan. Contohnya, jika Anda menjalankan skrip PHP dengan argumen perintah baris seperti ini:

php skrip.php -u http://example.com

Array yang dikembalikan oleh $_SERVER['argv'] akan berisi:

[ 'skrip.php', '-u', 'http://example.com' ]

Anda dapat menggunakan array ini untuk mendapatkan argumen perintah baris yang diperlukan oleh skrip PHP Anda.

Kesimpulan

Memiliki kemampuan untuk mendapatkan argumen perintah baris di PHP sangat berguna. Fungsi getopt() dan $_SERVER['argv'] sangat berguna untuk tugas ini. Dengan menggunakan salah satu fungsi ini, Anda dapat dengan mudah membuat skrip PHP Anda untuk menerima argumen berbasis perintah baris. Selamat mencoba!
