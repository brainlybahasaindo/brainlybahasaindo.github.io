Cara Cek Versi Php Yang Digunakan

Ketika Anda menggunakan platform web, seperti WordPress, versi PHP yang digunakan penting untuk diketahui. Terutama ketika Anda ingin menambahkan plugin dan tema baru, mengubah tampilan situs web, atau mengubah bagian lain dari situs web Anda. Anda harus mengetahui versi PHP yang sedang berjalan agar memastikan bahwa semua fungsionalitas yang diinginkan berjalan dengan baik. Jadi, bagaimana Anda bisa mengetahui versi PHP yang Anda gunakan?

Cara Cek Versi PHP Anda

Berikut adalah beberapa cara yang dapat Anda lakukan untuk mengetahui versi PHP yang Anda gunakan:

Cara 1 - Gunakan CPanel

Salah satu cara paling mudah untuk mengetahui versi PHP Anda adalah dengan menggunakan cPanel. Jika Anda menggunakan hosting web yang menawarkan cPanel sebagai control panel, Anda dapat masuk ke cPanel dan menemukan informasi tentang versi PHP yang Anda gunakan. Biasanya, Anda dapat menemukan informasi ini di bagian 'Software' atau 'Advanced'. Namun, jika Anda tidak dapat menemukan informasi ini di cPanel Anda, Anda dapat mencoba cara lain di bawah ini.

Cara 2 - Gunakan phpinfo()

Cara lain yang dapat Anda lakukan untuk mengetahui versi PHP yang Anda gunakan adalah dengan menggunakan fungsi phpinfo(). Ini adalah fungsi PHP yang menampilkan informasi tentang konfigurasi PHP Anda. Anda dapat menggunakan fungsi ini dengan membuat file baru yang disebut phpinfo.php dan menempatkan kode berikut di dalamnya:

Ketika Anda mengunggah file ini ke server web Anda, Anda dapat menemukan versi PHP yang Anda gunakan dengan mengunjungi URL dimana Anda mengunggah file tersebut. Misalnya, jika Anda mengunggah file ke folder public_html, Anda dapat mengunjungi alamat http://yourdomain.com/phpinfo.php untuk melihat informasi tentang versi PHP Anda.

Cara 3 - Gunakan Perintah Shell

Jika Anda menggunakan hosting yang menawarkan akses shell, Anda dapat menggunakan perintah yang disebut php -v untuk mengetahui versi PHP yang Anda gunakan. Jika Anda tidak tahu bagaimana cara menggunakan shell, Anda dapat meminta bantuan dari tim dukungan hosting Anda untuk membantu Anda menjalankan perintah ini.

Cara 4 - Tanyakan Pada Tim Dukungan Hosting Anda

Jika Anda tidak dapat menggunakan salah satu cara di atas untuk mengetahui versi PHP Anda, Anda selalu dapat menghubungi tim dukungan hosting Anda untuk membantu Anda. Tim dukungan hosting Anda dapat memberi tahu Anda versi PHP yang sedang berjalan pada server web Anda.

Kesimpulan

Seperti yang Anda lihat, ada beberapa cara yang dapat Anda lakukan untuk mengetahui versi PHP yang Anda gunakan. Jika Anda tidak dapat menggunakan salah satu cara di atas, Anda selalu dapat menghubungi tim dukungan hosting Anda untuk meminta bantuan. Jadi, cobalah salah satu dari cara di atas untuk mengetahui versi PHP yang Anda gunakan.
