Cara Menambahkan Ukuran Kertas Khusus Di Microsoft Excel

Jika Anda sering bekerja dengan file Excel, Anda mungkin pernah menemukan bahwa kertas default yang tersedia di Excel terkadang tidak sesuai dengan kebutuhan Anda. Untungnya, Anda dapat menambahkan ukuran kertas khusus ke Microsoft Excel di tahun 2023. Berikut adalah panduan untuk menambahkan ukuran kertas khusus ke Microsoft Excel.

Bagaimana Cara Menambahkan Ukuran Kertas Khusus di Microsoft Excel?

Untuk menambahkan ukuran kertas khusus di Microsoft Excel di tahun 2023, Anda harus melakukan beberapa langkah. Langkah-langkahnya adalah sebagai berikut:

Langkah 1: Buka File Excel

Pertama-tama, Anda harus membuka file Excel yang ingin Anda ubah. Untuk ini, Anda dapat menggunakan aplikasi Microsoft Excel di PC atau laptop. Anda juga dapat menggunakan aplikasi Microsoft Excel di perangkat iOS atau Android jika Anda menggunakan aplikasi Microsoft Office.

Langkah 2: Pilih Ukuran Kertas Khusus

Setelah membuka file Excel, Anda harus memilih ukuran kertas khusus yang ingin Anda tambahkan. Anda dapat memilih ukuran kertas khusus dari daftar yang tersedia di menu atau Anda juga dapat membuat ukuran kertas khusus yang sesuai dengan kebutuhan Anda. Setelah memilih ukuran kertas khusus, Anda dapat melanjutkan ke langkah berikutnya.

Langkah 3: Simpan File Excel

Setelah Anda memilih ukuran kertas khusus, Anda harus menyimpan file Excel yang telah Anda ubah. Untuk ini, Anda harus mengeklik tombol "Simpan" di menu atau Anda juga dapat menggunakan tombol "Ctrl + S" untuk menyimpan file Excel. Setelah menyimpan file Excel, Anda dapat menutup aplikasi dan membuka file lagi untuk melihat perubahan yang telah Anda lakukan.

Langkah 4: Cetak File Excel

Setelah Anda menyimpan file Excel yang telah Anda ubah, Anda dapat mencetaknya dengan menggunakan printer yang tersedia. Untuk ini, Anda harus memilih opsi "Cetak" di menu atau Anda juga dapat menggunakan tombol "Ctrl + P" untuk mencetak file Excel. Setelah mencetak file Excel, Anda akan melihat bahwa ukuran kertas khusus yang Anda tambahkan telah diprint.

Kesimpulan

Dari penjelasan di atas, dapat disimpulkan bahwa Anda dapat menambahkan ukuran kertas khusus ke Microsoft Excel di tahun 2023. Untuk ini, Anda harus membuka file Excel, memilih ukuran kertas khusus dan menyimpannya. Setelah itu, Anda dapat mencetak file Excel dan melihat bahwa ukuran kertas khusus yang Anda tambahkan telah diprint. Semoga informasi di atas bermanfaat bagi Anda.
