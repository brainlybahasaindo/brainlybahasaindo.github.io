Cara Mudah Mengubah Daftar Python Ke String Dengan Koma Dan Tanda Kurung Di 2023

Terbaik Pelajaran bahasa indonesia tips dan trik website . Cari apa saja topik Pelajaran bahasa indonesia ide di website ini.

Tahun 2023, Python telah menjadi bahasa pemrograman terpopuler di dunia. Dengan sintaks yang mudah dipahami, bahasa yang mudah dipelajari, dan komunitas yang luas, Python telah melampaui banyak bahasa pemrograman lainnya. Salah satu kegunaan yang paling umum dari Python adalah mengonversi daftar ke string. Di sini, kita akan melihat cara mudah untuk mengonversi daftar Python ke string dengan menggunakan koma dan tanda kurung.

Pertama-tama, pastikan Anda memiliki daftar yang ingin Anda konversi. Daftar ini harus dalam bentuk list atau tuple. Ubah daftar Anda menjadi tipe data yang sesuai jika perlu. Kemudian, gunakan fungsi str.join () yang akan menggabungkan elemen-elemen dalam daftar Anda menjadi string. Jangan lupa untuk menambahkan koma dan tanda kurung saat Anda menggunakan fungsi ini. Berikut contohnya:

Contoh:

daftar = ["Gula", "Garam", "Merica", "Kayu Manis"] string = str.join (",", daftar) print (string)

Output: Gula,Garam,Merica,Kayu Manis

Di contoh di atas, daftar awal kita adalah ["Gula", "Garam", "Merica", "Kayu Manis"]. Kita kemudian menggunakan fungsi str.join () untuk menggabungkan elemen-elemen dalam daftar kita menjadi string. Hasilnya adalah Gula,Garam,Merica,Kayu Manis. Anda dapat melihat bahwa kita telah berhasil mengonversi daftar Python ke string dengan menggunakan koma dan tanpa tanda kurung.

Penggunaan Tanda Kurung

Tanda kurung dapat digunakan ketika Anda ingin menambahkan kontrol pada string hasil output. Anda dapat menggunakan tanda kurung untuk membungkus string hasil output Anda. Berikut contohnya:

Contoh:

daftar = ["Gula", "Garam", "Merica", "Kayu Manis"] string = str.join (",", daftar) print ("(" + string + ")")

Output: (Gula,Garam,Merica,Kayu Manis)

Di contoh di atas, daftar awal kita adalah ["Gula", "Garam", "Merica", "Kayu Manis"]. Kita kemudian menggunakan fungsi str.join () untuk menggabungkan elemen-elemen dalam daftar kita menjadi string. Kami kemudian menambahkan tanda kurung di awal dan akhir string hasil output untuk menambahkan kontrol. Hasilnya adalah (Gula,Garam,Merica,Kayu Manis). Anda dapat melihat bahwa kita telah berhasil mengonversi daftar Python ke string dengan menggunakan koma dan tanda kurung.

Menggunakan Fungsi map ()

Selain menggunakan fungsi str.join (), Anda juga dapat menggunakan fungsi map () untuk mengonversi daftar Python ke string dengan koma dan tanda kurung. Fungsi map () akan memetakan elemen-elemen dalam daftar Anda ke string. Berikut contohnya:

Contoh:

daftar = ["Gula", "Garam", "Merica", "Kayu Manis"] string = map (str, daftar) print (string)

Output: ("Gula", "Garam", "Merica", "Kayu Manis")

Di contoh di atas, daftar awal kita adalah ["Gula", "Garam", "Merica", "Kayu Manis"]. Kita kemudian menggunakan fungsi map () untuk menggabungkan elemen-elemen dalam daftar kita menjadi string. Hasilnya adalah ("Gula", "Garam", "Merica", "Kayu Manis"). Anda dapat melihat bahwa kita telah berhasil mengonversi daftar Python ke string dengan menggunakan koma dan tanda kurung.

Kesimpulan

Dalam artikel ini, kita melihat cara mudah untuk mengonversi daftar Python ke string dengan menggunakan koma dan tanda kurung. Kita menyaksikan dua cara yang berbeda untuk melakukan ini, yaitu menggunakan fungsi str.join () dan fungsi map (). Selain itu, kita juga melihat cara menggunakan tanda kurung untuk menambahkan kontrol pada string hasil output. Dengan demikian, kita telah belajar cara mengonversi daftar Python ke string dengan koma dan tanda kurung.
