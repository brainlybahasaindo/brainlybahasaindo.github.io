Cara Menggunakan Workbook Links Excel Online

Untuk menggunakan Workbook Links Excel Online, kamu harus memiliki akun Microsoft Office. Setelah membuat akun, kamu harus masuk ke akun Microsoft Office dan mengikuti langkah-langkah berikut:

  1. Klik tombol “Mulai” di layar utama.
  2. Klik tombol “Buat File Baru”.
  3. Pilih jenis file yang ingin kamu buat.
  4. Pilih “Workbook Links Excel Online”.
  5. Ketik nama file yang ingin kamu buat di kotak teks.
  6. Klik tombol “Buat”.
