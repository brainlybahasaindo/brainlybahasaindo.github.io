Cara Menggunakan Mysqli Pada Php 7

Terbaik Pelajaran bahasa indonesia tips dan trik website . Cari apa saja topik Pelajaran bahasa indonesia ide di website ini.

MySQLi adalah sebuah library yang menyediakan API untuk mengakses basis data MySQL dari bahasa pemrograman PHP. Dengan menggunakan MySQLi, kita dapat mengakses database MySQL dengan mudah dan cepat. Perlu dicatat bahwa library MySQLi baru tersedia mulai dari versi PHP 5.0.

Instalasi MySQLi

MySQLi dalam PHP 7 telah menjadi bagian dari PHP itu sendiri, sehingga Anda tidak perlu melakukan instalasi tambahan untuk menggunakan MySQLi. Namun, Anda harus memastikan bahwa ekstensi MySQLi telah diaktifkan. Untuk mengeceknya, Anda dapat menggunakan fungsi phpinfo () yang tersedia di PHP. Jika Anda menggunakan cPanel, Anda juga dapat menggunakan menu phpMyAdmin untuk mengaktifkan ekstensi MySQLi.

Koneksi ke Server MySQLi

Untuk dapat menggunakan MySQLi, Anda harus terlebih dahulu menghubungkan ke server MySQL. Anda dapat melakukannya dengan menggunakan fungsi mysqli_connect () di PHP. Fungsi ini membutuhkan beberapa argumen untuk menghubungkan ke server. Argumen pertama adalah nama host dari server, argumen kedua adalah nama pengguna, argumen ketiga adalah kata sandi, dan argumen terakhir adalah nama database.

Setelah Anda berhasil menghubungkan ke server, Anda dapat menggunakan variabel koneksi yang dikembalikan oleh fungsi mysqli_connect () untuk melakukan berbagai operasi pada basis data.

Menjalankan Query

Setelah Anda terhubung ke server MySQL, Anda dapat mulai menggunakan bahasa query untuk mengambil data dari basis data. Anda dapat menggunakan fungsi mysqli_query () untuk menjalankan query. Fungsi ini membutuhkan dua argumen, yaitu koneksi yang dikembalikan oleh fungsi mysqli_connect () dan string query.

Misalnya, Anda dapat menggunakan query berikut untuk mengambil data dari tabel mahasiswa:

SELECT * FROM mahasiswa;

Untuk menjalankan query ini, Anda dapat menggunakan fungsi mysqli_query () seperti yang ditunjukkan di bawah ini:

$query ="SELECT * FROM mahasiswa";
 $result = mysqli_query($connection, $query);

Mengambil Data

Setelah Anda berhasil menjalankan query, Anda dapat menggunakan fungsi mysqli_fetch_assoc () untuk mengambil hasilnya. Fungsi ini mengembalikan baris berikutnya dari hasil query sebagai array associative. Anda dapat menggunakan array ini untuk mengambil nilai kolom tertentu yang diperlukan.

Misalnya, Anda dapat menggunakan kode berikut untuk mengambil nama dan nilai dari tabel mahasiswa:

while($row = mysqli_fetch_assoc($result)) {
 echo "Nama : ".$row['nama']." Nilai : ".$row['nilai']."
";
 }

Menutup Koneksi

Setelah Anda selesai menggunakan MySQLi, jangan lupa untuk menutup koneksi dengan menggunakan fungsi mysqli_close (). Fungsi ini akan menutup koneksi yang dibuka oleh fungsi mysqli_connect ().

Misalnya, Anda dapat menggunakan kode berikut untuk menutup koneksi:

mysqli_close($connection);

Kesimpulan

MySQLi adalah library yang tersedia dalam PHP 7 yang memungkinkan Anda untuk mengakses basis data MySQL dengan mudah. Instalasi MySQLi tidak diperlukan karena telah menjadi bagian dari PHP itu sendiri. Untuk dapat menggunakan MySQLi, Anda harus terlebih dahulu menghubungkan ke server MySQL menggunakan fungsi mysqli_connect (). Anda dapat menjalankan query menggunakan fungsi mysqli_query () dan mengambil hasilnya menggunakan fungsi mysqli_fetch_assoc (). Setelah selesai menggunakan MySQLi, jangan lupa untuk menutup koneksi dengan menggunakan fungsi mysqli_close ().
