Mengubah Timestamp Python Menjadi Waktu Tanggal

Apa Itu Timestamp?

Timestamp adalah sebutan untuk menggambarkan jumlah detik yang berlalu sejak 1 Januari 1970. Timestamp juga dapat digunakan untuk menyimpan tanggal dan waktu dalam bentuk numerik. Pada Python, timestamp dapat digunakan untuk menyimpan dan mengubah waktu dan tanggal menjadi bentuk numerik. Dengan cara ini, kita dapat menggunakan timestamp untuk mengonversi waktu dan tanggal ke dalam bentuk numerik dan sebaliknya.

Bagaimana Cara Mengubah Timestamp Python ke Waktu Tanggal?

Untuk mengubah timestamp Python menjadi waktu tanggal, kita dapat menggunakan fungsi time() dari library time. Fungsi time() memungkinkan kita untuk mengonversi tanggal dan waktu ke dalam bentuk timestamp. Untuk mengubah timestamp Python ke waktu tanggal, kita dapat menggunakan fungsi datetime.fromtimestamp(). Fungsi ini memungkinkan kita untuk mengubah timestamp menjadi waktu dan tanggal dalam format yang ditentukan. Format yang dapat dipilih antara lain: YYYY-MM-DD HH:MM:SS, YYYY/MM/DD HH:MM:SS, dan YYYY-MM-DD.

Contoh Mengubah Timestamp Python Menjadi Waktu Tanggal

Untuk mengubah timestamp Python menjadi waktu tanggal, kita dapat menggunakan contoh berikut ini:

import datetime

timestamp = 1582144400

date_time = datetime.datetime.fromtimestamp(timestamp)

print(date_time)

Output yang dihasilkan adalah: 2020-02-17 15:00:00

Apa Itu Library Time?

Library time adalah library yang berisi fungsi dan konstanta yang berhubungan dengan pengolahan waktu. Library ini dapat digunakan untuk mengubah format waktu dan tanggal dalam Python. Library ini juga menyediakan fungsi seperti time(), localtime(), mktime(), dan clock() yang dapat digunakan untuk mengolah waktu dan tanggal. Library ini juga menyediakan fungsi seperti sleep() yang dapat digunakan untuk menunda eksekusi program selama jangka waktu tertentu.

Kesimpulan

Dengan menggunakan library time dan datetime, kita dapat dengan mudah mengubah timestamp Python menjadi waktu tanggal. Fungsi time() dari library time memungkinkan kita untuk mengonversi tanggal dan waktu menjadi bentuk timestamp, sedangkan fungsi datetime.fromtimestamp() memungkinkan kita untuk mengubah timestamp menjadi waktu dan tanggal dalam format yang ditentukan. Dengan cara ini, kita dapat dengan mudah mengolah waktu dan tanggal dalam Python.
