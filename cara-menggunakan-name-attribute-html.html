Apa itu Atribut Nama?

Atribut nama adalah elemen yang memungkinkan Anda untuk menambahkan informasi tambahan pada tag HTML Anda. Atribut ini digunakan untuk memberi nama kepada suatu tag dan menambahkan informasi tambahan yang berkaitan dengan tag tersebut. Namun, jangan lupa bahwa atribut nama hanya dapat digunakan pada tag HTML tertentu. Anda tidak dapat menambahkan atribut nama ke semua tag HTML.
