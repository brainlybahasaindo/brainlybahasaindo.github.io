HTML adalah sebuah bahasa markup yang digunakan oleh web developer untuk membuat dan mendesain suatu halaman web. HTML secara harfiah berarti HyperText Markup Language. HTML sendiri merupakan bahasa komputer yang dapat membuat struktur dan menghubungkan sebuah halaman web agar dapat dilihat oleh pengguna. HTML sendiri terdiri dari berbagai macam tag, yang digunakan untuk menyusun, membentuk, serta menghubungkan sebuah halaman web.

HTML sendiri merupakan bahasa yang sangat sederhana dan mudah dipahami. HTML juga merupakan bahasa yang dapat dipelajari dengan cepat dan mudah.
