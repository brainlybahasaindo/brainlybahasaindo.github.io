Cara Mudah Memilih Nama Database Dari Information Schema Di Mysql 2023

Terbaik Pelajaran bahasa indonesia tips dan trik website . Cari apa saja topik Pelajaran bahasa indonesia ide di website ini.

MySQL adalah salah satu sistem manajemen basis data relasional yang paling populer saat ini. Menggunakan MySQL, Anda dapat dengan mudah membuat dan mengelola berbagai jenis database. Namun, terkadang, Anda mungkin ingin mengetahui nama database yang dibuat di server MySQL sebelumnya. Untungnya, MySQL telah menyediakan informasi tentang basis data yang dibuat di server MySQL dalam tabel informasi schema. Dalam artikel ini, kami akan menjelaskan cara memilih nama database dari Information Schema di MySQL 2023.

Apa itu Information Schema?

Information Schema (INFORMATION_SCHEMA) adalah sebuah tabel yang disediakan oleh MySQL untuk menyimpan metadata tentang server. Metadata adalah data yang menggambarkan data yang disimpan di server. Metadata dapat berupa informasi tentang tabel, indeks, views, dan lainnya. Information Schema adalah tempat yang tepat untuk mencari informasi tentang nama database di server MySQL.

Bagaimana Cara Mendapatkan Nama Database dari Information Schema?

MySQL menyimpan nama database di tabel INFORMATION_SCHEMA.TABLES. Untuk mendapatkan nama database, Anda dapat menggunakan perintah SELECT berikut:

SELECT TABLE_SCHEMA FROM INFORMATION_SCHEMA.TABLES;

Perintah di atas akan mengembalikan semua nama database yang ada di server MySQL. Anda juga dapat menggunakan klausa WHERE untuk memfilter nama database yang ingin Anda lihat. Sebagai contoh, Anda dapat menggunakan perintah berikut untuk mendapatkan nama database yang diawali dengan huruf “a”:

SELECT TABLE_SCHEMA FROM INFORMATION_SCHEMA.TABLES WHERE TABLE_SCHEMA LIKE 'a%';

Ini adalah cara paling mudah untuk mendapatkan nama database dari Information Schema di MySQL 2023. Jika Anda memiliki pertanyaan atau komentar lain tentang cara memilih nama database dari Information Schema, jangan ragu untuk bertanya di komentar di bawah ini.
