Cara Menggunakan Convert Excel To Txt

Cara menggunakan convert excel to txt cukup mudah. Pertama, anda harus membuka file excel yang akan anda ubah menjadi file teks. Setelah itu, anda bisa mengklik tombol "Save As" yang ada di menu File. Pada jendela yang muncul, anda bisa pilih "Save As Type" sebagai "Text (Tab Delimited)". Setelah itu, anda bisa menyimpan file tersebut dengan mengklik tombol "Save".

Kedua, anda bisa membuka file teks yang sudah anda buat dengan menggunakan program text editor seperti Notepad. Setelah itu, anda bisa mengedit file tersebut sesuai dengan kebutuhan. Jika anda ingin mengubah format file teks, anda bisa menggunakan program text editor untuk melakukannya.

Kekurangan Menggunakan Convert Excel To Txt
