Pengertian Trigger di MySQL

MySQL adalah salah satu platform relational database management system (RDBMS) yang paling populer. Trigger, yang juga merupakan sebuah fitur yang ditawarkan oleh MySQL, digunakan untuk memicu atau mengeksekusi aksi tertentu ketika event tertentu terjadi. MySQL trigger dapat dipicu oleh INSERT, UPDATE, dan DELETE. Sebuah trigger dapat berisi satu atau beberapa perintah, dan juga bisa berisi perintah kompleks. Trigger adalah salah satu cara yang efektif untuk menjaga integritas data pada database MySQL.

Membuat Trigger Penjumlahan di MySQL

Pernyataan CREATE TRIGGER digunakan untuk membuat sebuah trigger di MySQL. Berikut adalah sintaks dasar untuk membuat trigger di MySQL:

CREATE TRIGGER trigger_name trigger_time trigger_event ON table_name FOR EACH ROW BEGIN -- trigger code END;

Di sini, trigger_name adalah nama trigger yang dibuat, trigger_time adalah waktu saat trigger dipicu (yaitu, BEFORE, AFTER, atau INSTEAD OF), trigger_event adalah event yang dipicu (yaitu, INSERT, UPDATE, atau DELETE) dan table_name adalah nama tabel yang ditetapkan untuk trigger. Selanjutnya, trigger_code adalah block kode yang berisi perintah yang akan dieksekusi saat trigger dipicu.

Contoh Trigger Penjumlahan di MySQL

Pada contoh berikut, kita akan membuat sebuah trigger penjumlahan di MySQL. Trigger ini akan dipicu setelah data ditambahkan ke tabel 'orders'. Trigger akan menjumlahkan jumlah total pesanan dan menyimpannya dalam kolom 'total_amount' dari tabel 'orders'.

CREATE TRIGGER orders_after_insert AFTER INSERT ON orders FOR EACH ROW BEGIN SET new.total_amount = new.quantity * new.unit_price; END;

Menghapus Trigger di MySQL

Perintah DROP TRIGGER digunakan untuk menghapus trigger di MySQL. Berikut adalah sintaks dasar untuk menghapus trigger di MySQL:

DROP TRIGGER trigger_name;

Contoh Menghapus Trigger di MySQL

Berikut adalah contoh penggunaan DROP TRIGGER untuk menghapus trigger 'orders_after_insert' yang dibuat sebelumnya:

DROP TRIGGER orders_after_insert;

Kesimpulan

MySQL trigger adalah sebuah fitur yang memungkinkan kita untuk memicu atau mengeksekusi aksi tertentu ketika event tertentu terjadi. Perintah CREATE TRIGGER digunakan untuk membuat trigger di MySQL. Perintah DROP TRIGGER digunakan untuk menghapus trigger di MySQL. Dalam contoh ini, kita telah belajar bagaimana membuat dan menghapus trigger penjumlahan di MySQL.
