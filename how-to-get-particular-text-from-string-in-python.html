Cara Mengambil Teks Tertentu Dari String Di Python

Terbaik Pelajaran bahasa indonesia tips dan trik website . Cari apa saja topik Pelajaran bahasa indonesia ide di website ini.

Python adalah bahasa pemrograman yang populer dan mudah digunakan bagi pemula. Sama seperti bahasa pemrograman lainnya, Python juga dilengkapi dengan berbagai macam fungsi dan operasi. Salah satu operasi yang sering digunakan adalah mengambil teks tertentu dari sebuah string. Dalam artikel ini, kami akan menunjukkan cara untuk mengambil teks tertentu dari string di Python.

Apa itu String?

String adalah tipe data di Python yang digunakan untuk menyimpan kumpulan karakter teks. String dapat berupa teks biasa, angka, simbol, dan karakter lainnya. String biasanya ditulis dengan tanda kutip tunggal atau ganda. Contoh string adalah: "Hello World".

Cara Mengambil Teks Tertentu dari String di Python

Ada beberapa cara yang dapat Anda gunakan untuk mengambil teks tertentu dari string di Python. Beberapa di antaranya adalah dengan menggunakan fungsi built-in Python seperti substring, split, dan find. Berikut adalah beberapa contoh bagaimana cara menggunakan setiap fungsi tersebut.

Cara Menggunakan Substring

Substring adalah fungsi built-in Python yang dapat digunakan untuk mengambil sebagian dari string. Fungsi ini membutuhkan dua argumen, yaitu posisi awal dan panjang substring yang ingin diambil. Berikut adalah contoh bagaimana cara menggunakan substring.

Contoh 1:

Jika Anda ingin mengambil teks dari string "Hello World", Anda dapat menggunakan substring seperti yang ditunjukkan:

 >>> string ="Hello World">>> substring = string[6:11]
 >>> print(substring)
 World

Contoh 2:

Jika Anda ingin mengambil teks dari string "Python Programming", Anda dapat menggunakan substring seperti yang ditunjukkan:

 >>> string ="Python Programming">>> substring = string[0:6]
 >>> print(substring)
 Python

Cara Menggunakan Split

Split adalah fungsi built-in Python yang dapat digunakan untuk memecah string menjadi daftar substring. Fungsi ini juga membutuhkan argumen yaitu delimiter. Delimiter adalah karakter yang digunakan untuk memisahkan string. Berikut adalah contoh bagaimana cara menggunakan split.

Contoh 1:

Jika Anda ingin memecah string "Hello World" menjadi daftar substring, Anda dapat menggunakan split seperti yang ditunjukkan:

 >>> string ="Hello World">>> substring = string.split(' ')
 >>> print(substring)
 ['Hello', 'World']

Contoh 2:

Jika Anda ingin memecah string "Python Programming" menjadi daftar substring, Anda dapat menggunakan split seperti yang ditunjukkan:

 >>> string ="Python Programming">>> substring = string.split(' ')
 >>> print(substring)
 ['Python', 'Programming']

Cara Menggunakan Find

Find adalah fungsi built-in Python yang dapat digunakan untuk menemukan posisi substring dalam string. Fungsi ini membutuhkan dua argumen, yaitu substring yang ingin dicari dan posisi awal dari substring tersebut. Berikut adalah contoh bagaimana cara menggunakan find.

Contoh 1:

Jika Anda ingin menemukan posisi substring "World" dalam string "Hello World", Anda dapat menggunakan find seperti yang ditunjukkan:

 >>> string ="Hello World">>> pos = string.find('World', 0)
 >>> print(pos)
 6

Contoh 2:

Jika Anda ingin menemukan posisi substring "Python" dalam string "Python Programming", Anda dapat menggunakan find seperti yang ditunjukkan:

 >>> string ="Python Programming">>> pos = string.find('Python', 0)
 >>> print(pos)
 0

Kesimpulan

Dalam artikel ini, kami telah menunjukkan cara mengambil teks tertentu dari string di Python. Dengan menggunakan fungsi built-in seperti substring, split, dan find, Anda dapat dengan mudah mengambil teks tertentu dari string di Python. Selamat mencoba!
