Cara Menggunakan Bitnami Wordpress File Manager

Tahun 2023, Bitnami WordPress File Manager telah menjadi salah satu alat pengurusan fail yang paling banyak digunakan. Ini adalah alat yang boleh dipercayai dan kuat yang digunakan oleh para pembangun web untuk menyimpan dan menguruskan fail mereka. Dengan Bitnami WordPress File Manager, anda boleh mencipta, menyunting, menghapus, dan menguruskan fail, folder, dan bahagian laman web anda dengan mudah. Anda juga dapat mengakses fail anda di mana-mana di dunia melalui pelayar web yang selamat.

Pertama, anda perlu mencipta akaun Bitnami WordPress File Manager. Anda boleh melakukannya secara percuma dengan mendaftar di laman web Bitnami. Setelah mendaftar, anda boleh log masuk dan mulakan menggunakan Bitnami WordPress File Manager. Dalam akaun Bitnami, anda boleh menambah fail, folder, dan bahagian laman web anda. Anda juga boleh membuat fail dan folder baru, mengedit fail dan folder yang sedia ada, dan menghapuskan fail dan folder yang tidak diperlukan.

Cara Guna Bitnami WordPress File Manager

Apabila anda telah log masuk ke akaun Bitnami WordPress File Manager anda, anda boleh mula menggunakannya. Terdapat beberapa langkah mudah untuk bermula:

1. Pilih Fail dan Folder

Di Bitnami WordPress File Manager, anda boleh memilih fail dan folder untuk dikemaskini atau dihapuskan. Dengan mengklik pada ikon folder di sebelah kiri, anda boleh melihat fail dan folder yang anda miliki. Anda juga boleh membuat folder baru dengan mengklik butang "Create Folder".

2. Menambah dan Menguruskan Fail

Setelah anda membuat folder baru, anda boleh mulakan menambah fail ke dalam folder tersebut. Anda boleh melakukannya dengan mengklik butang "Add File" di sebelah kiri. Di sana, anda boleh memilih fail dari komputer anda dan memuat naiknya ke dalam folder yang anda pilih.

3. Menyunting dan Menghapuskan Fail

Selain menambah fail ke dalam folder, anda juga boleh menyunting fail yang sedia ada. Untuk melakukannya, anda perlu mengklik butang "Edit" di sebelah kanan fail yang hendak anda sunting. Di sana, anda boleh menyunting fail dengan mudah. Jika anda tidak memerlukan fail tersebut lagi, anda boleh menghapuskannya dengan mengklik butang "Delete".

4. Mengakses Fail di Mana-mana

Bitnami WordPress File Manager juga membenarkan anda mengakses fail anda di mana-mana di dunia melalui pelayar web yang selamat. Anda boleh mengakses fail anda dengan mengklik butang "Access Files" di sebelah kanan folder yang anda miliki. Di sana, anda boleh melihat dan mengakses fail anda dengan pelayar web yang selamat.

Kesimpulan

Itulah cara menggunakan Bitnami WordPress File Manager. Dengan alat ini, anda boleh mencipta, mengedit, menghapuskan, dan menguruskan fail, folder, dan bahagian laman web anda dengan mudah. Anda juga boleh mengakses fail anda di mana-mana di dunia melalui pelayar web yang selamat. Sekiranya anda masih tidak pasti tentang cara menggunakan Bitnami WordPress File Manager, anda boleh bertanya di forum Bitnami, atau mencari bantuan daripada seorang pakar web untuk membantu anda. Jadi, jika anda ingin menguruskan fail anda dengan lebih mudah dan pantas, Bitnami WordPress File Manager adalah pilihan yang tepat.
