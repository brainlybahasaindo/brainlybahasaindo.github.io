Bagaimana Anda Menggunakan Kamus Python untuk Menginisialisasi Data?

Inisialisasi kamus Python adalah proses pembuatan kamus dan menambahkan nilai ke dalamnya. Untuk membuat kamus, Anda harus menggunakan perintah "dict". Ini akan menciptakan kamus kosong yang dapat Anda isi dengan data. Setelah Anda memiliki kamus, Anda dapat menambahkan nilai dengan menggunakan sintaks "kamus [key] = value". Contohnya, jika Anda memiliki kamus yang berisi nama-nama kota dan kode posnya, Anda dapat menambahkan nama kota dan kode pos dengan menggunakan sintaks ini: "kamus ['New York'] ='NY'".

Ketika Anda menambahkan nilai ke dalam kamus, Anda juga dapat mengakses nilai tersebut dengan menggunakan sintaks "kamus [key]". Dalam contoh di atas, Anda dapat mengakses kode pos untuk New York dengan menggunakan sintaks "kamus ['New York']". Anda juga dapat menghapus nilai dari kamus dengan menggunakan sintaks "del kamus [key]". Dengan demikian, Anda dapat menggunakan kamus Python untuk menyimpan dan mengakses data secara efisien.
