Cara Menampilkan Preview Pane Di Windows 7

Terbaik Pelajaran bahasa indonesia tips dan trik website . Cari apa saja topik Pelajaran bahasa indonesia ide di website ini.

Windows 7 adalah sistem operasi yang luar biasa dan memiliki banyak fitur yang dapat membantu Anda melihat, mengakses, dan mengelola file dan folder Anda. Salah satu fitur yang sangat berguna adalah Preview Pane. Ini adalah panel yang dapat Anda gunakan untuk melihat file dan folder yang ada di jendela saat ini tanpa harus mengakses dan membukanya. Dengan menggunakan Preview Pane, Anda dapat melihat file dan folder secara cepat dan mudah tanpa harus mengaksesnya satu per satu. Berikut adalah panduan yang akan membantu Anda menampilkan Preview Pane di Windows 7.

Langkah 1: Membuka Folder Options

Pertama, Anda harus membuka Folder Options. Anda dapat melakukannya dengan mengklik tombol Start di taskbar, ketik "Folder Options" di kotak pencarian, kemudian tekan Enter. Atau Anda dapat klik kanan pada folder, pilih Properties, lalu pilih tab View. Setelah melakukan salah satu cara di atas, Folder Options akan muncul.

Langkah 2: Aktifkan Preview Pane

Di Folder Options, Anda akan melihat beberapa pilihan. Cari opsi yang bernama "Show preview pane". Aktifkan pilihan ini dengan mencentang kotak yang berada di sebelahnya. Pilihan ini akan membuat preview pane muncul di jendela Anda. Anda juga dapat mengatur apakah Anda ingin melihat file gambar, dokumen, atau keduanya di preview pane. Jika Anda hanya ingin melihat gambar, maka Anda harus mencentang kotak "Always show icons, never thumbnails". Jika Anda ingin melihat gambar dan dokumen, maka Anda harus mencentang kotak "Always show icons, never thumbnails" dan "Display file contents".

Langkah 3: Simpan Pengaturan

Setelah Anda mengatur pilihan yang Anda inginkan, jangan lupa untuk menyimpan pengaturan. Caranya adalah dengan mengklik tombol Apply dan OK di Folder Options. Dengan demikian, pengaturan Anda akan disimpan dan Preview Pane akan muncul di jendela Anda.

Langkah 4: Menggunakan Preview Pane

Sekarang Anda dapat menggunakan Preview Pane dengan mudah. Anda dapat memilih file atau folder yang ingin Anda lihat di panel preview. Ketika Anda memilih file, Anda akan dapat melihat isinya di panel preview. Jika Anda memilih folder, Anda akan dapat melihat gambar, dokumen, atau keduanya yang berada di dalamnya. Anda juga dapat menggunakan Preview Pane untuk memilih file atau folder dengan cepat dan mudah tanpa harus mengakses dan membukanya satu per satu.

Langkah 5: Mematikan Preview Pane

Jika Anda tidak ingin lagi menggunakan Preview Pane, Anda dapat dengan mudah mematikannya. Caranya adalah dengan membuka Folder Options seperti yang telah dijelaskan di atas, kemudian hapus centang pada kotak yang bernama "Show preview pane". Ketika Anda mengklik tombol OK, Preview Pane akan hilang dari jendela Anda.

Kesimpulan

Windows 7 memiliki banyak fitur yang berguna, salah satunya adalah Preview Pane. Ini adalah panel yang dapat Anda gunakan untuk melihat file dan folder tanpa harus mengakses dan membukanya satu per satu. Dengan mengikuti langkah-langkah di atas, Anda dapat dengan mudah menampilkan dan menggunakan Preview Pane di Windows 7. Jika Anda ingin mematikan Preview Pane, Anda juga dapat dengan mudah melakukannya.
