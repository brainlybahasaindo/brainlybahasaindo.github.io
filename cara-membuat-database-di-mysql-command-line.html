Cara Membuat Database Di Mysql Command Line

Apa Itu MySQL?

MySQL adalah salah satu sistem manajemen basis data (DBMS) yang populer. MySQL dikembangkan, didistribusikan, dan didukung oleh Oracle Corporation. MySQL adalah DBMS open source yang dapat digunakan gratis. Database MySQL dapat berisi tabel yang menyimpan data spesifik. Setiap tabel memiliki kolom dan baris yang berisi data penting. MySQL juga memiliki banyak fitur seperti trigger, fungsi, prosedur, dan lainnya. MySQL dirancang untuk digunakan dengan sistem operasi Windows, Linux, dan Mac.

Cara Membuat Database di MySQL Command Line

MySQL dapat diakses menggunakan dua cara: command line dan phpMyAdmin. Command line adalah cara yang paling populer untuk mengakses MySQL. MySQL juga menyediakan antarmuka yang disebut phpMyAdmin yang memungkinkan Anda untuk mengelola database dengan mudah. Cara membuat database di MySQL menggunakan command line adalah sebagai berikut:

1. Login ke Command Line

Pertama, Anda harus login ke command line. Anda dapat membuka command line dengan mengetikkan perintah "mysql -u username -p" di command prompt. Setelah Anda menekan enter, Anda akan diminta untuk memasukkan password. Setelah Anda berhasil login, Anda akan melihat tampilan command prompt berikut: mysql>.

2. Buat Database Baru

Kemudian, Anda dapat membuat database baru dengan mengetikkan perintah "CREATE DATABASE nama_database". Setelah Anda menekan enter, Anda akan melihat tampilan berikut: Database 'nama_database' berhasil dibuat. Selanjutnya, Anda perlu menggunakan database baru yang telah Anda buat dengan mengetikkan perintah "USE nama_database". Setelah Anda menekan enter, Anda akan melihat tampilan berikut: Database 'nama_database' diubah.

3. Buat Tabel Baru

Setelah Anda telah menggunakan database baru, Anda dapat membuat tabel baru dengan mengetikkan perintah "CREATE TABLE nama_tabel (nama_kolom1 tipe_data, nama_kolom2 tipe_data, ...)". Setelah Anda menekan enter, Anda akan melihat tampilan berikut: Tabel 'nama_tabel' berhasil dibuat. Selanjutnya, Anda dapat memeriksa apakah tabel telah berhasil dibuat dengan mengetikkan perintah "SHOW TABLES" dan menekan enter. Anda akan melihat daftar tabel yang terdapat di database Anda.

4. Masukkan Data ke Tabel

Setelah Anda telah membuat tabel, Anda dapat memasukkan data ke tabel dengan menggunakan perintah "INSERT INTO nama_tabel (nama_kolom1, nama_kolom2, ...) VALUES (nilai1, nilai2, ...)". Setelah Anda menekan enter, Anda akan melihat tampilan berikut: Baris berhasil dimasukkan. Anda dapat memeriksa apakah data telah berhasil dimasukkan ke tabel dengan mengetikkan perintah "SELECT * FROM nama_tabel" dan menekan enter. Anda akan melihat daftar data yang tersimpan di tabel.

5. Ubah Tabel

MySQL juga memungkinkan Anda untuk mengubah tabel yang telah Anda buat. Anda dapat menambah, menghapus, atau mengubah kolom tabel dengan menggunakan perintah "ALTER TABLE nama_tabel ADD nama_kolom tipe_data, DROP nama_kolom, ALTER nama_kolom tipe_data". Setelah Anda menekan enter, Anda akan melihat tampilan berikut: Tabel berhasil diubah. Anda dapat memeriksa apakah tabel telah berhasil diubah dengan mengetikkan perintah "DESCRIBE nama_tabel" dan menekan enter. Anda akan melihat daftar kolom yang terdapat pada tabel.

6. Hapus Tabel

Jika Anda tidak lagi memerlukan tabel yang telah Anda buat, Anda dapat menghapus tabel dengan mengetikkan perintah "DROP TABLE nama_tabel". Setelah Anda menekan enter, Anda akan melihat tampilan berikut: Tabel berhasil dihapus. Anda dapat memeriksa apakah tabel telah berhasil dihapus dengan mengetikkan perintah "SHOW TABLES" dan menekan enter. Anda akan melihat daftar tabel yang terdapat di database Anda.

7. Hapus Database

Jika Anda tidak lagi memerlukan database yang telah Anda buat, Anda dapat menghapus database dengan mengetikkan perintah "DROP DATABASE nama_database". Setelah Anda menekan enter, Anda akan melihat tampilan berikut: Database 'nama_database' berhasil dihapus. Anda dapat memeriksa apakah database telah berhasil dihapus dengan mengetikkan perintah "SHOW DATABASES" dan menekan enter. Anda akan melihat daftar database yang terdapat di server MySQL.

8. Logout dari Command Line

Setelah Anda selesai mengelola database, Anda dapat logout dari command line dengan mengetikkan perintah "EXIT". Setelah Anda menekan enter, Anda akan melihat tampilan berikut: Anda berhasil logout dari MySQL. Dengan menggunakan command line, Anda dapat dengan mudah membuat, mengelola, dan menghapus database MySQL.

Kesimpulan

MySQL adalah salah satu DBMS yang populer dan banyak digunakan. MySQL dapat diakses menggunakan dua cara: command line dan phpMyAdmin. Command line adalah cara yang paling populer untuk mengakses MySQL. Cara membuat database di MySQL menggunakan command line adalah dengan login ke command line, membuat database baru, membuat tabel baru, memasukkan data ke tabel, mengubah tabel, menghapus tabel, dan menghapus database. Dengan menggunakan command line, Anda dapat dengan mudah membuat, mengelola, dan menghapus database MySQL.
