Apa Itu Hijrah?

Hijrah adalah perpindahan dari satu tempat ke tempat lain. Kata Hijrah berasal dari bahasa Arab yang berarti “untuk pindah”. Pada masa Nabi Muhammad SAW, Hijrah adalah proses perpindahan dari Mekkah ke Madinah. Proses Hijrah ini sebagai langkah untuk menghindari perlakuan penyiksaan dari para tokoh Quraisy di Mekkah. Dengan Hijrah, para pengikutnya dapat menyelamatkan diri dari perlakuan buruk para tokoh Quraisy.

Kenapa Para Pengikut Nabi Muhammad SAW Hijrah?

Para pengikut Nabi Muhammad SAW hijrah karena di Mekkah mereka mengalami penyiksaan dari para tokoh Quraisy. Para tokoh Quraisy di Mekkah tidak menyukai kehadiran Nabi Muhammad SAW dan para pengikutnya. Mereka menuduh Nabi Muhammad SAW sebagai orang yang menyebarkan ajaran yang salah. Para pengikut Nabi Muhammad SAW juga diancam dengan tuduhan palsu. Oleh karena itu, para pengikut Nabi Muhammad SAW hijrah ke Madinah untuk menghindari perlakuan penyiksaan dari para tokoh Quraisy. Hijrah ini juga sebagai cara untuk menyebarkan ajaran Islam yang sesungguhnya.

Apa yang Dikatakan Nabi Muhammad SAW tentang Hijrah?

Nabi Muhammad SAW menyarankan para pengikutnya untuk hijrah ke Madinah. Nabi Muhammad SAW juga mengatakan bahwa mereka akan mendapatkan hak istimewa jika mereka hijrah ke Madinah. Nabi Muhammad SAW juga menyatakan bahwa para pengikutnya yang meninggalkan Mekkah akan diberi perlindungan oleh Allah. Dengan demikian, Nabi Muhammad SAW menyarankan para pengikutnya untuk hijrah ke Madinah sebagai cara untuk menghindari penyiksaan dari para tokoh Quraisy di Mekkah.
