Pengenalan Tentang While Python

Python merupakan bahasa pemrograman yang mudah dipelajari. Bahasa pemrograman ini dapat digunakan untuk menulis kode yang berjalan di berbagai platform. Dengan bahasa pemrograman ini, Anda dapat membuat berbagai aplikasi, seperti aplikasi web, aplikasi desktop, dan lainnya. Salah satu fitur yang terkenal dari Python adalah fitur loop. Fitur loop ini digunakan untuk membuat kode yang berulang-ulang tanpa harus menulisnya berulang-ulang. Salah satu loop yang terkenal adalah loop while. Loop while dalam Python adalah loop yang akan diulangi berulang-ulang, selama kondisi yang ditentukan terpenuhi.

Cara Menggunakan While Loop di Python

Untuk menggunakan loop while dalam Python, Anda harus mengetahui sintaks dasar dari loop while. Sintaks dasar dari loop while adalah sebagai berikut: while (kondisi): #kode yang akan diulang Kondisi dalam loop while akan diperiksa setiap kali loop dijalankan. Jika kondisi bernilai benar, maka kode yang ada di dalam loop akan dijalankan. Jika kondisi bernilai salah, maka loop akan berakhir. Dengan menggunakan loop while, Anda dapat membuat kode yang berulang-ulang tanpa harus menulisnya berulang-ulang.

Contoh Penggunaan While Loop di Python

Untuk memahami lebih jauh tentang cara menggunakan loop while di Python, mari kita lihat contoh penggunaan loop while di Python. Anda dapat menggunakan loop while untuk menampilkan angka dari 1 sampai 10 seperti berikut: i = 1 while (i <= 10): print (i) i = i + 1 Di sini, variabel i digunakan untuk menyimpan nilai counter. Pertama-tama, nilai counter ditetapkan ke 1. Kemudian, kondisi i kurang dari atau sama dengan 10 diperiksa. Jika kondisi terpenuhi, maka kode dalam loop akan dijalankan. Kode dalam loop ini adalah mencetak nilai counter. Setelah mencetak nilai counter, nilai counter ditambah 1. Proses ini akan berulang-ulang sampai nilai counter lebih dari 10, yang berarti loop akan berakhir.

Cara Mengakhiri Loop While di Python

Selain menggunakan kondisi untuk mengakhiri loop while, Anda juga dapat menggunakan perintah break untuk mengakhiri loop while. Perintah break dapat digunakan untuk mengakhiri loop while kapan saja Anda inginkan. Misalnya, Anda dapat menggunakan perintah break untuk mengakhiri loop while ketika nilai counter bernilai 5. Contoh penggunaannya adalah sebagai berikut: i = 1 while (i <= 10): print (i) i = i + 1 if (i == 5): break Di sini, kita menambahkan kondisi if untuk memeriksa nilai counter. Jika nilai counter bernilai 5, maka loop while akan berakhir dengan menggunakan perintah break.

Kesimpulan

Itulah cara menggunakan loop while di Python. Dengan menggunakan loop while, Anda dapat membuat kode yang berulang-ulang tanpa harus menulisnya berulang-ulang. Anda juga dapat menggunakan kondisi atau perintah break untuk mengakhiri loop while kapan saja Anda inginkan. Sekarang Anda telah belajar tentang cara menggunakan while loop di Python, jadi silakan coba sendiri di komputer Anda.
