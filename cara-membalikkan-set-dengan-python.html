Cara Membalikkan Set Dengan Python

Apa itu Set?

Set adalah tipe data yang terdiri dari elemen-elemen yang unik, berurutan, dan tidak berindeks. Set terutama digunakan untuk operasi himpunan seperti menggabungkan, berbeda, gabungan, dan lainnya. Set juga digunakan untuk menghapus duplikat dari sebuah daftar. Dalam Python, ada dua jenis set, yaitu set normal dan frozenset. Set normal dapat berubah, sedangkan frozenset tidak bisa diubah setelah dibuat. Sebagian besar operasi yang tersedia untuk set normal juga dapat digunakan untuk frozenset.

Apa itu Membalikkan Set?

Membalikkan set adalah proses mengubah urutan elemen dalam himpunan. Pada dasarnya, set tidak dapat diurutkan, jadi untuk membalikkan set, Anda harus membuat set baru dengan elemen yang sama namun berurut secara berlawanan. Python memiliki fungsi bawaan yang disebut reversed () yang dapat digunakan untuk membalikkan set. Fungsi ini mengembalikan generator yang membungkus set dan mengembalikan elemen-elemen dalam urutan terbalik.

Cara Membalikkan Set dengan Fungsi reversed ()

Untuk membalikkan set dalam Python, gunakan fungsi reversed (). Fungsi ini mengembalikan generator yang mengandung elemen-elemen set yang berurutan secara terbalik. Anda dapat menggunakan fungsi ini dengan mudah dengan menggunakan loop for. Berikut adalah contoh singkat tentang cara membalikkan set dengan fungsi reversed ():

Contoh 1: Membalikkan Set dengan Fungsi reversed ()

Berikut adalah contoh singkat tentang cara membalikkan set dengan fungsi reversed ():

 # membuat set
 set = {1, 2, 3, 4, 5}
 
 # membalikkan set dengan fungsi reversed ()
 reversed_set = reversed(set)
 
 # mencetak set yang terbalik
 for i in reversed_set:
 print (i)
 
 # Hasil
 5
 4 
 3
 2
 1

Dalam contoh di atas, kami membuat set dan membalikkan set dengan fungsi reversed (). Kemudian kami menggunakan loop for untuk mencetak semua elemen dalam himpunan yang terbalik. Hasilnya adalah elemen-elemen set yang berurutan secara terbalik.

Cara Membalikkan Set dengan Slicing

Selain menggunakan fungsi reversed (), Anda juga dapat membalikkan set dengan menggunakan slicing. Berikut adalah contoh singkat tentang cara membalikkan set dengan menggunakan slicing:

Contoh 2: Membalikkan Set dengan Slicing

 # membuat set
 set = {1, 2, 3, 4, 5}
 
 # membalikkan set dengan slicing
 reversed_set = set[::-1]
 
 # mencetak set yang terbalik
 for i in reversed_set:
 print (i)
 
 # Hasil
 5
 4 
 3
 2
 1

Dalam contoh di atas, kami membuat set dan membalikkan set dengan menggunakan slicing. Kemudian kami menggunakan loop for untuk mencetak semua elemen dalam himpunan yang terbalik. Hasilnya adalah elemen-elemen set yang berurutan secara terbalik.

Cara Membalikkan Set dengan Perulangan

Selain menggunakan fungsi reversed () atau slicing, Anda juga dapat membalikkan set dengan menggunakan loop for. Berikut adalah contoh singkat tentang cara membalikkan set dengan menggunakan loop for:

Contoh 3: Membalikkan Set dengan Perulangan

 # membuat set
 set = {1, 2, 3, 4, 5}
 
 # membuat list kosong
 reversed_list = []
 
 # membalikkan set dengan loop for
 for i in set:
 reversed_list.insert(0,i)
 
 # mencetak list yang terbalik
 for i in reversed_list:
 print (i)
 
 # Hasil
 5
 4 
 3
 2
 1

Dalam contoh di atas, kami membuat set dan membalikkan set dengan menggunakan loop for. Kami membuat list kosong untuk menyimpan elemen set yang terbalik. Kemudian kami menggunakan loop for untuk membalikkan set dan menambahkan elemen-elemen set ke dalam list kosong. Akhirnya kami menggunakan loop for untuk mencetak semua elemen dalam list yang terbalik. Hasilnya adalah elemen-elemen set yang berurutan secara terbalik.

Kesimpulan

Dalam artikel ini, kami telah membahas tentang cara membalikkan set dengan Python. Kami telah membahas tentang tiga cara yang berbeda untuk membalikkan set, yaitu menggunakan fungsi reversed (), slicing, dan loop for. Setiap metode memiliki kelebihan dan kekurangannya sendiri. Namun, semua metode akan memberikan hasil yang sama, yaitu elemen-elemen set yang berurutan secara terbalik.
