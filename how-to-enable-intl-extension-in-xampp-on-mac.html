Cara Mengaktifkan Ekstensi Intl Pada Xampp Di Mac

Terbaik Pelajaran bahasa indonesia tips dan trik website . Cari apa saja topik Pelajaran bahasa indonesia ide di website ini.

Tahun 2023, dunia teknologi sudah semakin canggih. Berbagai macam alat telah dirancang untuk memudahkan pekerjaan para pengembang. Salah satunya adalah XAMPP, sebuah paket perangkat lunak yang menyertakan Apache, MySQL, dan banyak lagi. Namun, fitur ini juga memerlukan ekstensi yang disebut intl.

Intl adalah sebuah ekstensi yang digunakan untuk mengaktifkan berbagai fitur seperti multibyte string dan collation. Jadi, jika Anda ingin mengaktifkan intl extension di XAMPP di Mac, Anda harus mengikuti panduan berikut.

1. Download File intl.so

Yang perlu Anda lakukan pertama adalah mengunduh file intl.so. File ini tersedia untuk berbagai sistem operasi, jadi pastikan untuk mendapatkan versi yang tepat. File ini berisi kode yang diperlukan untuk mengaktifkan intl.so.

2. Tempatkan File di Folder Ekstensi

Setelah Anda mendownload intl.so, tempatkan file ini di folder ekstensi XAMPP. Folder ini berbeda untuk setiap sistem operasi, jadi Anda harus tahu letaknya. Biasanya, folder ini terletak di folder utama XAMPP. Sebaiknya, Anda membuat folder khusus untuk file intl.so.

3. Edit File php.ini

Selanjutnya, Anda harus mengedit file php.ini. File ini berisi berbagai konfigurasi yang dapat Anda ubah. Cari baris “extension=php_intl.so” dan pastikan untuk menghapus tanda pagar (comment) sebelumnya. Jika tanda pagar ada, hapus dan konfigurasi akan diaktifkan.

4. Mulai Ulang Server

Setelah mengedit php.ini, Anda harus memulai ulang server. Pergi ke folder utama XAMPP, lalu klik tombol “Start”. Ini akan memulai ulang server dan memastikan bahwa konfigurasi berlaku.

5. Cek Instalasi

Terakhir, Anda harus memeriksa apakah instalasi intl telah berhasil. Caranya adalah dengan membuka halaman phpinfo di server Anda. Halaman ini akan menampilkan informasi tentang semua ekstensi yang diinstal. Cari seksi “Intl” dan pastikan bahwa statusnya adalah “enabled”.

Kesimpulan

Mengaktifkan intl extension di XAMPP di Mac cukup mudah. Yang perlu Anda lakukan pertama adalah mendownload file intl.so. Tempatkan file di folder ekstensi XAMPP, lalu edit php.ini dan hapus tanda pagar. Selanjutnya, mulai ulang server dan periksa instalasi. Dengan mengikuti panduan ini, Anda akan dengan mudah mengaktifkan intl extension di XAMPP di Mac.
