Apa Itu Php Artisan Server?

Terbaik Pelajaran bahasa indonesia tips dan trik website . Cari apa saja topik Pelajaran bahasa indonesia ide di website ini.

PHP Artisan adalah sebuah baris perintah yang dapat digunakan untuk menjalankan berbagai macam tugas pemeliharaan dan pengembangan web. Artisan adalah sebuah fitur dari Laravel, framework PHP modern, yang memungkinkan Anda untuk menulis tugas-tugas yang berulang secara cepat dan mudah. Artisan memberi Anda sebuah antarmuka konsol untuk menjalankan berbagai macam tugas seperti menghasilkan skema database baru, membuat model, mengatur jalur routing, membuat controller dan view, dll.

Selain itu, PHP Artisan juga memungkinkan Anda untuk menjalankan server lokal di komputer Anda. Server ini akan menjalankan situs web Laravel lokal di komputer Anda. Dengan server ini, Anda tidak perlu mengkonfigurasi server web lokal seperti Apache atau Nginx. Anda dapat dengan mudah menjalankan situs web yang dibuat menggunakan Laravel dengan menggunakan PHP Artisan Server.

Bagaimana Cara Menjalankan PHP Artisan Server?

Untuk menjalankan server PHP Artisan, Anda harus berada di direktori root aplikasi Laravel Anda. Setelah itu, buka terminal Anda dan ketikkan perintah berikut:

php artisan serve

Setelah itu, server lokal Anda akan mulai berjalan. Setelah server berjalan, Anda dapat membuka situs web Anda pada browser Anda dengan mengakses alamat berikut:

http://localhost:8000

Cara Mengubah Port PHP Artisan Server

Oleh karena itu, Anda mungkin ingin mengubah port yang digunakan oleh server lokal Anda. Anda dapat mengubah port ini dengan menambahkan opsi --port ke perintah php artisan serve. Sebagai contoh, jika Anda ingin mengubah port ke 8080, Anda harus menjalankan perintah berikut:

php artisan serve --port=8080

Setelah itu, Anda harus mengakses situs web Anda pada port 8080. Anda dapat mengaksesnya dengan mengakses alamat berikut:

http://localhost:8080

Kesimpulan

Itulah cara menjalankan server PHP Artisan. Dengan server ini, Anda dapat dengan mudah menjalankan situs web Laravel lokal di komputer Anda. Selain itu, Anda juga dapat dengan mudah mengubah port yang digunakan oleh server lokal Anda dengan menambahkan opsi --port ke perintah php artisan serve.
