Membuat Variabel Pada Php

Apa Itu Variabel?

Variabel adalah sebuah cara untuk menyimpan nilai. Nilai ini bisa berupa angka, teks, atau boolean. Variabel dibuat agar kita lebih mudah untuk mengakses data yang disimpan. Di PHP, dalam membuat variabel, kita harus memberi nama untuk variabel tersebut. Jika tidak, PHP tidak akan dapat membedakan di antara variabel dan nilai lainnya.

Membuat Variabel di PHP

Membuat variabel di PHP relatif mudah. Kita hanya perlu menggunakan tanda dollar ($) sebelum nama variabel. Variabel dapat berupa huruf, angka, dan underscore (_). Jika variabel berupa 2 atau lebih kata, gunakan camelcase (dengan huruf kapital pada awal kata setelah kata pertama). Perhatikan contoh berikut:

$nama ="John Doe";
$umur = 25;
$tinggiBadan = 170;
$isMahasiswa = true;

Pada contoh di atas, kita telah membuat 4 variabel. $nama berupa teks, $umur berupa angka, $tinggiBadan berupa angka, dan $isMahasiswa berupa boolean (true atau false).

Mengakses Variabel di PHP

Untuk mengakses variabel, kita hanya perlu menuliskan nama variabel. Sebagai contoh, jika kita ingin mengakses nilai dari variabel $nama, kita hanya perlu mengetik $nama. Maka, PHP akan mengembalikan nilai dari variabel tersebut, yaitu "John Doe".

Mengubah Nilai Variabel di PHP

Jika kita ingin mengubah nilai dari variabel, kita hanya perlu menuliskan variabel lagi dengan nilai baru. Sebagai contoh, jika kita ingin mengubah nilai dari $nama menjadi "Jane Doe", kita hanya perlu menuliskan $nama ="Jane Doe" di dalam kode program. Maka, nilai dari variabel $nama akan berubah menjadi "Jane Doe".

Menghapus Variabel di PHP

Untuk menghapus variabel di PHP, kita hanya perlu menggunakan fungsi unset(). Sebagai contoh, jika kita ingin menghapus variabel $nama, kita hanya perlu menuliskan unset($nama) di dalam kode program. Maka, variabel $nama akan dihapus.

Menggabungkan Nilai Variabel di PHP

Kadang-kadang kita ingin menggabungkan nilai dari beberapa variabel menjadi satu. Untuk melakukannya, kita dapat menggunakan operator titik (.). Sebagai contoh, jika kita ingin menggabungkan variabel $nama dan $umur menjadi "John Doe (25)", kita hanya perlu menggunakan operator titik seperti berikut:

echo $nama . " (" . $umur . ")";

Maka, kode di atas akan mengembalikan nilai "John Doe (25)".

Penutup

Itulah cara untuk membuat variabel di PHP. Jika kita ingin menggunakan variabel dalam kode program, kita harus membuat variabel terlebih dahulu. Kita juga dapat mengakses, mengubah, menghapus, dan menggabungkan nilai variabel. Dengan memahami cara-cara di atas, kita dapat lebih mudah membuat program PHP yang lebih kompleks.
