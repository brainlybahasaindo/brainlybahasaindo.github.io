Apa itu User Account Control?

User Account Control (UAC) adalah sebuah sistem keamanan yang dibuat untuk membantu melindungi komputer Windows Anda. Ini mengizinkan administrator untuk mengatur tingkatan akses pengguna yang berbeda. UAC juga membantu Anda membatasi yang dapat mengakses atau mengubah data yang ada di komputer Anda. UAC menampilkan sebuah pesan untuk menanyakan persetujuan pengguna sebelum mengizinkan pengguna untuk mengubah status sistem atau data.

Mengapa Anda Harus Mematikan User Account Control di Windows 11?

User Account Control (UAC) diciptakan untuk membantu melindungi data yang ada di komputer Windows Anda. UAC akan menampilkan sebuah pesan yang meminta persetujuan untuk mengizinkan pengguna untuk mengubah status sistem atau data. Namun, dalam beberapa situasi, Anda mungkin ingin mematikan UAC. Jika Anda bekerja dengan program yang diperlukan untuk mengubah setting sistem atau data, Anda mungkin ingin mematikan UAC sehingga Anda tidak terus-menerus menerima pesan UAC. Selain itu, jika Anda bekerja dengan banyak program, Anda mungkin ingin mematikan UAC sehingga Anda tidak perlu mengklik melalui banyak pesan UAC.

Cara Mematikan User Account Control di Windows 11

Langkah 1: Buka Control Panel

Untuk mematikan UAC, Anda harus membuka Control Panel. Anda dapat melakukannya dengan mengklik Start dan mengetikkan "Control Panel" di kotak pencarian. Setelah itu, Anda akan melihat Control Panel di daftar hasil pencarian. Klik Control Panel untuk membukanya.

Langkah 2: Buka User Account Control Settings

Setelah Anda membuka Control Panel, Anda harus mencari User Account Control Settings. Cari kata "User Account Control" di panel kontrol dan klik tombol "Change User Account Control settings". Setelah itu, Anda akan melihat sebuah jendela yang berjudul "User Account Control Settings".

Langkah 3: Mematikan User Account Control di Windows 11

Di jendela User Account Control Settings, Anda akan melihat sebuah slider dengan tiga pilihan. Jika Anda ingin mematikan UAC, pilih opsi "Never Notify". Setelah Anda memilih opsi ini, klik tombol "OK" untuk menyimpan pengaturan. UAC sekarang telah dinonaktifkan di Windows 11.
