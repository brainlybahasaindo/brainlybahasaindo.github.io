Cara Periksa Apakah Cookie Sudah Kedaluwarsa Di Javascript

Apa Itu Cookies?

Cookies adalah berkas data yang disimpan di komputer pengguna saat mereka mengunjungi situs web. Cookies disimpan dalam folder yang berbeda di setiap browser, seperti Chrome, Firefox, atau Safari. Cookies dapat digunakan untuk menyimpan informasi seperti preferensi pengguna, informasi login, dan banyak lagi. Cookies juga dapat digunakan untuk meningkatkan pengalaman pengguna saat mengunjungi situs web. Sebagai contoh, jika Anda mengunjungi situs web dan mengklik tombol 'Login', maka Anda akan menerima cookie yang berisi informasi seperti nama pengguna dan kata sandi Anda. Ini memungkinkan Anda untuk masuk ke situs web tanpa harus mengisi informasi login lagi.

Apa Itu Kedaluwarsa Cookies?

Kedaluwarsa cookies adalah batas waktu yang ditentukan untuk cookie. Setelah kedaluwarsa, cookie tidak lagi valid dan harus dihapus dari sistem. Ini bisa bervariasi dari satu menit hingga bertahun-tahun. Kedaluwarsa cookies digunakan untuk mengontrol masa berlaku cookie, karena cookie yang lama mungkin sudah tidak valid atau sudah tidak sesuai dengan tujuan awalnya. Ini juga dapat digunakan untuk memastikan bahwa cookie yang sama tidak akan dimuat ulang berulang kali. Kedaluwarsa dapat ditentukan secara manual atau secara otomatis.

Bagaimana Cara Memeriksa Apakah Cookie Sudah Kedaluwarsa di Javascript?

Jika Anda ingin memeriksa apakah cookie telah kedaluwarsa di Javascript, Anda harus menggunakan metode 'getTime()'. Metode ini akan mengembalikan waktu dalam milidetik sejak tanggal 1 Januari 1970. Dengan menggunakan metode ini, Anda dapat membandingkan waktu sekarang dengan waktu kedaluwarsa cookie dan menentukan apakah cookie telah kedaluwarsa atau belum. Berikut ini adalah contoh kode yang dapat Anda gunakan untuk memeriksa apakah cookie telah kedaluwarsa di Javascript:

 var now = new Date().getTime();
 
 var expires = parseInt(getCookie("expires"));
 
 if (now < expires) {
 } else {
 }

Bagaimana Cara Menentukan Kedaluwarsa Cookie di Javascript?

Untuk menentukan kedaluwarsa cookie di Javascript, Anda dapat menggunakan metode 'setTime()'. Metode ini memungkinkan Anda untuk menentukan waktu kedaluwarsa cookie dalam milidetik. Berikut ini adalah contoh kode yang dapat Anda gunakan untuk menentukan kedaluwarsa cookie di Javascript:

 var now = new Date().getTime();
 
 var expires = now + (60 * 60 * 1000); // 1 jam
 
 setCookie("expires", expires);

Kesimpulan

Kedaluwarsa cookie di Javascript dapat dengan mudah diperiksa dengan menggunakan metode 'getTime()' dan 'setTime()'. Dengan menggunakan metode ini, Anda dapat dengan mudah memeriksa apakah cookie telah kedaluwarsa atau belum, dan menentukan waktu kedaluwarsa cookie. Dengan menggunakan kedaluwarsa cookies, Anda dapat mengontrol masa berlaku cookie dan memastikan bahwa cookie yang sama tidak dimuat ulang berulang kali.
