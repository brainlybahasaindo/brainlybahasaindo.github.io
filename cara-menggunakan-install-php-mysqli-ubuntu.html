Cara Menggunakan PHP MYSQLI Ubuntu 2023

Pertama-tama, Anda perlu mengunduh paket PHP MYSQLI terbaru. Anda bisa menemukannya di situs web resmi PHP. Setelah mengunduh paket, Anda harus menginstalnya di mesin Anda. Setelah itu, Anda harus mengaktifkan ekstensi MySQLi di file konfigurasi PHP. Setelah itu, Anda harus mengaktifkan modul ekstensi MySQLi di server. Anda juga harus mengatur konfigurasi port dan alamat IP untuk mengakses database.

Langkah 1: Instal PHP MYSQLI

Untuk menginstal PHP MYSQLI, Anda perlu menggunakan perintah apt-get. Pertama-tama, buka Terminal dan jalankan perintah berikut:

sudo apt-get install php-mysql

Setelah menjalankan perintah di atas, Anda harus menunggu beberapa saat hingga proses instalasi selesai. Setelah itu, Anda harus memeriksa apakah PHP MYSQLI telah terinstal dengan benar dengan cara menjalankan perintah berikut:

php -m | grep mysqli

Langkah 2: Mengaktifkan Ekstensi MySQLi

Setelah menginstal PHP MYSQLI, Anda harus mengaktifkan ekstensi MySQLi di file konfigurasi PHP. Buka file konfigurasi PHP Anda dan cari bagian berikut:

extension=mysqli

Kemudian, aktifkan ekstensi dengan menghapus tanda pagar di depan baris tersebut. Setelah itu, simpan perubahan dan restart server. Anda juga harus memastikan bahwa ekstensi telah diaktifkan dengan benar dengan cara menjalankan perintah berikut:

php -m | grep mysqli

Langkah 3: Mengaktifkan Modul Ekstensi MySQLi

Untuk mengaktifkan modul ekstensi MySQLi, Anda harus menjalankan perintah berikut:

sudo a2enmod mysql

Setelah menjalankan perintah di atas, Anda harus menunggu beberapa saat hingga proses instalasi selesai. Setelah itu, restart web server Anda untuk memastikan bahwa modul ekstensi telah diaktifkan dengan benar.

Langkah 4: Mengatur Konfigurasi Port dan Alamat IP

Setelah mengaktifkan modul ekstensi MySQLi, Anda harus mengatur konfigurasi port dan alamat IP untuk mengakses database. Buka file konfigurasi PHP Anda dan cari bagian berikut:

$cfg['Servers'][$i]['host'] ='localhost';

Kemudian, ubah baris tersebut menjadi:

$cfg['Servers'][$i]['host'] ='127.0.0.1';

Setelah itu, simpan perubahan dan restart server. Anda juga harus memeriksa apakah konfigurasi telah diterapkan dengan benar dengan cara menjalankan perintah berikut:

netstat -an | grep 3306
