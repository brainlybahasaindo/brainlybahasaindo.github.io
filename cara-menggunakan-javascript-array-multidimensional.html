Cara Menggunakan Array Multidimensional Di Javascript

Apa Itu Array Multidimensional?

Array multidimensional adalah sebuah struktur data yang menyimpan satu atau lebih array di dalamnya. Setiap array yang disimpan di dalam array multidimensional bisa memiliki jumlah elemen yang berbeda. Array multidimensional juga dikenal dengan sebutan array dua dimensi, array tiga dimensi, atau array lebih dari tiga dimensi. Biasanya, istilah array multidimensional digunakan ketika array memiliki lebih dari dua dimensi.

Kapan Kita Menggunakan Array Multidimensional?

Array multidimensional biasanya digunakan ketika kita ingin menyimpan sejumlah data yang saling terkait. Misalnya, jika kita ingin membuat sebuah tabel nilai untuk sekelas, maka array multidimensional adalah pilihan yang tepat. Kita dapat menyimpan data tentang nama siswa, nilai ujian, dan nilai akhir di dalam array multidimensional. Data yang disimpan di dalam array multidimensional akan lebih mudah dibaca dan diproses.

Cara Membuat Array Multidimensional di JavaScript

Membuat array multidimensional di JavaScript sama seperti membuat array biasa, hanya saja kita perlu menyimpan array lagi di dalam array tersebut. Berikut ini adalah contoh sederhana dari membuat array multidimensional di JavaScript:

var arrayMulti = [[1,2,3], [4,5,6], [7,8,9]];

Di atas adalah contoh sederhana dari membuat array multidimensional. Kita dapat melihat bahwa kita telah membuat array dengan tiga elemen, dan setiap elemen adalah array lain dengan tiga elemen juga. Di dalam array multidimensional ini, kita dapat menyimpan data seperti nama siswa, nilai ujian, dan nilai akhir.

Cara Mengakses Element di Dalam Array Multidimensional

Untuk mengakses element di dalam array multidimensional, kita harus menggunakan dua indeks. Indeks pertama akan menunjukkan ke element array yang berada di luar, sedangkan indeks kedua akan menunjukkan ke element array yang berada di dalam. Berikut ini adalah contoh sederhana dari mengakses element di dalam array multidimensional:

arrayMulti[1][2]; // Output: 6

Di atas adalah contoh sederhana dari mengakses element di dalam array multidimensional. Di sini, kita mengakses element yang berada di indeks ke-1 array luar, dan indeks ke-2 array di dalam. Hasil yang kita dapat adalah angka 6.

Cara Mengubah Element di Dalam Array Multidimensional

Untuk mengubah element di dalam array multidimensional, kita harus mengakses element tersebut dengan menggunakan dua indeks, kemudian mengganti element tersebut dengan nilai baru. Berikut ini adalah contoh sederhana dari mengubah element di dalam array multidimensional:

arrayMulti[1][2] = 10;

Di atas adalah contoh sederhana dari mengubah element di dalam array multidimensional. Di sini, kita mengakses element yang berada di indeks ke-1 array luar, dan indeks ke-2 array di dalam, kemudian mengganti nilai 6 dengan 10.

Cara Menambahkan Element di Dalam Array Multidimensional

Untuk menambahkan element di dalam array multidimensional, kita harus mengakses element tersebut dengan menggunakan dua indeks, kemudian menambahkan element baru dengan nilai yang diinginkan. Berikut ini adalah contoh sederhana dari menambahkan element di dalam array multidimensional:

arrayMulti[2].push(11);

Di atas adalah contoh sederhana dari menambahkan element di dalam array multidimensional. Di sini, kita mengakses element yang berada di indeks ke-2 array luar, kemudian menambahkan element baru dengan nilai 11.

Cara Menghapus Element di Dalam Array Multidimensional

Untuk menghapus element di dalam array multidimensional, kita harus mengakses element tersebut dengan menggunakan dua indeks, kemudian menghapus element tersebut dengan menggunakan fungsi array splice. Berikut ini adalah contoh sederhana dari menghapus element di dalam array multidimensional:

arrayMulti[2].splice(2, 1);

Di atas adalah contoh sederhana dari menghapus element di dalam array multidimensional. Di sini, kita mengakses element yang berada di indeks ke-2 array luar, kemudian menghapus element yang berada pada indeks ke-2 dengan menggunakan fungsi splice.

Kesimpulan

Array multidimensional adalah sebuah struktur data yang memungkinkan kita untuk menyimpan sejumlah data yang saling terkait. Array multidimensional dapat digunakan untuk menyimpan data seperti nama siswa, nilai ujian, dan nilai akhir. Membuat, mengakses, mengubah, menambahkan, dan menghapus element di dalam array multidimensional di JavaScript sama seperti array biasa, hanya saja kita harus menggunakan dua indeks untuk mengakses element yang berada di dalam array luar.
