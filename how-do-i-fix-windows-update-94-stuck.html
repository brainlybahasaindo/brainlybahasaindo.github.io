Cara Memperbaiki Windows Update 94 Yang Terjebak

Windows Update 94 adalah versi terbaru dari sistem operasi Windows yang dirilis tahun 2023. Namun, beberapa pengguna telah mengalami kesulitan saat mencoba memperbaruinya. Salah satu masalah yang paling umum adalah saat Windows Update 94 terjebak. Ini bisa sangat mengganggu dan membuat orang merasa frustrasi. Jika Anda juga mengalami masalah ini, Anda tidak perlu khawatir. Ada beberapa cara untuk memperbaiki Windows Update 94 yang terjebak.

Cara Memperbaiki Windows Update 94 Yang Terjebak

1. Periksa Koneksi Internet Anda

Jika Anda mengalami masalah dengan Windows Update 94 yang terjebak, mungkin karena koneksi internet Anda tidak stabil. Jadi, yang pertama kali harus Anda lakukan adalah memeriksa koneksi internet Anda. Cobalah untuk membuka beberapa situs web dan lihat apakah Anda dapat mengaksesnya dengan lancar. Jika Anda mengalami masalah dengan koneksi internet Anda, Anda harus menghubungi penyedia layanan internet Anda dan meminta bantuan.

2. Perbarui Driver

Kemungkinan besar masalah Anda disebabkan oleh driver yang usang atau rusak. Jadi, yang harus Anda lakukan adalah memperbarui driver Anda. Untuk memperbarui driver Anda, buka Pengontrol, klik Device Manager, pilih perangkat yang ingin Anda perbarui, dan klik Perbarui Driver. Anda juga bisa menggunakan alat pembaruan driver yang disediakan oleh Microsoft. Alat ini akan membantu Anda memperbarui semua driver yang usang atau rusak.

3. Bersihkan Sistem Anda

Jika Anda mengalami masalah dengan Windows Update 94 yang terjebak, mungkin karena sistem Anda terlalu penuh dengan file sampah. Jadi, yang harus Anda lakukan adalah membersihkan sistem Anda. Anda dapat menggunakan alat bantu sistem seperti CCleaner untuk membersihkan sistem Anda. Alat ini akan menghapus semua file sampah dan membuat kinerja sistem Anda lebih optimal.

4. Matikan Firewall

Ketika Anda mencoba memperbarui Windows Update 94, firewall dapat menghalangi proses pembaruan. Jadi, untuk memastikan bahwa Windows Update 94 berjalan dengan lancar, matikan firewall Anda. Cara terbaik untuk melakukannya adalah dengan menggunakan Control Panel. Buka Control Panel, klik System and Security, lalu klik Windows Firewall. Aktifkan opsi Turn Windows Firewall Off.

5. Jalankan Perbaikan Sistem

Jika semua langkah di atas tidak berhasil, Anda harus mencoba menjalankan Perbaikan Sistem. Perbaikan Sistem dapat memperbaiki masalah yang mungkin terjadi dengan sistem Anda. Untuk menjalankan Perbaikan Sistem, buka Control Panel, klik System and Security, lalu klik System. Klik tombol Perbaikan Sistem dan ikuti petunjuk yang muncul di layar.

6. Gunakan Troubleshooter

Troubleshooter adalah alat bantu yang dapat membantu Anda menyelesaikan masalah dengan Windows Update 94. Cara terbaik untuk menggunakan Troubleshooter adalah dengan membuka Control Panel, klik Troubleshooting, lalu klik View All. Pilih Windows Update dan ikuti petunjuk yang muncul di layar.

7. Gunakan Perintah CMD

Anda juga dapat menggunakan perintah Command Prompt untuk memperbaiki Windows Update 94. Cara terbaik untuk menggunakan perintah Command Prompt adalah dengan membuka Command Prompt dengan akses administrator, lalu ketikkan perintah berikut: wuauclt.exe /updatenow. Tunggu hingga proses selesai dan restart komputer Anda.

8. Hapus Folder Windows Update

Jika Anda masih mengalami masalah dengan Windows Update 94, Anda dapat mencoba menghapus folder Windows Update. Cara terbaik untuk melakukannya adalah dengan membuka File Explorer, lalu masuk ke folder C:\Windows\SoftwareDistribution. Di folder ini, Anda dapat menemukan folder Windows Update. Hapus folder ini dan restart komputer Anda.

9. Instal Ulang Windows

Jika semua langkah di atas tidak berhasil, Anda harus mencoba untuk menginstal ulang Windows. Cara terbaik untuk melakukannya adalah dengan membuka Pengontrol, klik Sistem dan Keamanan, lalu klik Backup dan Restore. Klik Restore My Files dan ikuti petunjuk yang muncul di layar. Ini akan menghemat data Anda dan memungkinkan Anda menginstal ulang Windows.

10. Hubungi Microsoft

Jika semua langkah di atas tidak berhasil, Anda harus menghubungi Microsoft dan meminta bantuan. Microsoft akan memberikan Anda layanan dukungan teknis dan membantu Anda menyelesaikan masalah dengan Windows Update 94. Jadi, jika Anda masih mengalami masalah, jangan ragu untuk menghubungi Microsoft dan meminta bantuan.

Kesimpulannya, ada beberapa cara untuk memperbaiki Windows Update 94 yang terjebak. Anda dapat memeriksa koneksi internet Anda, memperbarui driver Anda, bersihkan sistem Anda, matikan firewall Anda, jalankan Perbaikan Sistem, gunakan Troubleshooter, gunakan perintah CMD, hapus folder Windows Update, instal ulang Windows, dan hubungi Microsoft. Semoga informasi ini bermanfaat.
